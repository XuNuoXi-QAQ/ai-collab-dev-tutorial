这份指南将带你完成从安装 KeePassXC 到实现 Linux 桌面、浏览器和安卓手机三端同步的全部配置。

---

### 第一部分：在 Manjaro 上安装 KeePassXC

#### ⭐ 方法一：使用 Pacman（最推荐）
这是最简单、最稳定的方法：
```bash
sudo pacman -S keepassxc
```

#### 🖱️ 方法二：使用 Pamac 图形界面
1. 打开 **"添加/删除软件"** (Pamac)
2. 搜索 `keepassxc`
3. 点击安装即可

#### 📦 方法三：使用 Flatpak（沙盒版）
更新独立但可能影响浏览器集成：
```bash
flatpak remote-add --user --if-not-exists flathub https://dl.flathub.org/repo/flathub.flatpakrepo
flatpak install --user flathub org.keepassxc.KeePassXC
```

#### 📂 方法四：使用 AppImage（便携版）
从 [KeePassXC 官网](https://keepassxc.org/download) 下载，赋予执行权限后直接运行：
```bash
chmod +x KeePassXC-*.AppImage
```

> **⚠️ 特别提醒**：不要安装软件源中的 `keepass` 包！那是依赖 Mono 的原版，在 Linux 上不稳定且容易崩溃。**KeePassXC** 才是原生应用。

### 第二部分：创建你的密码数据库

1. **新建数据库**：启动 KeePassXC，点击 **"新建数据库"**
2. **命名与配置**：为数据库命名，可调整加密解密时间（时间越长越安全，但解锁变慢）
3. **设置主密码**：**最关键的一步！** 设置一个足够复杂的密码。**请务必牢记，一旦遗忘无法恢复！**
4. **保存数据库**：将 `.kdbx` 文件保存在安全位置（如 `~/Documents/MyPasswords.kdbx`）

---

### 第三部分：基本使用

- **添加条目**：点击 **"添加新条目"**，填写标题、用户名、密码（可使用生成器）、URL
- **查看/复制**：双击条目查看详情，右键可复制用户名或密码
- **修改/删除**：选中条目后进行编辑或删除操作

### 第四部分：配置浏览器集成（实现自动填充）

#### 桌面端设置
1. 打开 KeePassXC，进入 **`工具` -> `设置`**
2. 切换到 **`浏览器集成`** 选项卡
3. 勾选 **"启用 KeePassXC 浏览器集成"**
4. 在"允许以下浏览器连接"列表中，勾选你使用的浏览器

#### 浏览器扩展安装（以 Firefox 为例）
1. 在 Firefox 扩展商店搜索 **"KeePassXC-Browser"** 并安装
2. 安装后，点击工具栏中的扩展图标
3. **确保 KeePassXC 桌面端已打开且数据库已解锁**
4. 点击 **"连接"** 按钮
5. 在 KeePassXC 弹出的窗口中，输入标识名（如 `manjaro-firefox`）
6. 点击 **"保存并允许访问"** 完成配对

**配对完成后**：
- 访问网站时，点击扩展图标即可自动填充
- 快捷键：`Alt + Shift + U`（可自定义）
- 可保存新密码、生成强密码

> **⚠️ 注意**：如果使用 Flatpak 安装 KeePassXC 或浏览器，可能因沙箱限制导致连接失败。建议优先使用 Pacman 安装。

### 第五部分：配置安卓客户端（KeePass2Android）

#### 安装应用
从以下渠道下载安装 **KeePass2Android**（免费、开源）：
- [GitHub Releases](https://github.com/PhilippC/keepass2android)
- F-Droid
- Google Play

#### 配置 WebDAV 同步（以坚果云为例）

**第一步：获取 WebDAV 信息（以坚果云为例）**
1. 登录坚果云网页版
2. 进入 **"账户信息" -> "安全选项"**
3. 在 **"第三方应用管理"** 中，点击 **"生成应用密码"**
4. 记录下：
   - 服务器地址：`https://dav.jianguoyun.com/dav/`
   - 用户名：你的坚果云注册邮箱
   - 密码：刚生成的**应用专用密码**（不是登录密码！）

> **其他云盘**：Nextcloud、Seafile 等支持 WebDAV 的云盘，或使用 Syncthing 等点对点同步工具，操作逻辑类似。

**第二步：在手机上配置**
1. 打开 KeePass2Android，点击 **"打开文件"**
2. 点击左上角菜单（汉堡图标），选择 **"WebDAV (HTTP/HTTPS)"**
3. 填写 WebDAV 连接信息：
   - **URL**：`https://dav.jianguoyun.com/dav/你的文件夹/密码库.kdbx`
   - **用户名**：你的云盘账号
   - **密码**：**应用专用密码**（不是你的登录密码）
4. 点击 **"连接"**
5. 解锁数据库：输入你的**主密码**

**第三步：额外功能**
- 支持**指纹或面部识别**解锁
- 提供系统级**自动填充服务**（Android 8.0+）
- 支持在手机其他 App 中自动填充密码
- 修改密码后自动同步回云端

### 第六部分：进阶配置

#### 安全设置（`工具` -> `设置` -> `安全`）
- **空闲时锁定数据库**：设置自动锁定时间
- **离开时锁定**：切换窗口时自动锁定
- **复制密码后清空剪贴板**：建议设置 10-30 秒后自动清空

#### 界面设置（`工具` -> `设置` -> `界面`）
- 切换深色/浅色主题
- 自定义工具栏布局

#### 密码生成器（`工具` -> `设置` -> `密码生成器`）
- 自定义生成规则：字符种类、长度等
- 设置默认密码复杂度

### 第七部分：多设备同步工作原理

```
┌─────────────────────────────────────────────────────────────┐
│                    云端存储 (WebDAV)                        │
│               https://dav.xxx.com/密码库.kdbx              │
└─────────────────────────────────────────────────────────────┘
                           │
          ┌────────────────┼────────────────┐
          │                │                │
          ▼                ▼                ▼
   ┌───────────┐   ┌───────────┐   ┌───────────┐
   │  Linux    │   │  Browser  │   │  Android  │
   │KeePassXC  │◄──│ 扩展插件  │   │KeePass2A  │
   └───────────┘   └───────────┘   └───────────┘
```

**同步机制**：
1. **数据库文件**存储在云端（WebDAV）
2. **所有设备**通过 WebDAV 协议访问同一个 `.kdbx` 文件
3. 任何设备上的**修改都会同步**到云端，其他设备实时可用
4. 主密码是**唯一钥匙**，云端只存储加密后的数据

### 第八部分：常见问题排查

| 问题 | 解决方案 |
| :--- | :--- |
| **浏览器扩展无法连接** | ① 确保 KeePassXC 桌面端正在运行 ② 确保数据库已解锁 ③ 检查是否通过 Flatpak/Snap 安装导致沙箱限制 |
| **WebDAV 同步失败** | ① 检查网络连接 ② 确认服务器地址正确 ③ 确认使用**应用专用密码**而非登录密码 ④ 检查 SSL 证书是否受信任 |
| **安卓无法打开数据库** | ① 确认 URL 路径完整包含 `.kdbx` 文件名 ② 检查应用专用密码是否正确 ③ 尝试使用"高级"模式设置自定义端口和路径 |
| **密码填充失败** | ① 确认对应条目已正确保存 URL ② 检查浏览器扩展是否仍处于连接状态 ③ 重新配对连接 |

### 💎 总结

完成以上配置后，你就拥有了一套完整的开源密码管理体系：

| 平台 | 应用 | 功能 |
| :--- | :--- | :--- |
| **💻 Manjaro** | KeePassXC | 密码管理、数据库编辑、浏览器集成 |
| **🌐 浏览器** | KeePassXC-Browser | 网页自动填充、密码保存 |
| **📱 Android** | KeePass2Android | 移动端访问、自动填充、生物识别 |

**三个平台通过 WebDAV 协议共享同一个加密数据库，实现数据实时同步。** 所有数据完全由你掌控，无需信任任何第三方服务器。
