DeepSeek++ 是一款为 DeepSeek 网页版注入“超能力”的浏览器扩展。它不修改官方服务器，完全在浏览器本地为 DeepSeek 装上“手脚”，让它从聊天机器人变成能执行命令、操作文件、搜索网页的 AI Agent。

🚀 核心功能

· 原生工具调用：让 AI 能自动识别并执行复杂任务。
· MCP工具系统：通过MCP协议连接外部服务，甚至能通过 OfficeCLI 直接处理本地 Word、Excel 文档。
· Agentic记忆系统：拥有跨对话的长期记忆，能记住你的偏好。
· Skill技能系统：支持通过/skill指令一键切换专家模式，也可从社区导入或自己编写新技能。
· 自动化任务：支持定时任务，可按计划自动执行预设指令。

⚙️ 工作原理

它在请求离开浏览器前进行“拦截”和“改造”：

1. 拦截：识别你以 / 开头的特殊指令（如/shell ls）。
2. 翻译：将指令翻译成 AI 能理解的自然语言指令（Prompt 增强）。
3. 执行：AI 返回工具调用指令，插件在本地执行（如列出文件），再把结果塞回对话。

🛠️ 如何安装

目前未上架商店，需手动安装：

1. 获取代码：克隆其GitHub仓库 zhu1090093659/deepseek-pp。
2. 构建项目：在项目目录执行 npm install 和 npm run build。
3. 加载插件：浏览器访问扩展管理页面（如 chrome://extensions/），开启“开发者模式”，点击“加载已解压的扩展程序”，选择 dist/chrome-mv3/ 目录。

⌨️ 怎么用？

安装后，在 DeepSeek 网页版聊天框输入以 / 开头的命令即可：

· /shell ls：列出当前目录的文件
· /memory 记住我喜欢简洁的回答风格：让 AI 记住偏好
· /web_search 2024年诺贝尔物理学奖得主：搜索网页
· /frontend-design 做一个登录页面：生成前端代码

⚠️ 注意事项

· 有门槛：需手动安装和构建，对非技术用户不友好。
· 本地执行：Shell命令在电脑上真实执行，请勿运行来历不明的指令。
· 积极迭代：项目很活跃，近30天有166个Issue和660个Star，未来会更完善。
