# 文件与目录操作命令精讲

文件与目录操作是 Linux 系统管理中最基础也最常用的命令。这组命令帮助你查看、创建、复制、移动、删除、搜索和压缩文件与目录。

---

## 一、核心概念

### 路径类型
| 类型 | 说明 | 示例 |
|------|------|------|
| **绝对路径** | 从根目录 `/` 开始 | `/home/nuoxi/文档/diary` |
| **相对路径** | 从当前目录开始 | `./documents`、`../project` |
| **主目录** | 用户目录 | `~`、`~nuoxi` |

### 特殊目录
| 符号 | 含义 |
|------|------|
| `.` | 当前目录 |
| `..` | 上级目录 |
| `~` | 当前用户的主目录 |
| `-` | 上一个工作目录 |

## 2. `ls` — 列出文件与目录

### 命令格式
```bash
ls                  # 列出当前目录
ls -la              # 列出所有文件（含隐藏），详细信息
ls -lh              # 人类可读的文件大小
ls -lR              # 递归列出所有子目录
ls -lt              # 按修改时间排序（最新在前）
```

### 参数说明
| 参数 | 含义 |
|------|------|
| `-l` | 长格式显示（权限、大小、时间等） |
| `-a` | 显示所有文件（含 `.` 开头的隐藏文件） |
| `-h` | 人类可读的文件大小（KB、MB、GB） |
| `-R` | 递归显示子目录 |
| `-t` | 按修改时间排序 |
| `-S` | 按文件大小排序 |
| `-r` | 反向排序 |

### 输出示例
```bash
ls -la
```
```
total 116
drwxr-xr-x 42 nuoxi nuoxi 4096 Jun 23 10:00 .
drwxr-xr-x  3 root  root  4096 Jun 20 14:00 ..
drwxr-xr-x  2 nuoxi nuoxi 4096 Jun 18 22:14 .agents
-rw-r--r--  1 nuoxi nuoxi  419 Jun 19 00:41 id_ed25519
-rw-------  1 nuoxi nuoxi 1234 Jun 23 09:30 .zshrc
```

### 权限格式解读
```
drwxr-xr-x 1 nuoxi nuoxi 4096 Jun 23 10:00 Documents
││││││││││  │       │      │     │           │
││││││││││  │       │      │     │           └─ 文件名
││││││││││  │       │      │     └─ 修改时间
││││││││││  │       │      └─ 文件大小
││││││││││  │       └─ 用户组
││││││││││  └─ 文件所有者
││││││││││
│││││││││└─ 其他用户权限 (r-x)
││││││││└─ 组权限 (r-x)
│││││││└─ 所有者权限 (rwx)
││││││└─ 文件类型 (d=目录, -=文件, l=链接)
```

### 实际应用
```bash
# 列出所有文件（含隐藏）
ls -la

# 按时间排序（最新的在最前）
ls -lt

# 按大小排序
ls -lS

# 人类可读的大小
ls -lh

# 只显示目录
ls -d */
```

## 3. `tree` — 树形显示目录结构

### 命令格式
```bash
tree                    # 显示当前目录树
tree -L 2 -a            # 2层，含隐藏文件（你经常用）
tree -d                 # 只显示目录
tree -I "node_modules"  # 排除指定目录
tree -o output.txt      # 输出到文件
```

### 参数说明
| 参数 | 含义 |
|------|------|
| `-L N` | 限制显示深度为 N 层 |
| `-a` | 显示隐藏文件 |
| `-d` | 只显示目录 |
| `-I` | 排除匹配的模式 |
| `-o` | 输出到文件 |

### 实际应用
```bash
# 查看当前目录树（2层）
tree -L 2

# 查看包括隐藏文件
tree -L 2 -a

# 排除 node_modules
tree -L 3 -I "node_modules"

# 输出到文件
tree -L 2 -a > structure.txt
```

## 4. `find` — 查找文件

### 命令格式
```bash
find / -name "文件名"              # 按文件名查找
find . -name "*.md" -type f        # 查找所有 Markdown 文件
find . -type d -name "Project"     # 查找目录
find . -size +100M                 # 查找大于 100MB 的文件
find . -mtime -7                   # 查找7天内修改的文件
```

### 参数说明
| 参数 | 含义 |
|------|------|
| `-name` | 按文件名匹配 |
| `-iname` | 忽略大小写匹配 |
| `-type f` | 只查找文件 |
| `-type d` | 只查找目录 |
| `-size` | 按大小过滤 |
| `-mtime` | 按修改时间过滤 |
| `-exec` | 对找到的文件执行命令 |

### 实际应用
```bash
# 查找 AGENTS.md 文件（你之前找过）
find / -name "AGENTS.md" 2>/dev/null

# 查找所有 Markdown 文件
find . -name "*.md" -type f

# 查找大于 100MB 的文件
find / -size +100M 2>/dev/null

# 查找7天内修改的文件
find . -mtime -7 -type f

# 查找并删除
find . -name "*.tmp" -type f -delete

# 查找并执行命令
find . -name "*.log" -exec rm -f {} \;

# 查找并统计行数
find . -name "*.py" -exec wc -l {} \+
```

### 常见查找路径
| 场景 | 命令 |
|------|------|
| 查找配置文件 | `find /etc -name "*.conf"` |
| 查找日志文件 | `find /var/log -name "*.log"` |
| 查找大文件 | `find / -size +1G 2>/dev/null` |
| 查找最近修改的文件 | `find . -mtime -1` |

## 5. `grep` — 搜索文件内容

### 命令格式
```bash
grep "关键词" 文件              # 在文件中搜索
grep -r "关键词" .              # 递归搜索当前目录
grep -l "关键词" *.md           # 只显示包含关键词的文件名
grep -i "关键词" 文件           # 忽略大小写
grep -n "关键词" 文件           # 显示行号
```

### 参数说明
| 参数 | 含义 |
|------|------|
| `-r` | 递归搜索子目录 |
| `-i` | 忽略大小写 |
| `-l` | 只显示文件名 |
| `-n` | 显示行号 |
| `-c` | 统计匹配行数 |
| `-v` | 反向匹配（不包含） |
| `-A N` | 显示匹配行及后 N 行 |
| `-B N` | 显示匹配行及前 N 行 |
| `-C N` | 显示匹配行及前后 N 行 |

### 实际应用
```bash
# 在笔记中搜索 RAG
grep -r "RAG" ~/文档/diary/

# 只返回文件名
grep -l "RAG" ~/文档/diary/*.md

# 搜索错误日志
grep -i "error" /var/log/syslog

# 搜索并显示前后5行
grep -i -C 5 "error" /var/log/syslog

# 排除注释和空行
grep -v "^#" config.conf | grep -v "^$"

# 统计匹配次数
grep -c "error" /var/log/syslog
```

## 6. `cp` — 复制文件或目录

### 命令格式
```bash
cp 源文件 目标文件           # 复制文件
cp -r 源目录 目标目录         # 递归复制目录
cp -i 源 目标                # 覆盖前提示确认
cp -u 源 目标                # 只复制更新的文件
cp -p 源 目标                # 保留文件属性（权限、时间）
```

### 参数说明
| 参数 | 含义 |
|------|------|
| `-r` | 递归复制目录（**复制目录时必须用**） |
| `-i` | 覆盖前提示确认 |
| `-u` | 只复制更新的文件 |
| `-p` | 保留文件属性 |
| `-a` | 归档模式（保留所有属性 + 递归） |
| `-v` | 显示复制过程 |

### 实际应用
```bash
# 复制文件
cp file.txt backup.txt

# 复制目录（必须用 -r）
cp -r project/ project_backup/

# 复制并保留属性
cp -p file.txt backup.txt

# 复制前提示确认
cp -i important.txt backup.txt

# 只复制新文件
cp -u source/* destination/

# 归档复制（保留所有属性）
cp -a project/ project_backup/
```

## 7. `rm` — 删除文件或目录

### 命令格式
```bash
rm 文件                    # 删除文件
rm -r 目录                 # 递归删除目录
rm -f 文件                 # 强制删除（不提示）
rm -rf 目录                # 强制递归删除（危险！）
```

### 参数说明
| 参数 | 含义 |
|------|------|
| `-r` | 递归删除目录 |
| `-f` | 强制删除（不提示确认） |
| `-i` | 删除前提示确认 |

### ⚠️ 安全警告
- `rm -rf` 是**最危险**的命令之一，一旦删除无法恢复。
- 删除前务必确认路径正确。
- 建议使用 `rm -ri` 交互式删除，或先 `ls` 确认。

### 实际应用
```bash
# 删除文件
rm file.txt

# 交互式删除（会询问确认）
rm -i file.txt

# 删除空目录
rmdir empty_dir

# 删除非空目录（递归）
rm -r project/

# 强制删除目录（危险）
rm -rf /tmp/cache/

# 删除前先确认
ls -la /tmp/cache/
rm -rf /tmp/cache/
```

## 8. `mv` — 移动或重命名

### 命令格式
```bash
mv 源 目标          # 移动或重命名
mv file.txt newname.txt  # 重命名
mv file.txt /tmp/        # 移动
mv -i 源 目标       # 覆盖前提示
```

### 实际应用
```bash
# 重命名文件
mv old-name.txt new-name.txt

# 移动文件到目录
mv file.txt ~/documents/

# 移动目录
mv project/ ~/projects/

# 覆盖前确认
mv -i important.txt backup/
```

## 9. `ln` — 创建链接

### 命令格式
```bash
ln -s 源 链接名        # 创建软链接（符号链接）
ln 源 链接名           # 创建硬链接
```

### 软链接 vs 硬链接
| 对比 | 软链接 | 硬链接 |
|------|--------|--------|
| 类似于 | Windows 快捷方式 | 同一个文件的另一个名字 |
| 可跨文件系统 | ✅ | ❌ |
| 可链接目录 | ✅ | ❌ |
| 原文件删除后 | 失效 | 仍然有效 |

### 实际应用
```bash
# 创建软链接
ln -s /usr/bin/python3 /usr/bin/python

# 创建链接到目录
ln -s ~/projects/current ~/current

# 查看链接指向
ls -l /usr/bin/python
# lrwxrwxrwx 1 root root 18 Jun 23 10:00 /usr/bin/python -> /usr/bin/python3
```

## 11. `tar` — 压缩与解压

### 命令格式
```bash
tar -czf 输出.tar.gz 源文件    # 压缩（gzip）
tar -xzf 文件.tar.gz           # 解压（gzip）
tar -cjf 输出.tar.bz2 源文件   # 压缩（bzip2）
tar -xvf 文件.tar              # 解压并显示详细过程
```

### 参数说明
| 参数 | 含义 |
|------|------|
| `-c` | 创建归档 |
| `-x` | 解压归档 |
| `-z` | 使用 gzip 压缩 |
| `-j` | 使用 bzip2 压缩 |
| `-f` | 指定文件名 |
| `-v` | 显示详细过程 |
| `-t` | 查看归档内容 |

### 实际应用
```bash
# 压缩目录
tar -czf backup.tar.gz ~/documents/

# 解压
tar -xzf backup.tar.gz

# 解压到指定目录
tar -xzf backup.tar.gz -C /tmp/

# 查看归档内容
tar -tzf backup.tar.gz

# 压缩多个文件
tar -czf archive.tar.gz file1.txt file2.txt file3.txt

# 解压 .tar.bz2
tar -xjf archive.tar.bz2
```

## 12. `diff` — 比较文件差异

### 命令格式
```bash
diff 文件1 文件2        # 比较两个文件
diff -u 文件1 文件2     # 统一格式输出（更易读）
diff -r 目录1 目录2     # 递归比较目录
```

### 输出示例
```bash
diff -u old-file new-file
```
```
--- old-file  2026-06-23 10:00:00.000000000 +0800
+++ new-file  2026-06-23 10:30:00.000000000 +0800
@@ -1,3 +1,3 @@
 This is the first line.
-This is the old version.
+This is the new version.
 This is the third line.
```

### 实际应用
```bash
# 比较两个文件
diff file1.txt file2.txt

# 统一格式比较
diff -u file1.txt file2.txt

# 比较目录
diff -r dir1/ dir2/
```

## 14. `wc` — 统计文件

### 命令格式
```bash
wc 文件              # 显示行数、单词数、字符数
wc -l 文件           # 只显示行数
wc -w 文件           # 只显示单词数
wc -c 文件           # 只显示字节数
```

### 实际应用
```bash
# 统计行数
wc -l script.sh

# 统计文件数
ls | wc -l

# 统计所有 Markdown 文件的行数
wc -l *.md

# 统计代码行数（排除空行）
grep -v "^$" main.py | wc -l
```

## 15. `cat` / `less` / `head` / `tail` — 查看文件内容

### 命令对比
| 命令 | 用途 | 适用场景 |
|------|------|----------|
| `cat` | 显示全部内容 | 小文件 |
| `less` | 分页浏览 | 大文件 |
| `head` | 显示开头部分 | 预览文件开头 |
| `tail` | 显示结尾部分 | 查看最新内容 |

### 实际应用
```bash
# 查看小文件
cat /etc/os-release

# 分页查看大文件
less /var/log/syslog
# 在 less 中：Space=下一页，b=上一页，q=退出，/=搜索

# 查看前10行
head -10 file.txt

# 查看后20行
tail -20 file.txt

# 实时查看文件变化（最常用）
tail -f /var/log/syslog
tail -f log.txt
```

## ⚠️ 安全提醒

1. **`rm -rf` 极其危险**：使用前务必确认路径正确，建议先用 `ls` 查看。
2. **`find -delete` 同样危险**：使用前先用 `-print` 预览要删除的文件。
3. **`cp -r` 会覆盖目标文件**：可用 `-i` 参数防止误覆盖。
4. **`mv` 会覆盖目标文件**：可用 `-i` 参数防止误覆盖。
5. **软链接要确保源文件存在**：否则链接会失效。
6. **压缩前确认文件大小**：避免压缩失败或磁盘空间不足。
