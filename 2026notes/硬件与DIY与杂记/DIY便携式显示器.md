# DIY 便携式显示器 — 从零搭建完全指南

## 一句话定位

利用**淘汰的笔记本屏幕**或**市售 LCD 面板** + **驱动板** + **外壳**，组装一台 HDMI/USB-C 输入的便携显示器，成本远低于市售成品。

---

## 核心组件

一台 DIY 便携显示器由四部分组成：

| 组件 | 作用 | 预算范围 |
|------|------|----------|
| **LCD 面板** | 显示核心 | 闲置 ≈ 0 元；全新面板 ¥100-500 |
| **驱动板（Controller Board）** | 将 HDMI/DP/USB-C 信号转为面板可识别的 LVDS/eDP | ¥40-150 |
| **电源模块** | 供电（5V/12V） | ¥10-60 |
| **外壳** | 保护 + 便携 | ¥0（3D 打印）~ ¥50（亚克力） |

---

## 第一步：获取 LCD 面板

### 来源一：淘汰的笔记本屏幕

这是最经济的选择。但需确认面板型号：

1. 拆下笔记本屏幕，查看**背面标签**
2. 记录面板型号，例如 `LP156WF4-SPB1`、`B156HAN01.2`
3. 用型号在屏库网（panelook.com）或淘宝搜索规格：
   - **接口类型**：LVDS（30-pin / 40-pin）或 eDP
   - **分辨率**：1920×1080、2560×1440 等
   - **通道数**：单通道 / 双通道 LVDS

### 来源二：市售全新面板

| 尺寸 | 典型分辨率 | 面板预算 | 总预算（含驱动板+外壳） |
|------|-----------|---------|------------------------|
| 7 英寸 | 1024×600 | ¥80-120 | ¥150-200 |
| 10.1 英寸 | 1280×800 | ¥120-180 | ¥200-300 |
| 13.3 英寸 | 1920×1080 | ¥180-300 | ¥300-450 |
| 15.6 英寸 | 1920×1080 | ¥200-400 | ¥350-600 |

---

## 第二步：选择驱动板（最关键步骤）

驱动板是 DIY 成败的核心。**必须根据面板型号逐一匹配**。

### 接口匹配规则

| 面板接口 | 需要购买的驱动板类型 |
|----------|---------------------|
| LVDS 30-pin 单通道 | LVDS 单通道驱动板 |
| LVDS 30-pin 双通道 | LVDS 双通道驱动板（带宽 ≥ 面板像素时钟） |
| LVDS 40-pin | LVDS 40-pin 驱动板 |
| eDP 30-pin | eDP 驱动板 |

### 关键参数验证

- **双通道 LVDS 带宽**：891 Mbps（单通道 445 Mbps），必须 ≥ 面板像素时钟
  - 例如 1920×1080 @ 60Hz ≈ 148.5 MHz，双通道可用，单通道不足
- **电压**：常见 3.3V / 5V / 12V，驱动板输出电压必须匹配面板背光供电
- **Pin 定义**：LVDS 30-pin 含 CLK+/CLK- 差分信号对，定义错误会导致无显示

### 购买渠道

淘宝搜索：「面板型号 + 驱动板」（如 `LP156WF4 驱动板`），商家通常已烧录对应固件。

### 输入接口选择

| 驱动板输入 | 适用场景 |
|-----------|----------|
| HDMI | 通用，连接电脑 / 游戏机 / 树莓派 |
| VGA | 老旧设备 |
| USB-C（DP Alt Mode） | 一根线同时传视频 + 供电 |
| DP | 高性能 PC |

---

## 第三步：供电方案

### 方案一：USB 5V 供电（推荐）

适合小尺寸面板（≤15.6 英寸）：

```
USB 5V → 升压模块（5V→12V）→ 驱动板 + 面板背光
```

### 方案二：12V 直流供电

适合大尺寸或高亮度面板：

```
12V DC 适配器 → 驱动板 → 面板
```

### 方案三：内置电池

```
锂电池（3.7V）× 2~3 串联 → 充放电保护板 → 升压模块 → 驱动板
```

适合做真正便携的设备，但增加了充电管理和安全设计的复杂度。

---

## 第四步：外壳设计

### 3D 打印（推荐）

- 设计分成**前面板**和**后盖**两部分
- 后盖预留 HDMI、USB-C、按键孔位
- 前面板开槽露出屏幕显示区域

### 亚克力 / 木板

- 激光切割或手工切割
- 比亚克力层叠粘合：面板层 → 驱动板层 → 后盖层

### 关键开孔清单

| 开孔 | 位置 |
|------|------|
| HDMI 接口 | 侧面或背部 |
| USB-C（供电） | 侧面 |
| OSD 按键（菜单/上下/确认） | 侧面或背部 |
| 扬声器格栅（如有） | 正面或侧面 |
| VESA 安装孔（可选） | 背部 |

---

## 第五步：组装与调试

### 组装顺序

1. 连接面板排线到驱动板（屏线，**务必小心插入，不可用力**）
2. 连接背光供电线
3. 连接 OSD 按键板
4. 接 HDMI / USB-C 信号源
5. **先通电测试显示正常**，再装入外壳

### 常见故障排查

| 现象 | 可能原因 | 解决方法 |
|------|---------|----------|
| 黑屏无背光 | 背光供电未接或电压不对 | 检查背光线序和电压 |
| 白屏有背光 | 屏线接触不良或 pin 定义错误 | 重新插拔屏线 |
| 花屏/条纹 | 驱动板固件不匹配 | 联系卖家重烧固件 |
| 画面闪烁 | 供电不足 | 更换更高功率电源 |
| 无信号 | HDMI 线或接口问题 | 换线测试 |

| 维修性 | 高（模块化） | 低 |

## 进阶玩法

- **触摸屏**：选购带触摸层的面板 + 触摸驱动板，实现触控
- **内置音箱**：驱动板通常自带音频功放，接两只小喇叭即可
- **多屏阵列**：多块面板 + USB 显卡/DisplayLink，搭建便携多屏工作站
- **摄影监视器**：加装遮光罩 + 相机热靴支架，成本远低于专业监视器
