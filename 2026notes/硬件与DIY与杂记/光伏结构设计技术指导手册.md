## 第一章 总则

### 1.1 编制目的与意义

随着全球能源转型的加速推进，光伏发电作为清洁能源的重要组成部分，近年来在全球范围内实现了爆发式增长。中国作为全球最大的光伏制造和应用市场，截至2025年底，光伏累计装机容量已突破800GW，年度新增装机超过200GW。光伏支架作为光伏发电系统的关键支撑结构，其设计质量直接关系到光伏电站的安全性、可靠性和经济效益。

本手册的编制目的如下：

**第一，规范设计行为。** 光伏支架结构设计涉及结构力学、材料科学、工程地质、气象学、电气工程等多学科知识，专业性强、技术复杂度高。通过系统化的技术指导，使设计人员准确把握设计要点，规范设计流程，减少设计失误。

**第二，统一技术标准。** 当前光伏支架设计领域存在多种技术路线和设计方法，部分设计人员对标准规范的理解存在偏差。本手册以现行国家及行业标准为依据，系统梳理各项技术要求，促进设计标准的统一应用。

**第三，提高设计效率。** 本手册将分散于多本规范中的技术要求进行整合，提供完整的设计参数和计算方法，方便设计人员快速查阅和应用，提高设计工作效率。

**第四，促进技术创新。** 在总结传统设计经验的基础上，本手册融入了柔性支架、跟踪支架、BIPV等新型结构形式的设计方法，为技术创新提供技术支撑。

**第五，指导工程实践。** 本手册不仅包含规范性的技术要求，还纳入了大量工程实践中积累的经验数据和常用行话，帮助技术人员更好地理解现场沟通语言，缩短理论与实践的差距。

### 1.4 设计基本要求

光伏支架结构设计应满足以下基本要求：

#### 1.4.1 安全性要求

安全性是结构设计的首要要求，具体包括：

1. **强度要求**：所有结构构件及其连接节点在规定的荷载作用下，其承载力应满足要求，不发生破坏。
2. **刚度要求**：结构在荷载作用下产生的变形应控制在允许范围内，不影响光伏组件的正常工作和支架的使用功能。
3. **稳定性要求**：结构在荷载作用下应保持整体稳定和局部稳定，不发生倾覆、滑移、屈曲等失稳破坏。

#### 1.4.2 适用性要求

1. 支架结构应满足光伏组件、逆变器、汇流箱等设备的安装要求，提供足够的安装空间和操作空间。
2. 支架结构应满足运行维护的要求，为检修人员提供安全的工作通道和操作平台。
3. 支架倾角应满足光伏组件最佳发电倾角的要求（或可调节至最佳倾角）。
4. 跟踪支架应保证跟踪精度和运行可靠性。

#### 1.4.3 耐久性要求

1. 钢支架应进行有效的防腐处理，在设计使用年限内不应发生影响结构安全的腐蚀。
2. 铝合金支架应进行有效的表面处理，保证其耐腐蚀性能。
3. 基础混凝土应满足耐久性要求，在有冻融、侵蚀性地下水等不利条件下应采取相应措施。
4. 连接件、紧固件应采用不锈钢或经过有效防腐处理的钢材。

#### 1.4.4 经济性要求

1. 在满足安全性、适用性和耐久性的前提下，应尽量降低结构造价。
2. 应通过优化设计，合理选用材料规格和截面尺寸，减少材料用量。
3. 应综合考虑制造、运输、安装和维护的全生命周期成本，选择最优方案。
4. 应根据项目具体情况，合理确定支架类型（固定/跟踪、钢/铝），进行技术经济比较。

#### 1.4.5 环保要求

1. 支架材料和施工工艺应符合环保要求，减少对环境的负面影响。
2. 支架基础施工应尽量减少对地表的扰动，施工完成后应恢复植被。
3. 在水上光伏项目中，应采取措施保护水体环境。
4. 支架材料应优先选用可回收利用的材料。

## 第二章 光伏结构设计基础理论

### 2.1 光伏结构设计原理

光伏结构设计的核心任务是在满足光伏发电功能要求的前提下，确保支架结构在规定的设计使用年限内具有足够的可靠性。这一任务涉及以下基本原理：

#### 2.1.1 结构-荷载-环境耦合作用

光伏支架结构处于自然环境之中，同时承受来自光伏组件和设备的重力荷载，以及风、雪、温度、地震等环境作用。结构设计的本质是建立“结构体系”与“外部作用”之间的平衡关系，即：

结构抗力 ≥ 外部作用效应

其中，结构抗力取决于材料的性能、构件的截面尺寸和连接方式；外部作用效应取决于荷载的大小、分布和组合方式。

#### 2.1.2 光伏结构的特殊性

与一般建筑结构相比，光伏支架结构具有以下特殊性：

**（1）荷载特征不同**

光伏支架结构以风荷载为控制性荷载，而非以重力荷载为主。在大部分地区，风荷载（尤其是负风压/上拔力）是决定支架构件截面和基础尺寸的关键因素。这与常规建筑以竖向荷载为主的特点显著不同。

**（2）结构形式轻巧**

光伏支架属于轻型钢结构或铝合金结构，构件截面小、自重轻，对荷载的敏感性高。结构设计中对稳定性的关注程度高于对强度的关注。

**（3）对变形敏感**

光伏组件对支架变形较为敏感，过大的变形可能导致组件隐裂、发电效率下降，甚至组件破裂。因此，正常使用极限状态的变形控制非常重要。

**（4）环境暴露充分**

光伏支架完全暴露于室外环境中，长期承受风、雨、雪、紫外线、温度变化等自然因素的作用，防腐和耐久性设计尤为重要。

**（5）设计使用年限短**

光伏支架的设计使用年限通常为25年，而一般建筑结构为50年。这一特点既降低了部分设计标准（如疲劳），也对材料耐久性提出了与25年相匹配的要求。

#### 2.1.3 光伏结构设计的核心问题

光伏结构设计需要解决以下几个核心问题：

1. **荷载确定问题**：如何准确确定作用于支架上的各种荷载，尤其是风荷载和雪荷载。
2. **结构选型问题**：在固定支架、跟踪支架、刚性支架、柔性支架等多种方案中选择最优的结构形式。
3. **构件设计问题**：如何确定构件的截面尺寸和材料规格，使其满足强度、稳定和变形要求。
4. **节点设计问题**：如何设计连接节点，使荷载能够安全、可靠地传递。
5. **基础设计问题**：如何设计支架基础，使其满足承载力和变形要求。
6. **防腐设计问题**：如何选择合适的防腐措施，保证结构在设计使用年限内的耐久性。

### 2.2 结构可靠度设计方法

#### 2.2.1 可靠度的基本概念

结构可靠度是指结构在规定的时间内、在规定的条件下，完成预定功能的概率。结构可靠度设计方法的核心思想是承认荷载和结构抗力都具有随机性，通过概率分析来保证结构的安全性。

**结构功能函数**可表示为：

Z = R - S

其中，R为结构抗力（随机变量），S为荷载效应（随机变量）。

- 当 Z > 0 时，结构处于可靠状态；
- 当 Z = 0 时，结构处于极限状态；
- 当 Z < 0 时，结构处于失效状态。

结构的失效概率 P_f = P(Z < 0)，可靠概率 P_s = 1 - P_f。

#### 2.2.2 可靠指标

在实际工程中，直接计算失效概率较为困难，通常采用可靠指标β来度量结构的可靠度：

β = μ_Z / σ_Z

其中，μ_Z为功能函数的平均值，σ_Z为功能函数的标准差。

可靠指标β与失效概率P_f的对应关系如下：

| β | P_f | 安全等级 |
|---|-----|---------|
| 2.0 | 2.28×10⁻² | 较低 |
| 3.0 | 1.35×10⁻³ | 一般 |
| 3.5 | 2.33×10⁻⁴ | 较高 |
| 4.0 | 3.17×10⁻⁵ | 很高 |
| 4.2 | 1.33×10⁻⁵ | 极高 |

GB 50068《建筑结构可靠度设计统一标准》规定的目标可靠指标，对于安全等级为三级的光伏支架，其可靠指标β不小于3.2（对应于约25年的设计使用年限）。

#### 2.2.3 设计方法的发展历程

结构设计方法经历了以下发展阶段：

**（1）容许应力法（20世纪50年代以前）**

要求结构在标准荷载作用下的最大应力不超过材料的容许应力（屈服强度除以安全系数）。该方法将荷载和抗力均视为确定性值，未考虑其随机性，安全系数取值主观性较强。

**（2）破损阶段设计法（20世纪50-60年代）**

以结构破坏时的承载力作为设计依据，引入荷载系数和材料强度系数。该方法比容许应力法更科学，但仍未充分考虑荷载和抗力的变异性。

**（3）极限状态设计法（20世纪70年代至今）**

以概率理论为基础的极限状态设计法，采用分项系数来表达荷载和抗力的不确定性。我国现行结构设计标准均采用该方法。

#### 2.2.4 分项系数设计法

本手册采用以概率理论为基础的极限状态设计法，通过分项系数来考虑各种不确定性。设计表达式为：

γ₀ S ≤ R

其中：
- γ₀——结构重要性系数
- S——荷载效应组合设计值（由各荷载分项系数组合而成）
- R——结构构件承载力设计值（由材料分项系数折减后的强度计算）

分项系数的取值反映了对荷载和抗力不确定性的综合考虑。荷载分项系数越大，说明该荷载的变异性越大或其对结构的安全性影响越大；材料分项系数越大，说明材料强度的离散性越大。

### 2.3 极限状态设计法

#### 2.3.1 极限状态的定义

结构的极限状态是指结构或结构构件能够满足设计规定的某一功能要求的临界状态。超过这一状态，结构便不能满足该功能要求。

结构设计应将结构控制在可靠的极限状态之内，即结构在各种可能荷载作用下的响应不应超过规定的极限状态。

#### 2.3.2 承载能力极限状态

承载能力极限状态对应于结构或结构构件达到最大承载力、出现破坏、失稳或倾覆等状态。具体包括：

1. **强度破坏**：构件截面应力达到材料强度极限，发生屈服或断裂。
2. **失稳破坏**：构件或结构整体丧失稳定性，发生屈曲。
3. **倾覆**：结构整体绕某一边缘转动而倾倒。
4. **滑移**：结构沿基础底面发生水平滑动。
5. **疲劳破坏**：在反复荷载作用下发生疲劳断裂。

承载能力极限状态是结构设计中最基本的验算内容，所有结构构件和连接节点都必须进行承载能力极限状态验算。

在承载能力极限状态设计中，荷载效应应采用基本组合，按下式计算：

S_d = γ_G S_Gk + γ_Q S_Qk + Σ γ_Q ψ_c S_Qik

式中各符号含义见荷载组合章节。

#### 2.3.3 正常使用极限状态

正常使用极限状态对应于结构或结构构件达到正常使用或耐久性能的某项规定限值的状态。具体包括：

1. **变形过大**：构件挠度或位移超过允许值，影响设备正常使用或外观。
2. **振动**：结构振动幅度或频率超过允许值，影响设备工作或人体舒适度。
3. **局部损坏**：出现裂缝、腐蚀等影响耐久性的局部损坏。

正常使用极限状态设计时，荷载效应应采用标准组合：

S_d = S_Gk + S_Qk + Σ ψ_c S_Qik

光伏支架结构中最主要的正常使用极限状态验算是挠度验算，即确保主梁和檩条在荷载作用下的最大挠度不超过允许值。

#### 2.3.4 两种极限状态的关系

承载能力极限状态和正常使用极限状态是结构设计不可分割的两个方面：

- **承载能力极限状态是“安全性”的保证**——确保结构不倒塌、不破坏。
- **正常使用极限状态是“适用性”和“耐久性”的保证**——确保结构在正常使用条件下满足功能要求。

通常情况下，满足承载能力极限状态要求的结构不一定能满足正常使用极限状态的要求（如构件强度足够但刚度不足）；反之亦然。设计中应同时验算两种极限状态。

### 2.4 荷载与作用分类体系

#### 2.4.1 荷载与作用的定义

- **作用**：施加在结构上的力或引起结构变形、内力的其他因素（如温度变化、地基沉降等）。
- **荷载**：狭义的作用，指直接施加在结构上的力。

在光伏支架结构设计中，需要考虑的“作用”包括力荷载（如风、雪、自重）和非力作用（如温度效应、地震作用等）。

#### 2.4.2 荷载分类

根据GB 50009《建筑结构荷载规范》，荷载按随时间的变异特性可分为三类：

**（1）永久荷载（恒荷载）**

在结构设计使用年限内，其量值不随时间变化或其变化与平均值相比可以忽略不计的荷载。包括：
- 结构自重（支架、组件、设备等）
- 土压力
- 预应力

永久荷载的变异性较小，其分项系数取值也相对较小。

**（2）可变荷载（活荷载）**

在结构设计使用年限内，其量值随时间变化且其变化与平均值相比不可忽略的荷载。包括：
- 风荷载
- 雪荷载
- 施工检修荷载
- 温度荷载（部分情况下）

可变荷载的变异性较大，其分项系数取值相对较大。

**（3）偶然荷载**

在结构设计使用年限内不一定出现，而一旦出现其量值很大且持续时间很短的荷载。包括：
- 地震作用
- 爆炸力
- 撞击力

#### 2.4.3 荷载的代表值

荷载的代表值是用于结构设计的荷载值，包括：

**（1）标准值**

荷载的基本代表值，是结构设计时采用的荷载基本值。对于永久荷载，标准值通常取平均值为代表值；对于可变荷载，标准值通常取具有一定保证率的统计特征值（如50年一遇的最大值）。

**（2）组合值**

当多个可变荷载同时作用时，考虑到各荷载最大值同时出现的概率较小，采用组合值系数对标准值进行折减后的荷载值。

**（3）频遇值和准永久值**

用于正常使用极限状态设计的荷载代表值，分别对应于荷载的频遇值和准永久值。

### 2.5 结构分析模型建立

#### 2.5.1 结构分析的基本任务

结构分析的目的是确定结构在荷载作用下的内力和变形，为构件设计和节点设计提供依据。基本任务包括：

1. 建立结构的力学模型；
2. 计算荷载效应（内力、变形）；
3. 验算结构的承载能力和正常使用性能。

#### 2.5.2 结构计算简图的建立

建立合理的计算简图是结构分析的关键步骤，应注意以下原则：

1. **合理简化**：在保证计算精度的前提下，对结构进行合理简化，减少计算工作量。
2. **传力明确**：计算简图应能真实反映结构的传力路径。
3. **边界条件正确**：支座条件、约束方式应与实际构造相符。
4. **荷载传递准确**：荷载的施加方式和作用点应与实际情况一致。

#### 2.5.3 光伏支架的典型计算模型

**（1）单榀刚架模型**

对于固定支架，常用的计算模型是取横向一榀刚架进行分析。该模型将支架简化为由立柱、主梁和斜撑组成的平面刚架，荷载通过檩条以集中力或均布力的形式传递到主梁上。

**（2）空间刚架模型**

对于大型光伏支架或受力复杂的情况，宜采用空间刚架模型进行分析。空间模型能更准确地反映结构的整体受力性能，尤其适用于跟踪支架和不规则布置的支架。

**（3）索-桁架模型**

对于柔性支架，应采用索-桁架模型进行分析。该模型将主索视为柔性构件，考虑几何非线性效应，通过迭代计算确定索的变形和内力。

#### 2.5.4 分析软件与工具

当前常用的光伏支架结构分析软件包括：

| 软件名称 | 适用场景 | 特点 |
|---------|---------|------|
| PKPM | 常规钢结构设计 | 国内主流，符合中国规范 |
| SAP2000 | 通用结构分析 | 功能强大，适用于复杂结构 |
| MIDAS Gen | 通用结构分析 | 界面友好，非线性分析能力强 |
| ANSYS | 有限元分析 | 精度高，适用于特殊结构分析 |
| ABAQUS | 非线性分析 | 非线性处理能力强 |
| 3D3S | 钢结构专业软件 | 适用于轻钢和空间结构 |
| PVsyst | 光伏系统设计 | 主要用于发电量模拟，兼有结构布置功能 |

## 第三章 光伏阵列与组件术语详解

### 3.1 光伏组件（PV Module）

#### 3.1.1 定义与基本概念

**光伏组件（Photovoltaic Module）** ，又称太阳电池组件或太阳能电池板，是指具有封装及内部联接的、能单独提供直流电的输出、最小不可分割的太阳电池组合装置。

光伏组件是光伏发电系统的最基本发电单元。一块光伏组件由若干片太阳电池片通过串并联连接，并经过层压封装而成。常用的光伏组件结构包括：

1. **上层**：低铁超白钢化玻璃（透光率≥91.5%）
2. **中层**：EVA（乙烯-醋酸乙烯酯共聚物）胶膜封装电池片
3. **下层**：背板（或玻璃，用于双玻组件）
4. **边框**：铝合金边框

#### 3.1.2 光伏组件的分类

按电池材料分类：

| 类型 | 转换效率 | 特点 | 应用 |
|------|---------|------|------|
| 单晶硅组件 | 21%-25% | 效率高、外观好 | 高端项目、户用 |
| 多晶硅组件 | 17%-20% | 性价比高 | 大型地面电站 |
| 薄膜组件 | 10%-15% | 弱光性好、柔性 | BIPV、特殊场景 |

按封装方式分类：

- **单玻组件**：正面玻璃+背面背板，是市场主流，重量轻、成本低。
- **双玻组件**：正面和背面均为玻璃，抗PID性能好，透光性好，适用于BIPV和阳光房。
- **柔性组件**：采用柔性材料封装，可弯曲，适用于曲面屋顶和特殊造型。

按应用场景分类：

- **标准组件**：用于地面电站和常规屋顶。
- **透光组件**：用于阳光房、光伏幕墙等需要采光的场景。
- **彩色组件**：用于建筑立面，外观与建筑协调。

#### 3.1.3 组件的机械参数

光伏组件的机械参数是支架结构设计的重要输入条件，主要包括：

| 参数 | 典型值 | 说明 |
|------|------|------|
| 外形尺寸 | 2094×1038×30mm（72片） | 标准规格 |
| 重量 | 25-32kg/块 | 影响支架荷载 |
| 承载能力（正面） | ≥5400Pa | 对应风荷载/雪荷载 |
| 承载能力（背面） | ≥2400Pa | 对应负风压 |
| 工作温度 | -40℃~+85℃ | 环境影响 |
| 冰雹冲击 | 直径25mm，速度23m/s | 抗冰雹能力 |

在进行支架结构设计时，应收集组件供应商提供的详细技术参数，包括组件的外形尺寸、重量、允许荷载、安装孔位置等。

### 3.2 光伏组件串（PV String）

#### 3.2.1 定义

**光伏组件串（PV String）** ，简称组件串或组串，是指在光伏发电系统中，将若干个光伏组件通过电缆串联连接，形成具有一定直流输出电压的电路单元。

光伏组件串是光伏方阵的基本电气组成单元。多个组件串通过并联方式接入直流汇流箱或逆变器。

#### 3.2.2 组件串的电气参数

组件串的输出电压为各组件输出电压之和，输出电流等于单个组件的输出电流（在串联电路中电流相等）。典型参数：

- 串联组件数量：通常为15-22块（取决于逆变器输入电压范围）
- 输出直流电压：600-1500V（系统电压等级）
- 输出直流电流：8-15A（取决于组件规格）

#### 3.2.3 组件串与支架设计的关系

组件串的布置方式对支架结构设计有以下影响：

1. **阵列排布**：组件串的排布方式决定了光伏方阵的几何布置，进而影响支架的平面布局。
2. **电缆敷设**：组件串之间的电缆连接需要在支架上预留走线槽或固定点。
3. **接线盒位置**：组件接线盒的位置影响支架构件的避让设计。
4. **安全间距**：组件串之间需要预留足够的间距，以便于安装和维护，同时也满足防火间距的要求。

### 3.3 光伏方阵（PV Array）

#### 3.3.1 定义

**光伏方阵（PV Array）** ，又称光伏阵列，是指由若干个太阳电池组件或太阳电池板在机械和电气上按一定方式组装在一起，并且有固定的支撑结构而构成的直流发电单元。

光伏方阵是光伏发电站中基本的直流发电单元。一个光伏方阵通常包含若干个组件串，通过直流汇流箱汇集后，经逆变器转换为交流电并入电网。

#### 3.3.2 方阵的组成

一个完整的光伏方阵包括：

1. **光伏组件**：发电的核心单元
2. **支架系统**：支撑和固定组件
3. **直流汇流箱**：汇集各组件串的直流电
4. **电缆与连接器**：电气连接
5. **接地系统**：防雷接地保护

#### 3.3.3 方阵的几何参数

光伏方阵的几何参数是支架结构设计的基本依据：

| 参数 | 定义 | 设计考虑 |
|------|------|---------|
| 方阵倾角 | 组件安装面与水平面的夹角 | 影响发电量和风荷载 |
| 方阵方位角 | 组件面向的方向（正南为0°） | 影响发电量 |
| 行间距 | 前后排方阵之间的距离 | 避免阴影遮挡 |
| 列间距 | 同排相邻方阵之间的距离 | 安装和维护空间 |
| 离地高度 | 组件最低点距地面高度 | 影响通风和运维 |

#### 3.3.4 方阵布置原则

光伏方阵的布置应遵循以下原则：

1. **朝向一致性**：同一方阵内各组件的朝向应保持一致，确保发电量最大化。
2. **避免遮挡**：前后排方阵之间应保持足够的间距，避免在冬至日前后产生阴影遮挡（通常要求冬至日上午9:00至下午3:00无遮挡）。
3. **安全间距**：方阵与方阵之间、方阵与边界之间应预留安全间距。
4. **电缆路径优化**：尽量减少电缆长度，降低线路损耗。
5. **运维通道**：预留足够的通道，便于设备检修和组件清洗。

### 3.4 容配比（Inverter Loading Ratio）

#### 3.4.1 定义

**容配比（Inverter Loading Ratio，ILR）** ，又称超配比，是指光伏组件总容量（DC，直流侧）与逆变器额定容量（AC，交流侧）的比值。

容配比 = 光伏组件总容量（kWp）/ 逆变器额定容量（kW）

#### 3.4.2 容配比的设计意义

容配比设计是光伏系统设计中的关键技术环节，其意义在于：

1. **补偿系统损耗**：光伏系统从组件到电网存在多种损耗（电缆损耗、逆变器损耗、温升损耗、灰尘遮挡损耗等），适当提高容配比可以补偿这些损耗，提高逆变器的实际出力。

2. **提高逆变器利用率**：光伏组件的实际输出功率通常低于其标称功率（实际辐照低于标准测试条件STC的1000W/m²），适当超配可以使逆变器在更多时间内运行在额定功率附近，提高设备利用率。

3. **降低度电成本**：在逆变器容量不变的情况下，增加组件容量可提高年发电量，从而降低度电成本（LCOE）。

#### 3.4.3 容配比的合理范围

容配比的取值应根据项目所在地的辐照资源条件、系统损耗、逆变器特性等因素综合确定：

| 地区类型 | 辐照条件 | 推荐容配比 | 说明 |
|---------|---------|-----------|------|
| 一类地区（如西藏、青海） | 辐照高 | 1.05-1.15 | 辐照资源好，超配不宜太高 |
| 二类地区（如新疆、甘肃） | 辐照中等 | 1.15-1.25 | 适中超配 |
| 三类地区（如中东部） | 辐照较低 | 1.25-1.40 | 适当提高超配补偿辐照不足 |

在工程实践中，容配比通常在1.0-1.4之间。需要注意的是，容配比过大会造成逆变器削峰损失增加，经济性反而下降，应通过发电量模拟进行优化计算。

### 3.5 MPPT（最大功率点跟踪）

#### 3.5.1 定义与原理

**MPPT（Maximum Power Point Tracking，最大功率点跟踪）** 是逆变器中的一种智能控制算法，其功能是通过实时调整光伏系统的工作电压或电流，使光伏组件始终工作在最大功率输出点（MPP），从而获取最大的发电量。

光伏组件的输出功率随工作电压的变化呈现非线性关系，存在一个最大功率点。当工作电压从0增加时，输出功率先增加后减小。MPPT算法通过不断扰动工作点并观察功率变化的方向，自动追踪到最大功率点。

#### 3.5.2 MPPT的常用算法

| 算法名称 | 原理 | 特点 |
|---------|------|------|
| 扰动观察法（P&O） | 周期性地改变工作电压，观察功率变化方向 | 算法简单，应用最广 |
| 电导增量法（INC） | 通过比较电导变化率判断最大功率点位置 | 精度较高，适用于辐照快速变化 |
| 恒压法（CVT） | 将工作电压固定为开路电压的某个比例（约0.76倍） | 简单但精度低 |
| 智能算法 | 模糊逻辑、神经网络等 | 精度高但计算量大 |

#### 3.5.3 MPPT与结构设计的关系

MPPT功能本身与结构设计没有直接关系，但以下方面存在间接关联：

1. **组件倾角优化**：MPPT的效率受组件辐照不均匀度的影响，支架倾角设计应尽量保证各组件辐照均匀。
2. **阴影遮挡**：阴影遮挡会导致组件输出曲线出现多个峰值，影响MPPT追踪效果。支架布置应避免组件间的相互遮挡。
3. **组串一致性**：同一MPPT通道内的组件串应具有相近的朝向和倾角，否则会影响MPPT效率。

### 3.6 光伏组件的电气性能参数

#### 3.6.1 标准测试条件（STC）

光伏组件的电气参数通常在标准测试条件下标定：

- 辐照度：1000 W/m²
- 光谱分布：AM1.5
- 电池温度：25°C

在实际运行中，组件的输出参数随辐照和温度的变化而变化，设计时应考虑最不利工况。

#### 3.6.2 关键电气参数

| 参数 | 符号 | 含义 | 典型值（以550W单晶组件为例） |
|------|------|------|---------------------------|
| 额定功率 | P_max | 最大输出功率 | 550Wp |
| 开路电压 | V_oc | 空载时的输出电压 | 49.8V |
| 短路电流 | I_sc | 短路时的输出电流 | 14.0A |
| 最大功率点电压 | V_mpp | 最大功率点对应电压 | 42.0V |
| 最大功率点电流 | I_mpp | 最大功率点对应电流 | 13.1A |
| 组件效率 | η | 光电转换效率 | 21.2% |
| 温度系数（P_max） | β_P | 功率随温度的变化率 | -0.35%/°C |
| 温度系数（V_oc） | β_V | 电压随温度的变化率 | -0.28%/°C |
| 温度系数（I_sc） | β_I | 电流随温度的变化率 | +0.05%/°C |

#### 3.6.3 组串电压计算

在进行组件串设计时，需计算组串的最大开路电压（低温工况）和最大功率点电压（高温工况）：

**低温开路电压**：V_oc_max = V_oc_STC × [1 + β_V × (T_min - 25)]

其中T_min为项目所在地的极端最低温度。

**高温MPP电压**：V_mpp_min = V_mpp_STC × [1 + β_V × (T_max - 25)]

其中T_max为项目所在地的极端最高温度。

组串电压应在逆变器的允许输入电压范围之内。

### 3.7 光伏组件的机械性能参数

#### 3.7.1 机械荷载

光伏组件的机械荷载能力是支架设计的重要依据。典型组件荷载能力如下：

- **正面荷载（压缩）** ：≥5400Pa（对应基本风压约1.5kN/m²，雪荷载约2.0kN/m²）
- **背面荷载（拉伸）** ：≥2400Pa（对应负风压）

在进行支架设计时，传递给组件的荷载不应超过组件的允许荷载。支架构件与组件的连接点应均匀分布，避免应力集中。

#### 3.7.2 安装要求

组件安装的关键要求：

| 项目 | 要求 | 原因 |
|------|------|------|
| 安装孔位置公差 | ≤±1mm | 保证组件安装准确性 |
| 压块接触宽度 | ≥25mm | 避免应力集中 |
| 螺栓扭矩 | 根据规格确定（如M8螺栓为10-15N·m） | 保证连接可靠性 |
| 组件间隙 | 10-20mm | 适应热胀冷缩 |
| 组件表面保护 | 避免划伤、撞击 | 保护电池片 |

#### 3.7.3 组件接线盒

组件接线盒位于组件背面的接线区域，盒内含有旁路二极管，用于防止热斑效应。接线盒的IP防护等级通常为IP67或IP68。

接线盒的位置影响支架檩条布置——檩条应避开接线盒，避免在接线盒处形成应力集中或遮挡。


## 第四章 光伏支架类型与选型指导

### 4.1 支架分类总览

光伏支架是光伏发电系统中用于支撑和固定光伏组件的结构系统。根据不同的分类维度，可对光伏支架进行系统分类：

#### 4.1.1 按运动特性分类

| 类型 | 运动特性 | 市场占比 | 发电增益 |
|------|---------|---------|---------|
| 固定支架 | 固定不变 | 70%-80% | 基准 |
| 可调支架 | 手动调节（2-4次/年） | 5%-10% | +3%~+8% |
| 平单轴跟踪 | 单轴旋转跟踪太阳 | 10%-15% | +10%~+20% |
| 斜单轴跟踪 | 倾斜轴旋转 | 1%-3% | +15%~+25% |
| 双轴跟踪 | 双轴360°旋转 | <1% | +20%~+35% |

#### 4.1.2 按结构形式分类

| 类型 | 结构特征 | 适用场景 |
|------|---------|---------|
| 刚性支架 | 刚性构件（型钢、铝型材） | 绝大多数场景 |
| 柔性支架 | 预应力钢索结构 | 大跨度、复杂地形 |
| 混合支架 | 索+刚性构件组合 | 特殊需求场景 |

#### 4.1.3 按材料分类

| 类型 | 材料 | 特点 |
|------|------|------|
| 钢支架 | Q235B/Q355B热轧或冷弯型钢 | 强度高、成本低、应用最广 |
| 铝合金支架 | 6061/6063铝合金型材 | 耐腐蚀、重量轻、成本较高 |
| 不锈钢支架 | 不锈钢型材 | 耐腐蚀性极好、成本最高 |

#### 4.1.4 按应用场景分类

| 场景 | 支架特点 | 特殊要求 |
|------|---------|---------|
| 地面电站 | 大阵列、标准化 | 基础形式多样 |
| 屋顶分布式 | 轻量化、夹具固定 | 荷载复核、防水 |
| 水上光伏 | 耐腐蚀、桩基/漂浮 | 水位适应、防腐 |
| 农光互补 | 大跨度、高离地 | 农业作业空间 |
| BIPV | 建筑一体化 | 建筑性能、美观 |
| 阳光房/棚 | 透光、美观 | 建筑功能、防水 |

### 4.2 固定支架

#### 4.2.1 定义与特点

固定支架是指光伏组件安装后倾角固定不变的支架类型。其特点包括：

- **结构简单**：由立柱、主梁、檩条、支撑等基本构件组成，无转动部件。
- **可靠性高**：无运动部件，故障率极低，维护量极小。
- **造价低廉**：单位瓦造价最低，是各类支架中最经济的方案。
- **适用范围广**：适用于各种地形和气候条件。
- **设计成熟**：技术最成熟，设计方法和施工工艺最为完善。

#### 4.2.2 固定支架的结构组成

典型的固定支架由以下部件组成：

1. **立柱**：垂直或倾斜设置，下端连接基础，上端连接主梁。是支架的主要竖向承重构件。
2. **主梁**：水平或倾斜设置，与立柱连接，是支架的主要水平承重构件。多个立柱支撑一根主梁。
3. **檩条**：垂直于主梁布置，直接支撑光伏组件，将组件荷载传递至主梁。
4. **斜撑**：连接立柱和主梁或地面，形成三角形稳定结构，提高支架整体刚度。
5. **连接件**：包括螺栓、焊接节点、压块等，用于连接各构件。
6. **基础**：将荷载传递至地基，常见形式有混凝土独立基础、微型桩、螺旋桩等。

#### 4.2.3 固定支架的布置形式

固定支架的布置主要取决于光伏方阵的几何参数：

**（1）倾角确定**

固定支架的倾角通常按以下方法确定：

- **最佳倾角法**：根据项目所在地的纬度，通过PVsyst等软件模拟计算年发电量最大的倾角。一般经验公式为：最佳倾角≈当地纬度×0.76+3.1°（应用于冬季负载偏重的系统）。
- **工程近似法**：对于大型地面电站，倾角通常取当地纬度±5°范围内。

**（2）跨度与间距**

- **纵向跨度**：沿组件长边方向的支架立柱间距，通常为2-4m。
- **横向排距**：前后排支架之间的距离，由阴影遮挡计算确定。

**（3）高度**

- **最低点离地高度**：通常为0.5-1.0m（地面电站），便于通风和运维。
- **对于农光互补项目**：离地高度≥2.5m，满足农业机械通行要求。

#### 4.2.4 固定支架的设计要点

1. **抗风设计**：固定支架的风荷载是最重要的设计荷载，应特别注意负风压（上拔力）的影响，基础设计往往由上拔力控制。

2. **温度效应**：固定支架为露天结构，温度变化大，应设置合理的温度缝或采用柔性连接，减小温度应力。

3. **节点设计**：固定支架主要承受静力荷载，节点可采用焊接或螺栓连接。焊接连接整体性好，但防腐要求高；螺栓连接便于安装，但节点刚度相对较低。

4. **基础选型**：应根据地质条件、施工工期和经济性综合确定基础形式。

### 4.3 可调倾角固定支架

#### 4.3.1 定义与原理

可调倾角固定支架是在标准固定支架的基础上，增加了角度调节机构，使光伏组件的倾角可以根据季节变化进行手动调节的支架类型。

其原理是：太阳的高度角随季节变化，夏季太阳高度角大、冬季太阳高度角小。通过调节组件倾角，使组件在冬季更接近垂直、在夏季更接近水平，从而提高不同季节的发电量。

#### 4.3.2 调节方式与频次

**调节方式**：
- 手动调节：通过插销、弧形槽或丝杆机构，人工调整倾角并固定。
- 电动调节（较少）：通过小型电机驱动调节，但成本较高，经济性不佳。

**调节频次**：
- 一年调节2次：春分和秋分各调节一次，分别在夏季倾角和冬季倾角之间切换。
- 一年调节4次：按四季分别设置倾角。
- 一年调节6次：更精细的调节，发电增益略高，但运维成本增加。

通常推荐一年调节2-4次，在经济性和发电增益之间取得平衡。

#### 4.3.3 发电增益

可调倾角固定支架的发电增益与调节频次和纬度有关：

| 调节频次 | 发电增益（相对固定最优倾角） | 适用范围 |
|---------|---------------------------|---------|
| 2次/年 | +3% ~ +5% | 低纬度地区 |
| 4次/年 | +5% ~ +8% | 中纬度地区 |
| 6次/年 | +6% ~ +10% | 高纬度地区 |

纬度越高，太阳高度角的季节性变化越大，可调倾角的增益越显著。

#### 4.3.4 设计要点

1. **调节机构设计**：调节机构应操作方便、固定可靠，在风荷载作用下不应发生松动或移位。
2. **限位措施**：应设置明确的限位装置，防止倾角超出允许范围。
3. **维护通道**：应预留人工调节的操作空间和安全通道。
4. **防腐要求**：调节机构的运动部件应做特殊防腐处理，防止因锈蚀导致调节功能失效。

### 4.4 平单轴跟踪支架

#### 4.4.1 定义与原理

平单轴跟踪支架是指光伏组件围绕一根水平轴（通常为南北方向）旋转，跟随太阳的东-西运动轨迹（方位角变化）的支架类型。

其工作原理是：太阳在一天中从东到西运动，通过驱动机构使组件绕水平轴旋转，始终保持组件面与太阳光线尽可能垂直（或保持一定的入射角），从而增加组件接收的太阳辐照量。

#### 4.4.2 结构组成

平单轴跟踪支架由以下主要部分组成：

1. **旋转主轴**：通常为圆管或方管，沿南北方向布置，是整个支架的旋转中心。
2. **支撑立柱**：支撑主轴的竖向构件，顶部安装轴承座。
3. **光伏组件安装梁**：垂直于主轴安装，用于固定光伏组件，与主轴同步旋转。
4. **驱动系统**：包括电机、减速器、推杆或回转驱动器，提供旋转动力。
5. **控制系统**：包括控制器、角度传感器、限位开关、通讯模块等，实现智能跟踪控制。
6. **基础**：承受荷载并固定立柱。

#### 4.4.3 技术参数

| 参数 | 典型值 | 说明 |
|------|------|------|
| 跟踪范围 | -60° ~ +60°（东西方向） | 以正南为0° |
| 跟踪精度 | ±1° ~ ±2° | 影响发电效率 |
| 旋转速度 | 0.5-2°/min | 慢速旋转，减少能耗 |
| 驱动功率 | 100-500W | 每台驱动器 |
| 自耗电 | 占发电量的0.5%-1.5% | 运营成本 |
| 大风保护 | 风速>20m/s时归位到0° | 保护结构安全 |

#### 4.4.4 发电增益与适用条件

**发电增益**：
平单轴跟踪支架相对固定支架的发电增益为10%-20%，具体增益取决于纬度、气候条件和跟踪策略。

| 纬度 | 增益范围 | 说明 |
|------|---------|------|
| 低纬度（<20°） | 10%-12% | 太阳高度角大，增益相对较低 |
| 中纬度（20°-35°） | 12%-18% | 增益适中 |
| 高纬度（>35°） | 15%-20% | 季节性变化大，增益较高 |

**适用条件**：
- 适合辐照资源较好的地区（年辐照>1400kWh/m²）
- 适合电价较高的地区，能较快回收跟踪系统的增量投资
- 适合地势平坦、场地开阔的项目
- 不适合多风沙、多冰雹等恶劣气候地区（易损坏跟踪机构）

#### 4.4.5 设计要点

1. **抗风设计**：跟踪支架在大风天气需归位至水平（0°）位置以减少风荷载。风荷载计算时应考虑不同跟踪角度下的受力状态。

2. **驱动系统选型**：驱动系统的功率和扭矩应根据支架的转动惯量和风荷载计算确定，并应有足够的裕量。

3. **控制系统可靠性**：跟踪控制系统应具备故障自诊断、远程监控和手动操作功能，确保在通讯中断等情况下的安全。

4. **地基沉降控制**：跟踪支架对不均匀沉降非常敏感，基础设计应严格控制沉降量。

5. **电缆管理**：跟踪支架的运动部分需要设置电缆拖链或旋转接头，防止电缆在运动中受损。

### 4.5 斜单轴跟踪支架

#### 4.5.1 定义与特点

斜单轴跟踪支架是指光伏组件围绕一根与水平面成一定角度（倾角）的轴旋转的支架类型。其旋转轴通常沿南北方向布置，但轴与水平面之间的夹角（轴倾角）使得组件在旋转过程中同时改变方位角和高度角。

斜单轴跟踪结合了平单轴和双轴的部分优点，在保持单轴结构简单性的同时，获得了更接近双轴的发电增益。

#### 4.5.2 与平单轴的对比

| 项目 | 平单轴 | 斜单轴 |
|------|-------|-------|
| 轴方向 | 水平 | 倾斜（与水平成α角） |
| 旋转范围 | -60°~+60° | -45°~+45°（受轴倾角影响） |
| 结构复杂度 | 较低 | 中等 |
| 发电增益 | 10%-20% | 15%-25% |
| 成本 | 基准 | 比平单轴高5%-15% |
| 适用纬度 | 低中纬度 | 中高纬度（>25°） |

#### 4.5.3 轴倾角的确定

轴倾角α是斜单轴支架的关键设计参数，通常取值为当地纬度的0.6-0.8倍。例如：

- 北纬30°地区：轴倾角取18°-24°
- 北纬40°地区：轴倾角取24°-32°

轴倾角的选择需进行发电量模拟优化，同时需考虑结构实现的可能性。

#### 4.5.4 设计要点

斜单轴支架的设计要点与平单轴类似，但需注意：
- 倾斜轴的结构受力更为复杂，需考虑轴向分力
- 基础设计需考虑倾斜轴产生的偏心荷载
- 安装和调试难度高于平单轴

### 4.6 双轴跟踪支架

#### 4.6.1 定义与原理

双轴跟踪支架是指具有两个独立旋转轴（一个水平轴和一个垂直轴）的支架类型，能够同时跟踪太阳的方位角变化和高度角变化，使光伏组件面始终垂直于太阳光线。

**两个旋转轴**：
- **方位角轴（垂直轴）** ：使组件在水平面内旋转，跟随太阳的东西向运动。
- **高度角轴（水平轴）** ：使组件在垂直面内旋转，跟随太阳的高度角变化。

#### 4.6.2 结构形式

双轴跟踪支架主要有两种结构形式：

| 形式 | 结构特点 | 适用场景 |
|------|---------|---------|
| T形双轴 | 垂直立柱顶端设水平旋转平台，平台上设倾斜旋转轴 | 大功率组件，地面电站 |
| 立柱式双轴 | 单立柱支撑，通过两个方向的角度调节机构实现跟踪 | 小型系统，聚光光伏 |

T形双轴跟踪支架是大型光伏电站中双轴跟踪的主流形式。

#### 4.6.3 发电增益与成本

双轴跟踪支架的发电增益最高，可达固定支架的20%-35%。但成本也最高：

| 项目 | 数值 |
|------|------|
| 发电增益 | +20% ~ +35% |
| 增量成本（相对固定支架） | +50% ~ +100% |
| 年维护成本 | 较高（运动部件多） |

由于双轴跟踪支架的成本较高，通常仅在以下条件下具有经济性：
- 高辐照地区（年辐照>1800kWh/m²）
- 高电价地区
- 聚光光伏（CPV）系统
- 土地成本极高的地区

#### 4.6.4 设计要点

1. **抗风设计**：双轴跟踪支架的风荷载特性复杂，不同跟踪角度下的风荷载差异大，需进行多角度风荷载分析。
2. **驱动系统**：需要两套独立的驱动系统（方位角驱动和高度角驱动），控制精度要求高。
3. **基础设计**：双轴跟踪支架的荷载具有显著的方向性和变化性，基础需同时承受水平力、竖向力和弯矩。
4. **自锁功能**：驱动系统应具备自锁功能，在停电或故障时能保持组件位置不变。

### 4.7 刚性支架

#### 4.7.1 定义

刚性支架是指由传统刚性构件（如型钢、钢管、铝合金型材等）通过焊接或螺栓连接组成的支架结构，其承载能力主要依靠构件自身的抗弯刚度和强度。

刚性支架是光伏支架中最基本、应用最广泛的类型，适用于绝大多数光伏项目。

#### 4.7.2 结构特点

刚性支架的结构特点如下：

1. **节点刚性**：构件之间的连接节点通常设计为刚接或铰接，节点刚度较大。
2. **整体性好**：各构件通过节点连接形成整体结构，共同抵抗荷载。
3. **传力路径清晰**：荷载由组件→檩条→主梁→立柱→基础，传力路径明确。
4. **构件截面多样**：可采用H型钢、C型钢、方钢管、圆钢管等截面形式。

#### 4.7.3 常用截面形式

| 截面形式 | 优点 | 适用构件 |
|---------|------|---------|
| H型钢 | 双向刚度大，加工方便 | 主梁、立柱 |
| C型钢（冷弯） | 截面利用率高，重量轻 | 檩条、支撑 |
| 方钢管 | 抗扭性能好，外观整齐 | 立柱、主梁 |
| 圆钢管 | 各向同性，风阻小 | 立柱、支撑 |
| 角钢 | 规格齐全，价格低 | 支撑、连接件 |

#### 4.7.4 设计计算要点

刚性支架的设计计算应按照GB 50017《钢结构设计标准》和NB/T 10115-2018的要求进行，主要内容包括：
1. 建立结构计算模型（通常采用一阶弹性分析）
2. 计算各种荷载组合下的内力和变形
3. 按承载能力极限状态验算构件强度和稳定
4. 按正常使用极限状态验算构件挠度
5. 设计连接节点

### 4.8 柔性支架（索结构光伏支架）

#### 4.8.1 定义与背景

柔性支架是指采用预应力钢索作为主要承重构件，通过拉索的张拉形成大跨度支撑结构的光伏支架类型。其核心特征是用“索”替代传统的刚性“梁”作为主要水平承重构件。

柔性支架近年来发展迅速，特别适用于以下场景：
- 山地、沟壑等复杂地形
- 需要大跨度的农光互补项目
- 污水处理厂、停车场等需要大空间净空的场所
- 对地表扰动要求低的生态敏感区

2025年发布的NB/T 11813-2025《索结构光伏支架技术规程》为柔性支架的设计和施工提供了标准依据。

#### 4.8.2 结构组成

柔性支架由以下部分组成：

1. **柔性系统**：
   - 预应力主索：承担主要竖向荷载的钢索
   - 稳定索：控制结构侧向位移的钢索
   - 防风索：抵抗风荷载的附加索
   - 组件固定夹具：将光伏组件固定在索上的专用夹具

2. **支架系统**：
   - 基础：承受拉索拉力并传递至地基
   - 钢立柱：支撑拉索端部的竖向构件
   - 钢梁：连接立柱并传递水平力
   - 支撑体系：保证整体稳定的斜撑

#### 4.8.3 跨度分类

根据NB/T 11813-2025，柔性支架按跨度分类如下：

| 跨度类别 | 跨度范围 | 设计特点 |
|---------|---------|---------|
| 小跨度 | L ≤ 25m | 索力较小，结构相对简单 |
| 中跨度 | 25m < L ≤ 45m | 常规柔性支架，应用最广 |
| 大跨度 | 45m < L ≤ 60m | 索力较大，需特殊设计 |
| 超跨度 | L > 60m | 需进行专项论证和试验验证 |

#### 4.8.4 受力特点

柔性支架与传统刚性支架的受力特点有本质区别：

| 项目 | 刚性支架 | 柔性支架 |
|------|---------|---------|
| 传力方式 | 弯矩+剪力 | 轴力（拉力） |
| 荷载-变形关系 | 线性 | 非线性（几何非线性） |
| 刚度来源 | 构件截面刚度（EI） | 索的预应力（张力刚度） |
| 变形特性 | 变形较小 | 变形较大 |
| 稳定性问题 | 压杆稳定 | 驰振、颤振（风致振动） |

柔性支架的设计必须考虑几何非线性效应——索的刚度依赖于其张力，张力越大，刚度越大。这一特性使得柔性支架的荷载-变形关系呈非线性。

#### 4.8.5 设计要点

**（1）索的初始预应力**

初始预应力的取值是柔性支架设计的关键。预应力过小，则结构刚度不足，变形过大；预应力过大，则索力过大，基础和锚锭设计困难。

初始预应力取值应综合考虑索结构形式、支承结构刚度、合理预应力值等因素，通过试算确定。一般原则是使索在常态荷载下保持张力，不发生松弛。

**（2）风荷载分析**

柔性支架对风荷载十分敏感，风致振动（包括驰振、涡激振动、颤振等）是需要特别注意的问题。设计时应：
- 考虑脉动风效应
- 进行风致振动分析
- 设置防风索或阻尼器抑制振动
- 大风工况下可考虑将组件调整至最小迎风角

**（3）锚锭系统**

索的端部锚锭系统是柔性支架的关键节点，将巨大索力传递至基础和地基。锚锭系统的设计应确保：
- 锚固可靠，不发生滑移
- 具有良好的疲劳性能
- 便于安装和张拉

**（4）边界条件**

柔性支架的边界条件对结构受力有显著影响。立柱和基础的刚度应考虑在计算模型之中，端部锚锭的约束条件应明确。

#### 4.8.6 柔性支架的优缺点

| 优点 | 缺点 |
|------|------|
| 跨度大，减少立柱数量 | 对风荷载敏感，存在风振问题 |
| 适应地形能力强 | 设计计算复杂（几何非线性） |
| 对地表扰动小 | 施工精度要求高（预应力张拉） |
| 材料用量少（相对刚性支架） | 维护难度较大（索的检测与更换） |
| 视觉效果通透 | 在强风区的适用性受限 |

### 4.9 支架材料选择与对比

#### 4.9.1 钢材

**（1）常用牌号**

| 牌号 | 屈服强度（N/mm²） | 抗拉强度（N/mm²） | 主要应用 |
|------|------------------|------------------|---------|
| Q235B | 235 | 375-500 | 常规支架、连接件 |
| Q355B | 355 | 470-630 | 高应力构件、大跨度支架 |
| Q420 | 420 | 520-680 | 特殊高强度需求 |

**（2）钢材的优缺点**

| 优点 | 缺点 |
|------|------|
| 强度高，承载力大 | 需要防腐处理 |
| 价格相对低廉 | 自重较大 |
| 加工和焊接性能好 | 耐腐蚀性差（需镀锌或涂装） |
| 规格齐全，供应方便 | — |

**（3）适用场景**

钢材是光伏支架最常用的材料，适用于所有类型的支架，尤其是大型地面电站。

#### 4.9.2 铝合金

**（1）常用牌号**

| 牌号 | 状态 | 抗拉强度（N/mm²） | 主要应用 |
|------|------|------------------|---------|
| 6061 | T6 | ≥290 | 高强铝合金支架 |
| 6063 | T5 | ≥160 | 常规铝合金支架 |
| 6063 | T6 | ≥205 | 高强度铝合金支架 |

**（2）铝合金的优缺点**

| 优点 | 缺点 |
|------|------|
| 耐腐蚀性好，无需镀锌 | 强度相对较低 |
| 重量轻（约钢材的1/3） | 弹性模量低（E≈70GPa） |
| 外观美观，适合BIPV | 成本较高（约为钢材的2-3倍） |
| 易于挤压成型 | 焊接性能差（主要采用螺栓连接） |
| 可回收利用 | 对热敏感，火灾性能差 |

**（3）适用场景**

铝合金支架适用于：腐蚀环境（近海、工业腐蚀区）、对重量敏感的场景（屋顶荷载受限）、BIPV项目（外观要求高）、阳光房（美观要求高）。

#### 4.9.3 不锈钢

不锈钢支架具有极佳的耐腐蚀性能，但成本高（约为普通钢材的4-6倍），仅在特殊腐蚀环境下使用（如海边、化工厂附近）。

常用牌号：SUS304、SUS316。

#### 4.9.4 材料选择建议

| 场景 | 推荐材料 | 理由 |
|------|---------|------|
| 一般地面电站 | 钢支架（Q235B/Q355B） | 经济性好，强度足够 |
| 近海腐蚀环境 | 钢支架（加强镀锌）+ 不锈钢连接件 | 镀锌+不锈钢螺栓保证防腐 |
| 屋顶荷载受限 | 铝合金支架 | 轻量化，减少屋面荷载 |
| BIPV/阳光房 | 铝合金支架 | 外观美观，耐腐蚀 |
| 大跨度/重载 | 钢支架（Q355B/Q420） | 强度高，承载力大 |
| 高寒地区 | 钢支架（注意低温韧性） | 铝合金低温性能较好，但成本高 |

### 4.10 支架选型决策方法

#### 4.10.1 选型影响因素

光伏支架选型应综合考虑以下因素：

**（1）技术因素**
- 项目规模与地形条件
- 地质与水文条件
- 气候条件（风速、雪压、温度）
- 场地可利用面积
- 结构安全性要求

**（2）经济因素**
- 支架初始投资成本
- 全生命周期维护成本
- 发电量提升带来的收益
- 度电成本（LCOE）指标

**（3）环境因素**
- 环境保护要求
- 土地复垦要求
- 景观协调要求

**（4）政策因素**
- 电价政策
- 土地政策
- 补贴政策

#### 4.10.2 选型决策流程

支架选型建议按以下流程进行：

```
第一步：确定项目基本条件
  ├── 项目规模、场址地形
  ├── 气象参数（风速、雪压、辐照）
  └── 地质条件

第二步：技术方案初步筛选
  ├── 排除不适用方案
  ├── 选出2-3个候选方案
  └── 初步结构分析

第三步：发电量模拟
  ├── 各方案发电量计算
  ├── 增量发电收益估算
  └── 考虑跟踪损耗

第四步：经济性比较
  ├── 初始投资估算
  ├── 运营维护成本估算
  ├── 全生命周期收益计算
  └── LCOE比较

第五步：综合评估与决策
  ├── 技术风险评估
  ├── 施工可行性评估
  └── 最终方案确定
```

#### 4.10.3 方案比选指标

常用的方案比选指标包括：

| 指标 | 计算方式 | 意义 |
|------|---------|------|
| 初始单位投资（元/W） | 总投资/装机容量 | 衡量初始投资水平 |
| 度电成本LCOE（元/kWh） | 全生命周期成本/总发电量 | 衡量综合经济性 |
| 净现值NPV（万元） | 收益现值-成本现值 | 衡量项目盈利性 |
| 投资回收期（年） | 初始投资/年净收益 | 衡量资金回收速度 |
| 内部收益率IRR | 使NPV=0的折现率 | 衡量投资回报率 |


## 第五章 结构构件与节点术语

### 5.1 主梁（Main Beam）

#### 5.1.1 定义与功能

主梁是光伏支架中承载光伏组件的主要水平承重构件，沿光伏方阵的纵向或横向布置，是支架结构中的主要受力构件之一。

主梁的功用是：连接立柱、支撑檩条、将光伏组件和檩条传递来的荷载传递给立柱，最终传递至基础。

#### 5.1.2 主梁的类型

按布置方向分类：
- **纵向主梁**：沿组件长边方向布置，与组件长边平行。
- **横向主梁**：沿组件短边方向布置，与组件短边平行。

通常光伏支架的主梁沿纵向布置，檩条沿横向布置，共同形成格栅结构。

按截面形式分类：
- **H型钢主梁**：截面为H形，双向刚度大，适用于大跨度支架。
- **方钢管主梁**：截面为方形，抗扭性能好，外观整齐。
- **C型钢主梁**：截面为C形，冷弯成型，适用于轻载支架。
- **圆钢管主梁**：截面为圆形，各向同性，风阻小。

#### 5.1.3 主梁的设计计算

主梁设计应按受弯构件（或压弯构件）进行验算：

**（1）强度验算**

受弯强度：σ = M / (γ_x W_x) ≤ f
受剪强度：τ = V·S / (I·t_w) ≤ f_v

**（2）整体稳定验算**

M_x / (φ_b W_x) ≤ f

其中φ_b为梁的整体稳定系数。

**（3）局部稳定验算**

应保证受压翼缘和腹板的宽厚比满足要求。

**（4）挠度验算**

δ ≤ [δ] = L/250

#### 5.1.4 构造要求

1. 主梁的截面高度一般不小于跨度的1/20-1/15。
2. 主梁应与立柱可靠连接，宜采用刚接或铰接。
3. 主梁上应设置檩条连接件，间距宜与组件宽度匹配。
4. 主梁的拼接节点应设置在弯矩较小处。

### 5.2 立柱（Column）

#### 5.2.1 定义与功能

立柱是光伏支架中垂直于主梁（或基础）布置的竖向承重构件，用于将上部结构的荷载传递至基础。立柱是支架的“腿”，支撑着整个上部结构。

#### 5.2.2 立柱的类型

按材料分类：
- **钢立柱**：采用H型钢、方钢管、圆钢管等，应用最广泛。
- **铝合金立柱**：采用铝合金型材，常用于铝合金支架。
- **混凝土立柱**：用于特殊场景，较少使用。

按截面形式分类：
- **圆管立柱**：各向受力性能一致，抗弯承载力高，美观。
- **方管立柱**：连接方便，抗弯性能好。
- **H型钢立柱**：强轴方向承载力高，适用于单向受力为主的场景。
- **角钢立柱**：用于小型支架或支撑构件。

按布置方式分类：
- **竖直立柱**：垂直于地面，受力最直接。
- **倾斜立柱**：与地面成一定角度，可优化受力路径。

#### 5.2.3 立柱的设计计算

立柱设计应按轴心受压构件或压弯构件进行验算：

**（1）轴压承载力**

N ≤ φ·A·f

其中φ为轴心受压稳定系数，与长细比和截面类型有关。

**（2）压弯承载力**

N/(φ_x A) + β_mx M_x/(γ_x W_x(1-0.8N/N_EX)) ≤ f

**（3）长细比验算**

λ = l₀/i ≤ [λ]

受压立柱的容许长细比应满足：
- 主要构件：[λ] ≤ 180
- 次要构件：[λ] ≤ 220

#### 5.2.4 构造要求

1. 立柱与主梁的连接节点应传力可靠。
2. 立柱与基础的连接方式（铰接/刚接）应与计算简图一致。
3. 立柱底部应设置必要的排水措施，防止积水腐蚀。
4. 立柱应沿支架长度方向均匀布置，间距宜为2-4m。

### 5.3 檩条（Purlin）

#### 5.3.1 定义与功能

檩条是光伏支架中承受横向荷载的次要梁，与主梁垂直设置，形成固定面，用于直接安装和固定光伏组件。

檩条是直接接触光伏组件的构件，其安装精度直接影响光伏组件的安装质量和安全。

#### 5.3.2 檩条的类型

按截面形式分类：

| 截面形式 | 特点 | 适用场景 |
|---------|------|---------|
| C型檩条 | 冷弯薄壁，截面利用率高 | 最常用 |
| Z型檩条 | 搭接方便，刚度好 | 大跨度 |
| 方管檩条 | 抗扭性能好 | 对扭转敏感处 |
| 角钢檩条 | 简单、便宜 | 小型支架 |

按材料分类：
- 钢檩条（Q235B/Q355B冷弯或热轧）
- 铝合金檩条（6061-T6/6063-T6挤压成型）

#### 5.3.3 檩条的设计计算

檩条承受的荷载来自光伏组件（自重+风+雪），通过组件边框传递给檩条。设计计算要点：

**（1）荷载传递**

光伏组件对檩条的荷载可按均布荷载计算（简化为均布线荷载或集中力）。

**（2）强度验算**

檩条属于双向受弯构件，需要考虑两个方向的弯矩组合：
- 竖向荷载（重力+雪荷载）
- 法向荷载（风荷载，正风压或负风压）

**（3）挠度验算**

δ ≤ [δ] = L/250（一般要求）

对组件安装精度要求较高的项目，挠度限值可适当提高（如L/300）。

**（4）局部稳定验算**

冷弯薄壁C型钢檩条应验算卷边和翼缘的局部稳定。

#### 5.3.4 构造要求

1. 檩条间距应与组件的宽度（或长度）匹配，每块组件至少由2-4根檩条支撑。
2. 檩条与主梁的连接宜采用螺栓连接，便于安装和调整。
3. 檩条端部应设置端撑，防止在风荷载作用下发生扭转。
4. 檩条跨中不宜设置拼接接头，如需拼接应设置在靠近支座处。

### 5.4 支撑（Brace）

#### 5.4.1 定义与功能

支撑是保证支架整体稳定的斜向构件，包括拉杆、撑杆等，两端分别连接于不同构件以增强结构的整体刚度和稳定性。

支撑的主要功能：
1. 提高支架的整体稳定性，防止侧向失稳。
2. 缩短构件的计算长度，提高构件的稳定承载力。
3. 传递水平力（如风荷载的水平分力）至基础。
4. 保证支架在安装和施工过程中的稳定。

#### 5.4.2 支撑的类型

按受力方式分类：
- **拉杆（张拉支撑）** ：只承受拉力，不承受压力。通常采用圆钢或钢索。
- **压杆（压撑）** ：可承受压力和拉力，通常采用角钢或钢管。
- **拉压杆**：既能受拉也能受压，采用角钢或钢管。

按布置方向分类：
- **纵向支撑**：沿支架长度方向布置，抵抗纵向水平力。
- **横向支撑**：沿支架宽度方向布置，抵抗横向水平力。
- **水平支撑**：在水平面内布置，提高结构整体扭转刚度。

#### 5.4.3 支撑的设计要点

1. **长细比控制**：压杆的长细比不应大于容许值（主要构件≤180，次要构件≤220）。
2. **连接节点**：支撑两端应通过节点板或连接件与主梁/立柱可靠连接。
3. **交叉支撑**：采用交叉支撑时，应保证两支撑在交叉点处不产生相互干扰。
4. **预紧力**：对于采用张拉支撑的系统，应施加适当的预紧力。

### 5.5 连接节点

#### 5.5.1 定义与重要性

连接节点是光伏支架不同构件之间的连接区域，是荷载传递的关键部位。节点的设计质量直接关系到整个结构的受力性能和安全性。

在光伏支架结构中，常见的连接节点包括：
- 主梁-立柱连接节点
- 主梁-檩条连接节点
- 支撑-主梁/立柱连接节点
- 立柱-基础连接节点（柱脚）
- 组件-支架连接节点

#### 5.5.2 节点的分类

按连接方式分类：
- **焊接节点**：通过焊缝连接，刚性好，整体性强，但施工要求高，防腐难度大。
- **螺栓连接节点**：通过螺栓连接，便于安装和拆卸，施工速度快，但节点刚度相对较小。
- **铆接节点**：采用铆钉连接，目前已很少使用。

按力学性能分类：
- **刚接节点**：能够传递弯矩，构件之间相对转角为零。
- **铰接节点**：不能传递弯矩，构件之间可以自由转动。
- **半刚接节点**：介于刚接和铰接之间，能传递部分弯矩。

#### 5.5.3 节点设计原则

1. **传力明确**：荷载传递路径应清晰、直接。
2. **承载力匹配**：节点的承载力不应小于被连接构件的承载力。
3. **构造简单**：节点构造应便于施工和质量检验。
4. **适应变形**：节点应能适应构件的热胀冷缩和安装误差。
5. **防腐可靠**：节点处应便于防腐处理，不留死角。

### 5.6 螺栓连接设计

#### 5.6.1 螺栓类型与性能等级

光伏支架常用的螺栓类型和性能等级：

| 螺栓类型 | 性能等级 | 屈服强度（N/mm²） | 抗拉强度（N/mm²） | 应用 |
|---------|---------|------------------|------------------|------|
| 普通螺栓 | 4.6级 | 240 | 400 | 次要连接 |
| 普通螺栓 | 8.8级 | 640 | 800 | 主要连接 |
| 高强度螺栓 | 10.9级 | 900 | 1000 | 重要节点 |
| 不锈钢螺栓 | A2-70 | 450 | 700 | 腐蚀环境 |

8.8级螺栓是光伏支架最常用的类型，不锈钢螺栓在近海等腐蚀环境中使用。

#### 5.6.2 螺栓连接的受力计算

**（1）受剪承载力**

单个普通螺栓的抗剪承载力：

N_v^b = n_v · (π·d²/4) · f_v^b

其中：n_v为受剪面数，d为螺栓直径，f_v^b为螺栓的抗剪强度设计值。

**（2）承压承载力**

单个螺栓的承压承载力：

N_c^b = d·Σt·f_c^b

其中：Σt为连接板的厚度之和，f_c^b为螺栓的承压强度设计值。

**（3）受拉承载力**

单个螺栓的抗拉承载力：

N_t^b = (π·d_e²/4) · f_t^b

其中：d_e为螺栓的有效直径，f_t^b为螺栓的抗拉强度设计值。

#### 5.6.3 螺栓群的布置要求

1. **间距要求**：
   - 螺栓中心间距：≥3d（d为螺栓直径）
   - 螺栓至板边距离：≥1.5d（切割边）或≥2d（轧制边）

2. **端距要求**：
   - 螺栓至构件端部距离：≥2d

3. **构造要求**：
   - 每个节点螺栓数量不宜少于2个
   - 螺栓应均匀布置，避免偏心
   - 螺栓长度应保证外露丝扣不少于2扣

#### 5.6.4 螺栓连接的施工要求

1. 螺栓应按照设计扭矩进行紧固，使用扭矩扳手。
2. 高强度螺栓应采用专用工具施拧，并按规定进行预拉力控制。
3. 螺栓连接件应设置弹簧垫圈或防松垫圈，防止松动。
4. 在腐蚀环境中，螺栓应优先采用不锈钢材质，或采取有效防腐措施。

### 5.7 焊接连接设计

#### 5.7.1 焊接方法

光伏支架中常用的焊接方法：

| 焊接方法 | 特点 | 适用厚度 |
|---------|------|---------|
| 手工电弧焊 | 设备简单、操作灵活 | 各种厚度 |
| CO₂气体保护焊 | 效率高、焊接质量好 | 薄板至中厚板 |
| 埋弧焊 | 效率最高、质量稳定 | 中厚板、长焊缝 |

对于光伏支架这类轻型钢结构，CO₂气体保护焊是应用最广泛的焊接方法。

#### 5.7.2 焊缝类型

按受力方式分类：
- **对接焊缝**：受力性能好，传力直接，适用于等厚或相近厚度构件的连接。
- **角焊缝**：构造简单，施工方便，适用于搭接和T形连接。
- **塞焊和槽焊**：用于特殊连接要求。

光伏支架中最常用的是角焊缝，因其施工简便、适应性好。

#### 5.7.3 焊缝的计算

**（1）对接焊缝**

轴心受力：σ = N/(l_w·t) ≤ f_t^w 或 f_c^w
受弯：σ = M / W_w ≤ f_w

其中：l_w为焊缝计算长度，t为焊缝计算厚度（通常取板厚），f_w为焊缝强度设计值。

**（2）角焊缝**

在焊缝计算中，角焊缝的计算截面取焊缝有效截面（焊缝的计算厚度h_e乘以焊缝的计算长度l_w）：

轴心受力：σ = N/(h_e·l_w) ≤ f_f^w
受弯：τ = V/(h_e·l_w) ≤ f_f^w
复合受力：√(σ² + τ²) ≤ f_f^w

#### 5.7.4 焊缝构造要求

1. **最小焊脚尺寸**：h_f,min ≥ 1.5√t（t为较厚焊件的厚度），且不小于4mm。
2. **最大焊脚尺寸**：h_f,max ≤ 1.2t_min（t_min为较薄焊件的厚度）。
3. **焊缝长度**：角焊缝的计算长度不应小于8h_f和40mm的较大值；侧缝的计算长度不宜大于60h_f。
4. **焊缝布置**：应避免焊缝集中，减少焊接变形和残余应力。
5. **焊条选用**：应与母材强度匹配，对于Q235B钢，推荐E43系列焊条；对于Q355B钢，推荐E50系列焊条。

#### 5.7.5 焊接质量检验

1. **外观检验**：焊缝表面应无裂纹、气孔、夹渣等缺陷。
2. **无损检测**：重要焊缝应进行超声波检测或射线检测。
3. **尺寸检验**：焊缝的焊脚尺寸和长度应满足设计要求。

### 5.8 节点域受力分析

#### 5.8.1 节点域的概念

节点域是指节点区域中，各构件交汇处形成的一个受力区域。在节点域内，来自不同构件的内力相互交汇、平衡，应力状态复杂。

对于光伏支架而言，节点域主要指梁柱连接节点处，主梁端部弯矩、剪力和立柱轴力、弯矩交汇的区域。

#### 5.8.2 节点域的受力特征

节点域的受力特征包括：
1. **应力集中**：节点域是应力集中区域，构件截面突变处应力显著高于平均应力。
2. **多向应力**：节点域内存在多个方向的应力分量，形成复杂应力状态。
3. **剪应力大**：节点域的剪应力通常较大，需特别注意。

#### 5.8.3 节点域设计要点

1. 节点域应设置足够的加劲肋，防止腹板局部失稳。
2. 节点域厚度应满足抗剪要求，可采用节点域厚度增大或设置斜加劲肋。
3. 焊缝布置应避开应力集中区域。
4. 节点设计应便于施工和质量检验。


## 第六章 荷载计算与作用效应分析

### 6.1 荷载分类与组合原则

#### 6.1.1 荷载分类汇总

根据GB 50797-2012和NB/T 10115-2018的规定，光伏支架结构设计的荷载分类如下：

| 荷载类别 | 具体内容 | 分项系数（效应不利时） | 分项系数（效应有利时） |
|---------|---------|---------------------|---------------------|
| 永久荷载G | 结构自重、组件重量 | 1.3 | ≤1.0 |
| 风荷载W | 正风压、负风压 | 1.5 | — |
| 雪荷载S | 屋面雪荷载 | 1.5 | — |
| 施工检修荷载M | 安装、检修荷载 | 1.5 | — |
| 温度荷载T | 温度效应 | 1.5 | — |
| 地震作用E | 水平/竖向地震 | 1.3 | — |

#### 6.1.2 荷载组合原则

荷载组合应遵循以下原则：

1. **永久荷载为主导**：当永久荷载效应对结构不利时，分项系数取1.3；有利时取≤1.0。
2. **可变荷载组合**：当有两个或两个以上可变荷载参与组合时，除主导可变荷载外，其他可变荷载应乘以组合值系数ψ_c（通常取0.7）。
3. **地震组合**：地震作用组合时，重力荷载分项系数取1.3（对一般结构），风荷载组合值系数取0.2（风荷载起控制作用时取0.3）。

#### 6.1.3 荷载组合工况

典型荷载组合工况包括（但不限于）：

| 工况编号 | 组合内容 | 用途 |
|---------|---------|------|
| 工况1 | 1.3G + 1.5W | 风荷载主导（正风压） |
| 工况2 | 1.3G + 1.5W（负风压） | 风荷载主导（负风压/上拔力） |
| 工况3 | 1.3G + 1.5S | 雪荷载主导 |
| 工况4 | 1.3G + 1.5M | 施工检修荷载主导 |
| 工况5 | 1.3G + 1.5W + 1.5×0.7S | 风+雪组合 |
| 工况6 | 1.3G + 1.5S + 1.5×0.7W | 雪+风组合 |
| 工况7 | 1.3G + 1.3E | 地震组合（竖向地震+重力） |
| 工况8 | 1.3G + 0.65W + 1.3E | 地震组合（含风荷载） |

### 6.2 永久荷载（恒荷载）计算

#### 6.2.1 组成内容

永久荷载包括：
1. 光伏组件自重
2. 支架结构自重（钢/铝合金构件）
3. 电缆和汇流箱等电气设备自重
4. 连接件、紧固件自重

#### 6.2.2 单位面积荷载估算

光伏支架的永久荷载可根据经验估算：

| 项目 | 荷载（kN/m²） | 说明 |
|------|-------------|------|
| 光伏组件 | 0.12-0.20 | 随组件功率和尺寸变化 |
| 支架结构（钢） | 0.05-0.10 | 随跨度而变化 |
| 支架结构（铝） | 0.03-0.06 | 铝合金重量轻 |
| 电缆等 | 0.02-0.05 | — |
| **合计（钢支架）** | **0.20-0.35** | — |
| **合计（铝支架）** | **0.18-0.30** | — |

#### 6.2.3 精确计算方法

**（1）组件自重**

按组件供应商提供的技术参数，单块组件的重量除以组件面积：

g_组件 = G_组件 / A_组件

**（2）支架自重**

按设计截面的单位长度重量乘以构件长度，累加后除以投影面积：

g_支架 = Σ(ρ·A_i·L_i) / A_总

其中ρ为材料密度（钢材7850kg/m³，铝合金约2700kg/m³），A_i为各构件截面积，L_i为各构件长度。

### 6.3 风荷载计算

#### 6.3.1 基本规定

风荷载是光伏支架结构设计的控制性荷载。根据NB/T 10115-2018的规定：

1. 风荷载标准值按下式计算：w_k = β_z·μ_s·μ_z·w₀
2. 基本风压w₀应按GB 50009附录D中**25年一遇**的数值取值。当该值小于0.30kPa时，应按0.30kPa采用。
3. 体型系数μ_s：一般取±1.3。

#### 6.3.2 基本风压

基本风压w₀根据全国基本风压分布图或附录D取值。典型城市的基本风压值（25年一遇，近似可按50年一遇的0.85倍估算）：

| 城市 | 50年一遇w₀（kN/m²） | 25年一遇w₀（约） |
|------|-------------------|----------------|
| 北京 | 0.45 | 0.38 |
| 上海 | 0.55 | 0.47 |
| 广州 | 0.50 | 0.43 |
| 乌鲁木齐 | 0.60 | 0.51 |
| 拉萨 | 0.45 | 0.38 |
| 西宁 | 0.45 | 0.38 |
| 哈尔滨 | 0.55 | 0.47 |

#### 6.3.3 风压高度变化系数

风压高度变化系数μ_z按GB 50009的规定取值，与地面粗糙度类别有关：

| 离地面高度（m） | A类 | B类 | C类 | D类 |
|---------------|-----|-----|-----|-----|
| 5 | 1.09 | 1.00 | 0.65 | 0.51 |
| 10 | 1.28 | 1.00 | 0.65 | 0.51 |
| 15 | 1.42 | 1.13 | 0.65 | 0.51 |
| 20 | 1.52 | 1.23 | 0.74 | 0.51 |
| 30 | 1.67 | 1.39 | 0.88 | 0.51 |
| 40 | 1.79 | 1.52 | 1.00 | 0.60 |

**地面粗糙度类别**：
- A类：近海、海岛、海岸、沙漠地区
- B类：田野、乡村、丛林、丘陵及房屋较稀疏的城镇
- C类：有密集建筑群的城市市区
- D类：有密集建筑群且房屋较高的城市市区

对于光伏电站，绝大多数位于B类（乡村、田野）或C类（郊区）地区。

#### 6.3.4 风荷载体型系数

光伏支架的体型系数与支架的倾角、组件的排列方式和风向有关。根据NB/T 10115-2018：

- 对于常规固定支架，体型系数**宜取±1.3**。
- 对于跟踪支架，体型系数应根据跟踪角度确定，宜通过风洞试验确定。
- 对于柔性支架，体型系数应通过风洞试验或数值风洞确定。

在初步设计阶段，可采用以下简化取值：
- 正风压（迎风面）：+1.3
- 负风压（背风面）：-1.3（上拔力）

#### 6.3.5 风振系数

对于高度小于30m的光伏支架，风振系数β_z可取1.0（不考虑风振效应）。对于高度超过30m的高支架，应按GB 50009的规定计算风振系数。

#### 6.3.6 风荷载计算示例

【例】某光伏电站位于B类地区，光伏支架最高点离地高度为8m，基本风压w₀=0.40kN/m²，倾角30°。

w_k = β_z·μ_s·μ_z·w₀

其中：β_z=1.0；μ_s=±1.3；μ_z（B类，8m）≈1.0

正风压：w_k = 1.0 × 1.3 × 1.0 × 0.40 = 0.52 kN/m²
负风压：w_k = 1.0 × (-1.3) × 1.0 × 0.40 = -0.52 kN/m²

#### 6.3.7 脉动风效应

风荷载并非静定的，而是随时间变化（脉动风）。对于常规光伏支架（刚度较大、自振频率较高），可忽略脉动风效应；对于柔性支架（刚度小、自振频率低），必须考虑脉动风效应和风致振动。

脉动风的特征参数：
- 湍流强度：约10%-30%
- 阵风因子：约1.3-2.0
- 风谱：用于描述脉动风能量在频率域分布

### 6.4 雪荷载计算

#### 6.4.1 基本雪压

基本雪压s₀按GB 50009附录D中**25年一遇**的数值取值。当该值小于0.30kN/m²时，应按0.30kN/m²采用。

典型城市的基本雪压值：

| 城市 | 50年一遇s₀（kN/m²） | 25年一遇s₀（约） |
|------|-------------------|----------------|
| 北京 | 0.40 | 0.34 |
| 哈尔滨 | 0.45 | 0.38 |
| 乌鲁木齐 | 0.80 | 0.68 |
| 上海 | 0.20 | 0.17 |
| 广州 | 0 | 0 |
| 拉萨 | 0.30 | 0.26 |

#### 6.4.2 雪荷载标准值

屋面雪荷载标准值按下式计算：

s_k = μ_r·s₀

其中μ_r为屋面积雪分布系数。

对于光伏支架，由于其倾角较大，雪荷载通常按以下方式取值：

| 倾角范围 | 积雪分布系数μ_r | 说明 |
|---------|---------------|------|
| 0°-30° | 1.0 | 按常规屋面取 |
| 30°-50° | 0.8-0.6（线性插值） | 倾角越大，积雪越少 |
| >50° | 0 | 不考虑雪荷载 |

#### 6.4.3 雪荷载的局部效应

在光伏支架设计中需注意雪荷载的局部效应：

1. **积雪滑落**：当组件倾角较大时，积雪可能滑落，在支架下边缘形成积雪堆积，产生额外荷载。
2. **不均匀积雪**：在风速较大时，积雪可能不均匀分布，在局部区域产生超过平均值的雪荷载。
3. **融雪再冻**：温度变化导致积雪融化后在夜间重新冻结，荷载形态变化。

### 6.5 施工与检修荷载

#### 6.5.1 荷载取值

施工检修荷载的标准值按以下规定取值：

- **施工荷载**（安装阶段）：取1.0 kN（集中荷载）作用于最不利位置，或按实际施工方法确定。
- **检修荷载**（运行阶段）：取1.0-2.0 kN（集中荷载），作用于检修人员可能站立的位置。
- **堆放荷载**：设备材料临时堆放的荷载应单独考虑，不应与检修荷载同时组合。

#### 6.5.2 设计考虑

1. 施工检修荷载应与风荷载、雪荷载等同时组合，考虑其最不利组合。
2. 设计时应在组件安装位置、检修通道等关键区域进行局部验算。
3. 檩条和组件的连接应能承受施工安装时的集中荷载。
4. 施工阶段和使用阶段的荷载工况应分别考虑。

### 6.6 温度荷载

#### 6.6.1 温度作用

温度作用是指由于温度变化引起的结构变形和内力。对于光伏支架，温度作用主要来源于日温差和年温差。

当支架长度小于120m时，温度荷载对光伏支架的影响通常较小，可不考虑。当支架连续长度≥120m时，应按温度荷载进行验算。

#### 6.6.2 温度荷载计算

温度荷载的计算参数：

- **最高计算温度**：T_max = 当地极端最高温度 + 太阳辐射温升（约20-30°C）
- **最低计算温度**：T_min = 当地极端最低温度
- **安装温度**：T_install = 施工安装时的实际温度

温度变化量：ΔT = T_max - T_min（对伸缩缝计算）

温度应力：σ_T = E·α·ΔT

其中：E为钢材弹性模量（2.06×10⁵ N/mm²），α为钢材线膨胀系数（1.2×10⁻⁵/°C）。

#### 6.6.3 温度效应控制措施

1. **设置温度缝**：沿支架长度方向每隔一定距离设置温度缝，释放温度变形。温度缝间距一般不超过120m。
2. **采用柔性连接**：在支架与基础之间采用长圆孔螺栓连接，允许一定量的滑动。
3. **合理选择安装时机**：在温度适中的季节安装，减小安装温度与极端温度的差值。

### 6.7 地震作用计算

#### 6.7.1 抗震设防要求

根据GB 50797-2012的规定：

1. 光伏支架的抗震设防烈度应按本地区抗震设防烈度确定。
2. 抗震设防烈度为6度及以上地区的支架结构，应进行抗震设计。
3. 抗震设防烈度为8度及以上时，支架宜采用钢结构。
4. 抗震设计时，结构重要性系数取1.0。

#### 6.7.2 地震作用计算方法

对于高度不超过40m的光伏支架，可采用底部剪力法计算水平地震作用：

**（1）水平地震作用标准值**

F_Ek = α₁·G_eq

其中：α₁为水平地震影响系数（与设防烈度和场地类别有关），G_eq为结构等效总重力荷载（取总重力荷载的85%）。

**（2）地震影响系数**

| 设防烈度 | 设计基本加速度 | 水平地震影响系数α_max |
|---------|-------------|-------------------|
| 6度 | 0.05g | 0.04 |
| 7度 | 0.10g/0.15g | 0.08/0.12 |
| 8度 | 0.20g/0.30g | 0.16/0.24 |
| 9度 | 0.40g | 0.32 |

#### 6.7.3 竖向地震作用

在以下情况下应考虑竖向地震作用：
- 设防烈度为8度及以上的大跨度结构
- 设防烈度为9度时所有结构

竖向地震作用标准值可取水平地震作用的65%。

#### 6.7.4 抗震构造措施

1. 支架结构应具有明确的计算简图和合理的地震作用传递途径。
2. 节点连接应具有足够的延性，避免脆性破坏。
3. 支架与基础的连接应可靠，不应发生拔出现象。
4. 在抗震设防区，基础埋深不应小于0.5m。

### 6.8 荷载组合详解

#### 6.8.1 承载能力极限状态组合

承载能力极限状态的荷载组合应采用基本组合，按以下公式：

γ₀·S ≤ R

其中荷载效应组合S按以下方式计算：

**（1）由可变荷载效应控制的组合**

S = γ_G·S_Gk + γ_Q1·S_Q1k + Σ_{i=2}ⁿ γ_Qi·ψ_ci·S_Qik

**（2）由永久荷载效应控制的组合**

S = γ_G·S_Gk + Σ_{i=1}ⁿ γ_Qi·ψ_ci·S_Qik

其中：
- γ_G——永久荷载分项系数（不利时1.3，有利时≤1.0）
- γ_Qi——第i个可变荷载分项系数（取1.5）
- S_Gk——永久荷载效应标准值
- S_Qik——第i个可变荷载效应标准值
- ψ_ci——第i个可变荷载的组合值系数（风、雪取0.7，施工荷载取0.7）

#### 6.8.2 正常使用极限状态组合

正常使用极限状态的荷载组合应采用标准组合：

S = S_Gk + S_Q1k + Σ_{i=2}ⁿ ψ_ci·S_Qik

挠度验算应采用标准组合，荷载分项系数均取1.0。

#### 6.8.3 地震作用组合

地震作用组合的效应按下式计算：

S = γ_G·S_GE + γ_Eh·S_Ehk + γ_Ev·S_Evk + ψ_w·γ_w·S_wk

其中：
- γ_G——重力荷载分项系数（1.3）
- γ_Eh——水平地震作用分项系数（1.3）
- γ_Ev——竖向地震作用分项系数（1.3）
- ψ_w——风荷载组合值系数（0.2，风荷载起控制作用时0.3）

#### 6.8.4 荷载组合的工况列表

以下是光伏支架设计常用的荷载组合工况：

| 序号 | 组合表达式 | 控制内容 |
|------|-----------|---------|
| C1 | 1.3G + 1.5W (+) | 正风压主导 |
| C2 | 1.0G + 1.5W (-) | 负风压（上拔力）主导 |
| C3 | 1.3G + 1.5S | 雪荷载主导 |
| C4 | 1.3G + 1.5M | 施工荷载主导 |
| C5 | 1.3G + 1.5W + 1.05S | 风+雪组合 |
| C6 | 1.3G + 1.5S + 1.05W | 雪+风组合 |
| C7 | 1.3G + 1.3E_h | 水平地震主导 |
| C8 | 1.3G + 1.3E_v | 竖向地震主导（8/9度区） |

### 6.9 荷载效应分析

#### 6.9.1 内力计算

在确定荷载组合后，采用结构力学方法计算各构件的内力：

**（1）弯矩M**：由弯曲产生的内力矩，是梁类构件的主要内力。

**（2）剪力V**：垂直于构件轴线的力，由横向荷载产生。

**（3）轴力N**：沿构件轴线的力，拉为正、压为负。立柱以轴压为主，拉索以拉力为主。

**（4）扭矩T**：绕构件轴线的力矩，在偏心荷载作用下产生。

#### 6.9.2 内力组合

各荷载工况下的内力应按照荷载组合系数进行组合，取最不利值：

M_max = Σ γ_i · M_i
V_max = Σ γ_i · V_i
N_max = Σ γ_i · N_i

#### 6.9.3 分析方法选择

| 分析方法 | 适用条件 | 特点 |
|---------|---------|------|
| 一阶弹性分析 | 常规刚性支架 | 简单、计算量小 |
| 二阶弹性分析（P-Δ） | 高柔结构、柔性支架 | 考虑几何非线性 |
| 弹塑性分析 | 罕遇地震验算 | 考虑材料非线性 |
| 非线性时程分析 | 柔性支架风振分析 | 考虑动力效应 |


## 第七章 构件设计与强度验算

### 7.1 构件设计基本原则

#### 7.1.1 设计流程

构件设计的基本流程如下：

```
第1步：确定构件计算简图
  └── 支座条件、荷载作用方式

第2步：计算内力
  └── 通过结构分析获得

第3步：初选截面
  └── 根据经验或估算初选

第4步：截面验算
  ├── 强度验算
  ├── 稳定验算
  ├── 刚度验算（变形）
  └── 局部稳定验算

第5步：调整截面
  └── 如不满足要求，增大截面后重新验算
```

#### 7.1.2 设计表达式

按承载能力极限状态设计的通用表达式：

**强度验算**：σ ≤ f 或 N/A ≤ f 或 M/W ≤ f

**稳定验算**：N/(φ·A) ≤ f 或 M/(φ_b·W) ≤ f

**刚度验算**：δ ≤ [δ]

#### 7.1.3 钢材强度设计值

常用钢材的强度设计值（N/mm²）：

| 牌号 | 厚度（mm） | 抗拉/抗压/抗弯f | 抗剪f_v | 端面承压f_ce |
|------|----------|---------------|--------|------------|
| Q235B | ≤16 | 215 | 125 | 325 |
| Q235B | 16-40 | 205 | 120 | 325 |
| Q235B | 40-100 | 200 | 115 | 325 |
| Q355B | ≤16 | 310 | 180 | 400 |
| Q355B | 16-40 | 295 | 170 | 400 |
| Q355B | 40-100 | 265 | 155 | 400 |

### 7.2 轴心受力构件

#### 7.2.1 轴心受拉构件

轴心受拉构件（如拉杆、拉索）的承载力验算：

**强度**：N/A_n ≤ f

其中：N为轴向拉力设计值，A_n为构件的净截面积，f为钢材抗拉强度设计值。

对于采用圆钢的拉杆，还应验算螺纹处的有效截面：

A_n = (π·d_e²)/4

其中d_e为螺纹有效直径。

#### 7.2.2 轴心受压构件

轴心受压构件（如立柱、压杆）需要验算强度、整体稳定和局部稳定：

**（1）强度验算**

N/A_n ≤ f

**（2）整体稳定验算**

N/(φ·A) ≤ f

其中φ为轴心受压稳定系数，根据长细比λ和截面类型查表确定。

**（3）局部稳定验算**

构件的翼缘和腹板应满足宽厚比要求，防止局部失稳：
- 翼缘：b/t ≤ (10+0.1λ)·√(235/f_y)
- 腹板：h₀/t_w ≤ (25+0.5λ)·√(235/f_y)

#### 7.2.3 长细比计算

长细比是轴心受压构件最重要的参数：

λ = l₀ / i

其中：l₀为构件的计算长度，i为截面回转半径（i = √(I/A)）。

**计算长度的确定**：
- 两端铰接：l₀ = l
- 一端固定、一端铰接：l₀ = 0.7l
- 两端固定：l₀ = 0.5l
- 一端固定、一端自由：l₀ = 2l

对于光伏支架立柱，通常可按两端铰接或一端固定（基础端）一端铰接（顶端）确定计算长度。

### 7.3 受弯构件

#### 7.3.1 受弯构件的强度验算

受弯构件（如主梁、檩条）需要验算：

**（1）抗弯强度**

M_x / (γ_x W_nx) + M_y / (γ_y W_ny) ≤ f

其中：
- M_x、M_y——绕强轴和弱轴的弯矩
- W_nx、W_ny——净截面模量
- γ_x、γ_y——塑性发展系数（一般取1.05）

对于双向受弯构件（如檩条），需同时考虑两个方向的弯曲应力。

**（2）抗剪强度**

τ = V·S / (I·t_w) ≤ f_v

其中：V为剪力设计值，S为截面面积矩，I为截面惯性矩，t_w为腹板厚度。

对于光伏支架中的受弯构件，抗剪通常不起控制作用，但在短跨重载情况下应验算。

**（3）局部压应力**

在荷载集中作用处（如主梁上支承檩条的位置），应验算局部压应力：

σ_c = ψ·F / (t_w·l_z) ≤ f

其中：F为集中力，l_z为荷载在腹板上的分布长度。

#### 7.3.2 受弯构件的整体稳定验算

当梁的受压翼缘没有足够的侧向支撑时，可能发生整体失稳（侧向扭转屈曲）：

M_x / (φ_b W_x) ≤ f

其中φ_b为梁的整体稳定系数，按GB 50017的规定计算。

光伏支架的主梁和檩条，由于有组件和支撑提供侧向约束，通常不需要验算整体稳定，但在大跨度无支撑情况下应进行验算。

#### 7.3.3 受弯构件的挠度控制

受弯构件在标准组合荷载作用下的最大挠度不应超过允许值：

δ_max ≤ [δ]

光伏支架的挠度限值（工程实践）：
- 主梁挠度：[δ] = L/250
- 檩条挠度：[δ] = L/250（组件安装要求高时取L/300）

### 7.4 压弯构件

#### 7.4.1 定义

压弯构件是指同时承受轴压力和弯矩的构件。光伏支架中的立柱在风荷载作用下，除承受轴向压力外，还承受由水平力产生的弯矩，属于典型的压弯构件。

#### 7.4.2 强度验算

N/A_n + M_x/(γ_x W_nx) + M_y/(γ_y W_ny) ≤ f

#### 7.4.3 稳定验算

压弯构件的稳定验算比轴心受压构件更为复杂，需要考虑弯矩作用下的二阶效应：

**（1）弯矩作用平面内的稳定**

N/(φ_x A) + β_mx M_x/(γ_x W_x (1-0.8N/N_EX)) ≤ f

其中：β_mx为等效弯矩系数，N_EX为欧拉临界力。

**（2）弯矩作用平面外的稳定**

N/(φ_y A) + η·β_tx M_x/(φ_b W_x) ≤ f

其中：φ_y为绕弱轴的稳定系数，φ_b为梁的整体稳定系数，η为调整系数。

对于光伏支架立柱，由于截面通常为双轴对称（方管、圆管），且弯矩作用在强轴方向，应特别注意弯矩作用平面内的稳定验算。

#### 7.4.4 N-M相关曲线

压弯构件的N-M相关曲线是判断其承载力的重要工具。当轴压力和弯矩的组合位于相关曲线内部时，构件安全；位于外部时，构件不安全。

对于简单截面，可采用简化相关公式。

### 7.5 拉弯构件

#### 7.5.1 定义

拉弯构件是指同时承受轴向拉力和弯矩的构件。光伏支架中较少出现拉弯构件，但在某些特殊工况（如负风压作用下）可能产生。

#### 7.5.2 验算方法

拉弯构件的验算相对简单：

N/A_n + M_x/(γ_x W_nx) + M_y/(γ_y W_ny) ≤ f

由于拉力对稳定有利（使构件趋于拉直），拉弯构件通常不需要验算稳定，仅需验算强度。

### 7.6 构件稳定理论

#### 7.6.1 稳定问题的分类

结构稳定问题可分为三类：

1. **分支点失稳（第一类稳定）** ：结构在某一荷载水平下，平衡状态发生分支，从稳定平衡变为不稳定平衡。理想的轴心受压构件的屈曲属于分支点失稳。

2. **极值点失稳（第二类稳定）** ：结构在荷载增加过程中，承载力达到极值后下降。具有初始缺陷的压杆失稳属于极值点失稳。

3. **跳跃失稳**：结构从一个平衡状态突然跳跃到另一个平衡状态。

光伏支架结构的稳定问题主要为分支点失稳（理想构件）和极值点失稳（实际构件）。

#### 7.6.2 轴心受压构件的弹性屈曲

欧拉临界力是轴心受压构件弹性屈曲的经典公式：

N_Er = π² E I / l₀²

转化为应力形式：σ_Er = π² E / λ²

当λ较大（细长构件）时，构件在应力远低于屈服强度时即发生弹性屈曲。

#### 7.6.3 轴心受压构件的弹塑性屈曲

当λ较小（短粗构件）时，构件的屈曲发生在材料进入塑性阶段。此时需采用切线模量理论：

N_cr = π² E_t I / l₀²

其中E_t为切线模量（应力-应变曲线在对应应力处的切线斜率）。

工程设计中采用稳定系数φ来统一考虑弹性屈曲和弹塑性屈曲。φ值与长细比λ和截面类型有关，可查GB 50017附录中的稳定系数表。

#### 7.6.4 提高稳定性的措施

1. **减小计算长度**：增设支撑或减少支撑间距。
2. **增大截面回转半径**：选择合理的截面形式（如圆管比H型钢具有更大的回转半径）。
3. **改善边界条件**：采用刚接而非铰接，提高约束刚度。
4. **提高钢材强度**：高强度钢材可在相同截面积下提高承载力。

### 7.7 局部稳定与加劲肋设计

#### 7.7.1 局部稳定

局部稳定是指构件组成板件（翼缘、腹板）在压应力作用下发生局部屈曲的现象。局部失稳虽不一定导致构件整体破坏，但会降低构件的承载力。

局部稳定的控制方法是对板件的宽厚比进行限制。

**（1）翼缘宽厚比限值**

对于工字形和H形截面：
- 弹性设计：b/t ≤ 15√(235/f_y)
- 弹塑性设计：b/t ≤ 13√(235/f_y)

对于方管和矩形管截面：
- b/t ≤ 40√(235/f_y)（考虑局部稳定）

**（2）腹板高厚比限值**

对于工字形和H形截面：
- 当腹板高厚比 h₀/t_w ≤ 80√(235/f_y) 时，可不设加劲肋
- 当 h₀/t_w 超过限值时，应设置加劲肋

光伏支架的构件截面较小，一般能满足局部稳定要求，但仍应予以验算。

#### 7.7.2 加劲肋设计

当腹板高厚比超过限值时，需要设置加劲肋：

1. **横向加劲肋**：垂直于腹板布置，提高腹板的抗剪稳定承载力。
2. **纵向加劲肋**：平行于腹板布置，提高腹板的抗弯稳定承载力。
3. **支撑加劲肋**：在支座处或集中力作用处布置。

对于光伏支架，由于构件截面通常不大，一般仅需在节点域设置必要的加劲肋。

### 7.8 长细比控制

#### 7.8.1 长细比的定义

长细比λ是构件计算长度l₀与截面回转半径i的比值：

λ = l₀ / i

长细比综合反映了构件的长度、截面形状和边界约束条件对稳定性的影响。长细比越大，构件越柔，稳定承载力越低。

#### 7.8.2 容许长细比

根据GB 50797-2012和NB/T 10115-2018的规定，并结合工程实践经验，光伏支架构件的容许长细比如下：

| 构件类型 | 容许长细比[λ] | 说明 |
|---------|-------------|------|
| 受压主要构件（立柱） | ≤180 | 主要承压构件 |
| 受压其他构件（支撑等） | ≤220 | 次要受压构件 |
| 受拉主要构件（主梁、檩条） | ≤350 | 受拉构件稳定要求较低 |
| 受拉柱间支撑 | ≤300 | 受拉支撑 |
| 预应力拉索 | 不限 | 受拉且预拉 |

#### 7.8.3 长细比的优化

当构件长细比超过容许值时，可通过以下方式优化：

1. **增大截面回转半径**：采用圆管代替方管，或增大截面尺寸。
2. **减小计算长度**：增设支撑点或系杆。
3. **改变边界条件**：采用刚接节点，减小计算长度系数。
4. **选用高强度材料**：但注意高强度材料弹性模量不变，对屈曲承载力提升有限。

### 7.9 变形与挠度验算

#### 7.9.1 变形控制的重要性

光伏支架对变形较为敏感，原因如下：

1. **组件受力**：过大的变形会导致光伏组件承受附加应力，可能引起组件隐裂或破裂。
2. **发电效率**：支架变形过大会导致组件之间产生相对位移，影响组件受光的一致性。
3. **设备安全**：跟踪支架的变形会影响跟踪精度。
4. **视觉感受**：变形过大影响外观质量。

#### 7.9.2 挠度限值

光伏支架的挠度限值（正常使用极限状态）：

| 构件 | 挠度限值 | 备注 |
|------|---------|------|
| 主梁 | L/250 | 对变形不敏感时 |
| 主梁 | L/300 | 对变形敏感时 |
| 檩条 | L/250 | 常规要求 |
| 檩条 | L/300 | 组件安装精度高时 |
| 立柱柱顶水平位移 | H/60 | 抗震/抗风要求 |

#### 7.9.3 挠度计算方法

挠度计算应采用标准组合（荷载分项系数取1.0），使用结构力学方法计算。

对于简支梁，最大挠度的一般公式：

δ_max = k·P·l³ / (E·I)

其中k为系数（取决于荷载分布），P为荷载，l为跨度，E为弹性模量，I为截面惯性矩。

常见荷载下的挠度公式：
- 均布荷载：δ_max = 5q·l⁴/(384EI)
- 跨中集中荷载：δ_max = P·l³/(48EI)
- 悬臂端集中荷载：δ_max = P·l³/(3EI)

### 7.10 疲劳验算

#### 7.10.1 疲劳问题

疲劳是材料在重复荷载作用下发生累积损伤、最终断裂的现象。对于光伏支架，可能的疲劳问题包括：

1. **风荷载反复作用**：特别是在多风地区，支架承受长期的风致振动。
2. **跟踪支架的反复运动**：跟踪支架每天转动，连接节点承受反复应力。
3. **温度循环应力**：昼夜温差导致的反复应力。

#### 7.10.2 是否需要疲劳验算

根据GB 50017的规定，在以下情况下应进行疲劳验算：
- 承受动力荷载重复作用的构件
- 设计使用年限内的应力循环次数超过10⁴次

对于常规光伏支架，由于风荷载的应力幅较小（应力比通常小于0.3），且钢结构疲劳强度较高，通常不需要进行专门的疲劳验算。

但对于跟踪支架（运动部件）和柔性支架（振动敏感），应考虑疲劳问题，采取以下措施：
1. 选用优质钢材和焊材
2. 控制焊缝质量等级
3. 避免应力集中（圆角过渡、避免焊缝交叉）
4. 设置阻尼装置减少振动


## 第八章 地基基础设计

### 8.1 岩土工程勘察要求

#### 8.1.1 勘察目的

地基基础设计必须依据岩土工程勘察报告进行。勘察的目的是查明场地的工程地质条件和水文地质条件，提供基础设计和施工所需的参数。

#### 8.1.2 勘察内容

光伏电站的岩土工程勘察应包括以下内容：

1. **地形地貌**：场地的地形、地貌特征。
2. **地层结构**：各土层的分布、厚度、埋深和物理力学性质。
3. **地下水**：地下水位、类型、变化幅度及其腐蚀性。
4. **不良地质**：是否存在滑坡、崩塌、泥石流、岩溶等不良地质作用。
5. **岩土参数**：提供地基承载力、压缩模量、抗剪强度等设计参数。
6. **腐蚀性评价**：地下水和土壤对钢材和混凝土的腐蚀性评价。

#### 8.1.3 勘察工作量

| 项目类型 | 勘探点间距 | 勘探深度 |
|---------|----------|---------|
| 大型地面电站 | 50-100m | 进入持力层≥3m |
| 分布式电站 | 100-200m | 进入持力层≥3m |
| 复杂地质条件 | 适当加密 | 穿透软弱层 |

勘探点数量不应少于3个。

### 8.2 基础类型与选型

#### 8.2.1 基础分类总览

根据GB 51101-2016《太阳能发电站支架基础技术规范》，光伏支架基础可按以下方式分类：

| 大类 | 子类 | 适用条件 |
|------|------|---------|
| 桩基础 | 预制桩（钢螺旋桩、锤击型钢桩、混凝土预制桩、预应力混凝土桩） | 各类土层 |
| 桩基础 | 灌注桩（微孔灌注桩等） | 干成孔工艺，土层稳定 |
| 扩展基础 | 混凝土独立基础、条形基础 | 地基承载力较好的场地 |
| 锚杆基础 | 岩石锚杆 | 岩石完整、埋深浅的场地 |

#### 8.2.2 各类基础的特点

**（1）微型灌注桩**

特点：
- 桩径小（通常150-300mm）
- 施工速度快，设备轻便
- 适用于各种土层
- 承载能力可靠

适用条件：各类土层，尤其是地基承载力较低的软土地区。

**（2）钢螺旋桩**

特点：
- 可直接旋入土层，施工极快
- 无需开挖，对地表扰动小
- 可回收利用（临时性项目）
- 抗拔承载力较好

适用条件：地下水位低、持力层埋深较浅的场地；冬季施工优势明显。

**（3）预制混凝土桩**

特点：
- 承载力高
- 质量稳定（工厂预制）
- 施工速度快（锤击或静压）
- 适用性强

适用条件：各类场地，尤其是承载力要求较高的项目。

**（4）扩展式基础**

特点：
- 采用现浇混凝土
- 施工质量易控制
- 适用性广泛
- 但对地表扰动较大

适用条件：地基承载力较好的场地，施工场地开阔。

**（5）岩石锚杆基础**

特点：
- 适用于岩石地区
- 承载力高
- 施工速度快

适用条件：岩石完整程度为较完整～完整，岩石埋深浅或直接出露的场区。

#### 8.2.3 基础选型流程

基础选型应综合考虑以下因素：

```
基础选型决策因素
├── 地质条件（土层类型、承载力、地下水）
├── 荷载特征（竖向力、水平力、弯矩、上拔力）
├── 施工条件（施工季节、设备进场条件）
├── 经济性（材料成本、施工成本、工期成本）
├── 环保要求（地表扰动、生态恢复）
└── 工期要求（施工速度）
```

#### 8.2.4 基础选型建议

| 地质条件 | 推荐基础类型 | 备选基础类型 |
|---------|------------|------------|
| 软土、淤泥 | 微型灌注桩 | 预制混凝土桩 |
| 粉土、黏性土 | 微型灌注桩、钢螺旋桩 | 扩展基础 |
| 砂土、碎石土 | 钢螺旋桩、预制桩 | 微型灌注桩 |
| 岩石地区 | 锚杆基础 | 扩展基础 |
| 地下水位高 | 预制桩、钢螺旋桩 | 避免灌注桩 |
| 冬季施工 | 钢螺旋桩、预制桩 | 避免现浇混凝土 |
| 生态敏感区 | 钢螺旋桩 | 微型灌注桩 |

#### 8.2.5 基础布置原则

1. **同一光伏方阵内**宜采用同类型基础形式，便于施工管理和质量控制。
2. 基础布置应使荷载均匀传递，避免偏心过大。
3. 基础的埋置深度应满足抗冻、抗拔和稳定性要求。
4. 基础之间的间距应满足施工设备和工艺的要求。

### 8.3 微型桩基础

#### 8.3.1 定义

微型桩基础是指桩径较小（通常为150-300mm）的灌注桩基础，采用小型钻机成孔后灌注混凝土或水泥砂浆而成。微型桩是光伏支架中应用最为广泛的基础形式之一。

#### 8.3.2 设计与构造要求

**（1）桩径与桩长**

- 桩径：150-300mm，常用200mm和250mm
- 桩长：3-8m，根据土层条件确定

**（2）材料要求**

- 混凝土强度等级：C25及以上
- 水泥砂浆强度等级：M30及以上
- 纵向钢筋：不少于4根HRB400级钢筋，直径12-16mm
- 箍筋：直径6-8mm，间距200-300mm

**（3）保护层厚度**

- 混凝土保护层厚度：不应小于35mm
- 水泥砂浆保护层厚度：不应小于35mm

**（4）桩顶构造**

桩顶应设置预埋螺栓或锚板，用于与立柱连接。预埋螺栓的埋置深度应满足锚固要求。

#### 8.3.3 承载力计算

微型桩的单桩承载力由桩侧摩阻力和桩端阻力共同提供：

R_a = (1/K)·(Σ q_si·l_i·u + q_p·A_p)

其中：
- q_si——桩侧第i层土的极限摩阻力标准值（kPa）
- l_i——桩在第i层土中的长度（m）
- u——桩身周长（m）
- q_p——桩端土极限承载力标准值（kPa）
- A_p——桩端截面积（m²）
- K——安全系数（取2-3）

#### 8.3.4 抗拔承载力

对于承受上拔力的基础（风荷载产生的上拔力），应验算桩的抗拔承载力：

R_a,up = (1/K)·Σ q_si·l_i·u + G_p

其中G_p为桩身自重（地下部分取浮重度）。

#### 8.3.5 施工要点

1. 成孔后应清除孔底沉渣，沉渣厚度不应大于50mm。
2. 钢筋笼应设置定位筋，保证保护层厚度。
3. 混凝土（或砂浆）应连续灌注，不得中断。
4. 灌注完成后应及时养护，养护期不少于7天。
5. 预埋螺栓定位应准确，允许偏差±5mm。

### 8.4 钻孔灌注桩基础

#### 8.4.1 定义

钻孔灌注桩是采用钻机成孔后放入钢筋笼、灌注混凝土而形成的大直径桩基础。在光伏工程中，钻孔灌注桩主要用于大型地面电站或承载力要求较高的项目。

#### 8.4.2 与微型桩的区别

| 项目 | 微型桩 | 钻孔灌注桩 |
|------|-------|----------|
| 桩径 | 150-300mm | 300-800mm |
| 施工设备 | 小型钻机 | 大型钻机 |
| 成孔方式 | 干成孔为主 | 泥浆护壁或干成孔 |
| 承载力 | 较低 | 较高 |
| 适用性 | 大部分场地 | 各类场地 |
| 成本 | 较低 | 较高 |

#### 8.4.3 设计要点

1. 桩身混凝土强度不低于C30。
2. 主筋直径不宜小于16mm，配筋率0.2%-0.65%。
3. 桩端应进入持力层不小于3倍桩径。
4. 当存在厚层软土时，桩端应穿透软土层。

#### 8.4.4 施工注意事项

1. 干成孔工艺要求土层不缩径、不塌孔。
2. 地下水位高时需采用泥浆护壁或钢护筒。
3. 清孔应彻底，沉渣厚度不应超过50mm。
4. 水下灌注混凝土应连续施工。

### 8.5 钢螺旋桩基础

#### 8.5.1 定义

钢螺旋桩（又称钢制螺旋地锚）是一种具有螺旋叶片的钢桩，通过旋转方式旋入土中，利用螺旋叶片的承压和土的摩擦力承受荷载。

钢螺旋桩是光伏支架基础中施工速度最快、对地表扰动最小的一种形式。

#### 8.5.2 结构组成

钢螺旋桩由以下部分组成：
1. **桩身**：钢管或圆钢（直径76-219mm）
2. **螺旋叶片**：焊接在桩身的螺旋状钢板（1-3片）
3. **桩顶连接件**：法兰或预埋螺栓，用于连接立柱

#### 8.5.3 设计承载力

钢螺旋桩的承载力由桩身强度、叶片承载力和土的摩阻力控制：

R_a = (1/K)·(q_p·A_p + Σ q_si·A_si)

其中A_p为螺旋叶片的投影面积，A_si为桩身与各土层的接触面积。

抗拔承载力主要由螺旋叶片提供。

#### 8.5.4 施工要点

1. 采用专用旋桩设备将桩旋入指定深度。
2. 旋入过程中应控制扭矩，防止扭断桩身。
3. 桩顶标高应严格控制，偏差±20mm。
4. 旋入完成后应进行承载力检测。

#### 8.5.5 钢螺旋桩的优缺点

| 优点 | 缺点 |
|------|------|
| 施工速度最快（单桩几分钟） | 对硬地层适用性差 |
| 无需开挖，无土方工程量 | 钢材用量大，成本较高 |
| 可回收利用 | 防腐要求高 |
| 适合冬季施工 | 承载力相对较低 |
| 对地表无扰动 | 需要专业施工设备 |

### 8.6 预制混凝土桩基础

#### 8.6.1 定义

预制混凝土桩是指预先在工厂预制、现场通过锤击或静压方式打入地层的混凝土桩。在光伏项目中，常用的预制桩包括预应力高强混凝土管桩（PHC桩）和实心方桩。

#### 8.6.2 适用条件

预制混凝土桩适用于：
- 各类土层（除坚硬岩层外）
- 承载力要求高的项目
- 地下水位高的地区
- 有锤击或静压施工条件的场地

#### 8.6.3 设计要点

1. 桩型选择应根据承载力要求和地层条件确定。
2. 桩端进入持力层的深度：黏性土、粉土不小于2倍桩径；砂土不小于1.5倍桩径。
3. 桩的接头强度应满足承受弯矩和剪力的要求。
4. 桩顶与立柱的连接应采用预埋螺栓或焊接方式。

#### 8.6.4 施工方法

- **锤击法**：用柴油锤或液压锤将桩打入地层，适用于各种土层。
- **静压法**：用静力压桩机将桩压入地层，无振动和噪音，适用于城市周边。

### 8.7 扩展式基础

#### 8.7.1 定义

扩展式基础（又称浅基础）是指将上部结构荷载通过基础底板直接传递至浅层地基的基础形式。在光伏工程中，扩展式基础通常为现浇混凝土独立基础或条形基础。

#### 8.7.2 设计要点

**（1）基础尺寸确定**

基础底面积由地基承载力决定：

A ≥ (N + G) / f_a

其中：N为上部结构竖向力，G为基础自重及覆土重，f_a为地基承载力特征值。

**（2）基础厚度确定**

基础厚度由抗冲切和抗弯承载力决定。

**（3）配筋设计**

扩展基础应配置双向钢筋网，最小配筋率0.15%。

#### 8.7.3 优缺点

| 优点 | 缺点 |
|------|------|
| 施工质量易控制 | 混凝土用量大 |
| 造价相对较低（非软土地区） | 对地表扰动大 |
| 施工设备简单 | 不适合软土和冬季施工 |
| 承载力可靠 | 土方工程量大 |

### 8.8 锚杆基础

#### 8.8.1 定义

锚杆基础是指通过在岩石中钻孔并灌注砂浆（或混凝土），将锚杆（钢筋）锚固于岩体中，形成抗拔基础的结构形式。

#### 8.8.2 适用条件

- 岩石完整程度为较完整～完整
- 岩石埋深浅或直接出露
- 需要抗拔承载力高的部位

#### 8.8.3 设计要点

1. 锚杆的锚固长度应通过计算确定，一般不小于2m。
2. 锚杆孔径：不小于锚杆直径的2倍。
3. 灌浆材料：M30及以上水泥砂浆。
4. 锚杆的间距：不小于锚固长度的1/2。

### 8.9 基础承载力验算

#### 8.9.1 地基承载力验算

基础底面平均压力应满足：

p_k ≤ f_a

基础底面边缘最大压力应满足：

p_k,max ≤ 1.2 f_a

其中：
- p_k = (N + G) / A
- p_k,max = p_k + M/W（偏心荷载时）

#### 8.9.2 桩基承载力验算

单桩承载力应满足：

Q_k ≤ R_a

其中Q_k为单桩竖向力标准值，R_a为单桩承载力特征值。

在风荷载等水平力作用下，还应验算桩的水平承载力。

#### 8.9.3 抗拔承载力验算

在负风压（上拔力）作用下，基础应满足抗拔稳定性：

G + R_up ≥ N_up

其中：G为基础及上覆土自重，R_up为基础抗拔承载力，N_up为上拔力设计值。

对于桩基础，抗拔承载力由桩侧摩阻力和桩身自重提供。

### 8.10 基础沉降计算

#### 8.10.1 沉降控制

光伏支架基础对沉降较为敏感，尤其是跟踪支架。不均匀沉降可能导致：
- 支架倾斜、变形
- 跟踪机构卡滞
- 组件受力不均

#### 8.10.2 沉降计算方法

采用分层总和法计算地基沉降：

S = Σ (Δσ_i / E_si) · Δh_i

其中：Δσ_i为第i层土的附加应力增量，E_si为第i层土的压缩模量，Δh_i为第i层土的厚度。

#### 8.10.3 沉降限值

| 项目 | 沉降差限值 | 备注 |
|------|----------|------|
| 固定支架 | 相邻基础沉降差≤0.001L | L为基础间距 |
| 跟踪支架 | 相邻基础沉降差≤0.0005L | 要求更严格 |
| 最大沉降量 | ≤50mm | 一般要求 |

### 8.11 基础抗拔与抗倾覆验算

#### 8.11.1 抗拔验算

对于承受上拔力的基础，应验算：

N_up ≤ G_b + R_up,f

其中：N_up为作用在基础上的上拔力设计值，G_b为基础及上覆土的有效自重，R_up,f为地基的抗拔承载力。

#### 8.11.2 抗倾覆验算

对于承受弯矩作用的基础，应验算：

M_ov ≤ M_r

其中：M_ov为倾覆力矩设计值，M_r为抗倾覆力矩设计值。

抗倾覆安全系数不应小于1.5（基本组合）。


## 第九章 防腐与耐久性设计

### 9.1 腐蚀机理与防腐策略

#### 9.1.1 钢材的腐蚀机理

钢材的腐蚀主要是电化学腐蚀过程。在潮湿环境中，钢材表面形成水膜，溶解空气中的CO₂、SO₂等气体后形成电解质溶液，钢材中的铁与碳形成微电池，铁发生氧化反应而被腐蚀。

腐蚀速率受以下因素影响：
1. **大气环境**：工业区＞城市＞乡村＞海洋（近海）＞干燥内陆
2. **相对湿度**：湿度＞60%时腐蚀速率显著加快
3. **温度**：温度升高，化学反应速率加快
4. **污染物**：SO₂、Cl⁻等加速腐蚀

#### 9.1.2 腐蚀环境分类

根据GB/T 15957《大气环境腐蚀性分类》，大气腐蚀环境可分为：

| 腐蚀等级 | 环境特征 | 典型区域 | 腐蚀速率（μm/年） |
|---------|---------|---------|----------------|
| C1 | 很低 | 加热的室内空间 | ≤1.3 |
| C2 | 低 | 未加热的室内、温带乡村 | 1.3-25 |
| C3 | 中等 | 城镇、工业区、亚热带乡村 | 25-50 |
| C4 | 高 | 工业区、沿海地区 | 50-80 |
| C5 | 很高 | 高湿度工业区、亚热带沿海 | 80-200 |
| CX | 极高 | 热带沿海、高腐蚀工业区 | >200 |

光伏支架长期暴露于室外，大多数项目处于C3-C5级腐蚀环境中。

#### 9.1.3 防腐策略

光伏支架的防腐策略主要包括：

1. **镀层保护**：通过热浸镀锌、镀锌铝镁等工艺在钢材表面形成保护层。
2. **涂层保护**：涂刷防腐涂料或粉末涂层。
3. **阴极保护**：采用牺牲阳极或外加电流方式保护（较少用于光伏支架）。
4. **材料替代**：采用铝合金或不锈钢替代普通钢材。
5. **环境隔离**：在基础部分采用混凝土包裹钢材。

对于光伏支架，热浸镀锌是最主要的防腐手段。

### 9.2 钢支架热浸镀锌防腐

#### 9.2.1 热浸镀锌原理

热浸镀锌是将经过表面处理的钢材浸入熔融锌液中，使锌与钢材表面发生冶金反应，形成锌-铁合金层和纯锌层的防腐工艺。

镀锌层的防腐蚀机理包括：
1. **物理隔离**：镀锌层将钢材与腐蚀介质隔开。
2. **阴极保护**：锌比铁具有更低的电极电位，在镀层破损处，锌作为阳极优先腐蚀，保护钢材。

#### 9.2.2 镀锌层厚度要求

根据GB/T 13912-2020《金属覆盖层钢铁制件热浸镀锌层技术要求及实验方法》：

| 制件厚度 | 局部镀层厚度（最小值） | 平均镀层厚度（最小值） |
|---------|---------------------|---------------------|
| ≥6mm | 70μm | 85μm |
| 3mm-<6mm | 55μm | 70μm |
| 1.5mm-<3mm | 45μm | 55μm |
| <1.5mm | 35μm | 45μm |

光伏支架的材料厚度通常为2-6mm，根据上表要求：

**工程实际做法**：
- 常规光伏支架：镀锌层平均厚度≥65μm（檩条≥55μm）
- 近海地区支架：桩体镀锌层≥100μm，上部支架≥85μm
- 强腐蚀环境：镀锌层≥85μm

#### 9.2.3 镀锌层质量检验

1. **外观检验**：镀层应平滑、均匀，无漏镀、起皮、气泡等缺陷。
2. **厚度检验**：采用磁性测厚仪检测，测点数量不少于5处/件。
3. **附着力检验**：采用划格法或弯曲试验检验镀层附着力。
4. **均匀性检验**：采用硫酸铜溶液浸渍试验检验镀层均匀性。

#### 9.2.4 镀锌后处理

1. 构件镀锌后应进行钝化处理，提高镀层的耐腐蚀能力。
2. 运输和安装过程中应避免镀层损伤。
3. 镀层损伤处应及时涂覆富锌涂料进行修补。

### 9.3 铝合金支架阳极氧化防腐

#### 9.3.1 阳极氧化原理

阳极氧化是将铝合金作为阳极放入电解液中，通过外加电流在铝合金表面形成一层氧化膜（Al₂O₃）的电化学工艺。氧化膜具有耐腐蚀、耐磨损和装饰性。

#### 9.3.2 氧化膜厚度要求

根据GB/T 5237.2《铝合金建筑型材 第2部分：阳极氧化、着色型材》，氧化膜厚度等级：

| 等级 | 平均膜厚（μm） | 局部膜厚（μm） | 应用 |
|------|--------------|--------------|------|
| AA10 | ≥10 | ≥8 | 一般用途 |
| AA15 | ≥15 | ≥12 | 建筑用途、户外 |
| AA20 | ≥20 | ≥16 | 严酷环境 |
| AA25 | ≥25 | ≥20 | 海洋环境 |

光伏铝合金支架推荐采用AA15级，近海环境推荐AA20级。

#### 9.3.3 质量检验

1. 膜厚检测：采用涡流测厚仪检测。
2. 封孔质量：采用磷铬酸法或硝酸预浸法检验。
3. 外观检测：色泽均匀，无电灼伤、膜层脱落等缺陷。

### 9.4 镀锌铝镁工艺

#### 9.4.1 定义

镀锌铝镁（Zn-Al-Mg）是一种新型镀层工艺，在锌液中加入铝和镁元素，形成Zn-Al-Mg合金镀层。该镀层的耐腐蚀性能显著优于纯锌镀层。

#### 9.4.2 性能优势

| 性能指标 | 镀锌铝镁 | 热浸镀锌（纯锌） |
|---------|---------|----------------|
| 耐盐雾腐蚀 | 2-5倍于纯锌 | 基准 |
| 耐划痕腐蚀 | 自愈能力强 | 无自愈能力 |
| 截面保护 | 切口处有保护 | 切口处无保护 |
| 镀层附着力 | 良好 | 良好 |

#### 9.4.3 应用要求

光伏支架采用镀锌铝镁时，最小镀层重量不应小于275g/m²（双面），对应镀层厚度约20μm（单面）。

### 9.5 涂层防腐体系

#### 9.5.1 涂层类型

对于热浸镀锌后仍需进一步提高防腐能力的构件，或镀层破损后的修补，可采用涂层防腐体系：

| 涂料类型 | 适用场景 | 特点 |
|---------|---------|------|
| 富锌底漆 | 镀锌层修补 | 锌粉含量≥80%，阴极保护 |
| 环氧云铁中间漆 | 中间层 | 屏蔽性好，附着力强 |
| 聚氨酯面漆 | 面层 | 耐候性好，装饰性强 |
| 氟碳面漆 | 高耐候要求 | 耐候性极好 |

#### 9.5.2 涂层配套体系

常用涂层配套体系（干膜总厚度≥120μm）：

| 涂层 | 涂料 | 干膜厚度（μm） |
|------|------|--------------|
| 底漆 | 环氧富锌 | 40-60 |
| 中间漆 | 环氧云铁 | 60-80 |
| 面漆 | 聚氨酯/氟碳 | 40-60 |

### 9.6 连接节点的防腐处理

#### 9.6.1 节点防腐的重要性

连接节点是光伏支架防腐的薄弱环节，原因如下：
1. **螺栓和焊缝处的镀层可能受损**
2. **节点存在缝隙，易积聚水和腐蚀介质**
3. **异种金属接触可能产生电偶腐蚀**

#### 9.6.2 节点防腐措施

1. **螺栓连接**：采用不锈钢螺栓（A2-70或A4-70），或在普通镀锌螺栓上涂覆防松防腐胶。
2. **焊接节点**：焊接完成后清除焊渣，涂刷富锌底漆，再涂覆面漆。
3. **缝隙处**：采用密封胶或环氧树脂封闭。
4. **异种金属接触**：在钢与铝接触处设置绝缘垫片，防止电偶腐蚀。

### 9.7 基础防腐设计

#### 9.7.1 钢材埋入部分的防腐

基础中埋入的钢结构（如预埋螺栓、钢柱脚）应特别注意防腐：

1. **混凝土包裹**：钢材埋入混凝土中的部分，由混凝土碱性环境提供保护。
2. **加大保护层厚度**：埋入混凝土中的钢材，保护层厚度不应小于50mm。
3. **涂刷环氧涂层**：在埋入部分涂刷环氧煤沥青等防腐涂料。

#### 9.7.2 混凝土基础的耐久性

1. **最小水泥用量**：不低于300kg/m³
2. **最大水灰比**：不大于0.55
3. **氯离子含量**：不超过0.1%
4. **抗冻等级**：严寒地区不低于F150，寒冷地区不低于F100

### 9.8 不同腐蚀环境下的防腐等级

根据项目所处环境，推荐如下防腐措施：

| 环境 | 腐蚀等级 | 钢支架防腐 | 连接件 |
|------|---------|----------|------|
| 干燥内陆 | C2-C3 | 镀锌65μm | 镀锌螺栓 |
| 一般内陆 | C3 | 镀锌65μm | 镀锌螺栓 |
| 工业污染区 | C4 | 镀锌85μm | 不锈钢螺栓 |
| 近海（2-5km） | C4-C5 | 镀锌85μm+涂装 | 不锈钢螺栓 |
| 沿海（<2km） | C5-CX | 镀锌100μm+涂装 | 不锈钢螺栓（316） |
| 水上光伏 | C5-CX | 镀锌100μm+特殊涂装 | 不锈钢螺栓（316） |

### 9.9 耐久性维护与检测

#### 9.9.1 定期检测

1. **年度检测**：检查支架外观，记录镀层破损和锈蚀情况。
2. **3-5年全面检测**：使用膜厚仪检测镀层厚度，评估防腐状态。
3. **特殊检测**：台风、暴雨等极端天气后应进行专项检查。

#### 9.9.2 维护措施

1. 发现镀层破损时，及时打磨清理后涂覆富锌涂料修补。
2. 螺栓松动时，重新紧固并涂覆防松胶。
3. 严重锈蚀构件应及时更换。


## 第十章 连接与节点设计

### 10.1 节点设计原则

#### 10.1.1 设计目标

节点设计的目标是在满足结构受力要求的前提下，实现构造简单、施工方便、经济合理。

节点设计的具体目标包括：
1. 传力可靠，承载力不低于被连接构件
2. 构造简单，便于制作和安装
3. 适应构件的热胀冷缩和安装误差
4. 便于防腐处理和后期维护

#### 10.1.2 节点分类

光伏支架的节点按位置可分为：
1. **梁柱节点**：主梁与立柱的连接
2. **梁梁节点**：主梁与檩条的连接
3. **柱脚节点**：立柱与基础的连接
4. **支撑节点**：支撑与主梁/立柱的连接
5. **组件节点**：光伏组件与支架的连接

#### 10.1.3 节点刚度对结构性能的影响

节点的刚度直接影响结构的内力分布和变形特性：

| 节点类型 | 弯矩传递 | 适用场景 |
|---------|---------|---------|
| 刚接 | 完全传递 | 刚架结构、框架结构 |
| 铰接 | 不传递 | 桁架结构、支撑体系 |
| 半刚接 | 部分传递 | 特殊设计 |

光伏支架中，主梁-立柱节点通常设计为铰接或半刚接（通过螺栓连接），柱脚节点通常设计为铰接（锚栓连接）或刚接（预埋钢板焊接）。

### 10.2 梁柱连接节点

#### 10.2.1 常见形式

光伏支架的梁柱连接节点主要有以下形式：

| 形式 | 做法 | 特点 | 应用 |
|------|------|------|------|
| 顶接式 | 主梁直接放置于立柱顶部 | 构造简单，传力明确 | 应用最广泛 |
| 侧接式 | 主梁通过连接板与立柱侧面连接 | 便于调节高度 | 精度要求高时 |
| 抱箍式 | 通过抱箍将主梁固定在立柱上 | 安装方便 | 小型支架 |
| 焊接式 | 主梁与立柱直接焊接 | 刚度大，整体性好 | 永久性结构 |

#### 10.2.2 顶接式节点设计

顶接式节点是光伏支架中最常用的梁柱连接方式。主梁通过连接件（连接板或U形螺栓）固定于立柱顶端。

**设计要点**：
1. 立柱顶部应设置顶板，顶板厚度不宜小于6mm。
2. 主梁与顶板通过螺栓连接，螺栓不少于2个。
3. 连接件应满足抗剪和承压要求。
4. 可采用长圆孔适应安装误差。

#### 10.2.3 节点承载力验算

连接节点的承载力应按以下内容验算：
1. 螺栓的抗剪和承压承载力
2. 连接板的抗弯和抗剪承载力
3. 焊缝的承载力（若采用焊接）
4. 被连接构件的局部承压承载力

### 10.3 梁梁连接节点

#### 10.3.1 主梁与檩条连接

主梁与檩条的连接节点通常采用螺栓连接。连接方式有以下几种：

1. **直接搭接**：檩条直接放置于主梁上，用螺栓固定。适用于C型檩条。
2. **连接板连接**：通过角钢或T型连接板将檩条与主梁连接。适用于各种截面形式。
3. **夹具连接**：采用专用夹具将檩条夹持在主梁上。适用于快速安装。

**设计要点**：
1. 每个节点螺栓数量不宜少于2个。
2. 连接板厚度不宜小于4mm。
3. 檩条端部与主梁之间宜留3-5mm间隙，适应热胀冷缩。

### 10.4 柱脚节点

#### 10.4.1 柱脚形式

光伏支架的柱脚节点有以下形式：

| 形式 | 做法 | 特点 |
|------|------|------|
| 锚栓固定 | 通过预埋锚栓将柱脚板固定于基础 | 简单、广泛采用 |
| 焊接固定 | 立柱直接焊接于预埋钢板上 | 刚度大，不可调 |
| 插入式 | 立柱插入基础预留孔中 | 整体性好 |

#### 10.4.2 锚栓固定柱脚设计

锚栓固定柱脚是最常用的形式，设计要点如下：

1. **柱脚底板**：厚度不宜小于8mm，平面尺寸应满足锚栓布置要求。
2. **锚栓**：直径不宜小于16mm，埋入深度满足锚固要求。
3. **调平螺母**：柱脚底板下设调平螺母，便于调整立柱垂直度。
4. **二次灌浆**：柱脚底板与基础顶面之间用高强度灌浆料填充。

#### 10.4.3 柱脚承载力验算

柱脚节点应验算：
1. 锚栓抗拉承载力
2. 底板下混凝土局部承压承载力
3. 柱脚底板抗弯承载力
4. 锚栓抗剪承载力（如有水平力）

### 10.5 支撑连接节点

#### 10.5.1 支撑端部连接

支撑端部连接节点设计要点：
1. 采用节点板连接，节点板厚度不宜小于6mm。
2. 螺栓数量不应少于2个。
3. 节点板与构件的焊缝应满足承载力要求。
4. 支撑端部应设耳板或叉形连接件，便于销轴连接。

#### 10.5.2 交叉支撑节点

当采用交叉支撑时，两支撑在交叉点处的连接应满足：
1. 交叉点处可采用螺栓连接或焊接，使两支撑协同工作。
2. 交叉点处的连接应能传递支撑之间的相互作用力。
3. 当一支撑受压屈曲时，另一支撑应能独立承受荷载。

### 10.6 组件与支架连接节点

#### 10.6.1 连接方式

光伏组件与支架的连接方式主要有两种：

| 方式 | 做法 | 特点 | 适用 |
|------|------|------|------|
| 螺栓连接 | 通过组件边框上的安装孔用螺栓固定在支架上 | 连接可靠，每个组件4个螺栓 | 主流方式 |
| 压块连接 | 用压块将组件边框压紧在支架檩条上 | 安装快速，适应不同尺寸 | 应用广泛 |

#### 10.6.2 螺栓连接节点设计

1. 螺栓采用不锈钢材质（A2-70）。
2. 螺栓直径：M6-M8。
3. 每个组件至少4个螺栓连接点。
4. 螺栓扭矩：M6为6-8N·m，M8为10-15N·m。
5. 设置平垫圈和弹簧垫圈，防止松动。

#### 10.6.3 压块连接节点设计

1. 压块材质：铝合金或不锈钢。
2. 压块宽度：不宜小于25mm，与组件边框的接触宽度不小于25mm。
3. 压块间距：沿组件长边方向间距不宜大于600mm。
4. 压块螺栓：采用不锈钢螺栓，扭矩应满足要求。

### 10.7 铰接与刚接节点

#### 10.7.1 铰接节点的实现

铰接节点实现方法：
1. **单螺栓连接**：仅用一个螺栓连接构件，允许自由转动。
2. **销轴连接**：采用销轴连接，转动灵活。
3. **长圆孔连接**：采用长圆孔允许相对滑动和转动。

#### 10.7.2 刚接节点的实现

刚接节点实现方法：
1. **全焊缝连接**：构件之间采用全焊透焊缝连接。
2. **高强螺栓连接**：采用摩擦型高强螺栓连接，节点刚度大。
3. **加劲肋加强**：在节点处设置加劲肋，提高节点刚度。

#### 10.7.3 节点刚度对计算简图的影响

节点的实际刚度与计算简图中的假设应一致。如：
- 计算时假设为铰接，节点应设计为铰接
- 计算时假设为刚接，节点应设计为刚接
- 如实际刚度介于两者之间，应采用半刚接分析

### 10.8 节点承载力验算

#### 10.8.1 焊缝承载力

对接焊缝：σ = N/(l_w·t) ≤ f_w
角焊缝：τ_f = N/(h_e·l_w) ≤ f_f^w

#### 10.8.2 螺栓承载力

抗剪：N_v^b ≤ min(N_v^b, N_c^b)
抗拉：N_t^b ≤ N_t^b

复合受力：√[(N_v/N_v^b)² + (N_t/N_t^b)²] ≤ 1

#### 10.8.3 连接板承载力

连接板应验算：
1. 净截面抗拉强度
2. 毛截面抗拉强度（对有孔洞的截面）
3. 抗剪强度（在螺栓孔处）
4. 承压强度（在螺栓孔壁处）

### 10.9 节点构造要求

#### 10.9.1 螺栓布置要求

1. 螺栓间距≥3d（d为螺栓直径）
2. 螺栓至板边距离≥1.5d（切割边）
3. 螺栓至板端距离≥2d
4. 螺栓不应少于2个/节点

#### 10.9.2 焊缝构造要求

1. 最小焊脚尺寸≥4mm
2. 最大焊脚尺寸≤1.2t_min
3. 焊缝计算长度≥8h_f且≥40mm
4. 避免焊缝交叉和应力集中

#### 10.9.3 防腐与防松

1. 所有外露连接件应采用不锈钢或经镀锌处理。
2. 螺栓应设置防松垫圈或涂防松胶。
3. 焊接节点焊接完成后应及时进行防腐处理。


## 第十一章 特殊场景结构设计

### 11.1 柔性（索结构）光伏支架设计

#### 11.1.1 设计依据与适用范围

柔性支架设计应遵循NB/T 11813-2025《索结构光伏支架技术规程》的规定。适用于采用预应力钢索作为主要承重构件的光伏支架结构。

柔性支架适用于：
- 大跨度场景（跨度>20m）
- 复杂地形（山地、沟谷）
- 对地表扰动要求低的生态敏感区
- 农光互补项目（需要大空间净空）

#### 11.1.2 结构体系

柔性支架的结构体系包括：

**（1）索系**

根据索的布置方式，柔性支架的索系可分为：
- **单向索系**：仅沿一个方向布置主索，结构简单，适用于小跨度。
- **双向索系**：纵横两个方向布置索，形成索网，适用于大跨度。
- **交叉索系**：索呈交叉布置，结构稳定，适用于中等跨度。

**（2）支承系统**

包括端部锚锭、立柱、横梁和支撑体系。支承系统将索力传递至地基。

#### 11.1.3 设计荷载与工况

柔性支架的设计荷载与刚性支架相同，但需特别注意以下工况：

1. **风荷载**：柔性支架对风荷载敏感，需考虑脉动风效应和风致振动。
2. **雪荷载**：积雪在柔性支架上的分布可能与刚性支架不同，需专门分析。
3. **温度荷载**：温度变化对索的张力有显著影响，应重点考虑。
4. **初始预应力**：是柔性支架的基本工况，直接决定结构刚度和形态。

#### 11.1.4 索的设计

**（1）索的材料**

- 钢绞线：高强钢丝绞合而成，强度高（f_pk=1770-1860MPa），是主索的常用材料。
- 钢丝绳：由多股钢丝捻制而成，柔韧性好，适用于连接和吊挂。

**（2）索的强度验算**

索的承载力按下式验算：

N ≤ A_p·f_pu / K

其中：N为索拉力设计值，A_p为索的有效截面积，f_pu为索的极限抗拉强度，K为安全系数（不小于2.5）。

**（3）索的疲劳验算**

对于风振频繁的地区，应验算索的疲劳承载力。索的疲劳寿命与应力幅有关，可采用S-N曲线法进行评估。

**（4）索的锚固**

索端锚具分为：
- **热铸锚**：将索端嵌入金属铸体内，用于高应力索。
- **冷铸锚**：通过夹片或树脂锚固，用于低应力索。
- **夹片锚**：利用夹片夹持索体，施工方便，常用。

锚固区的设计应确保锚固可靠，并具有足够的疲劳性能。

#### 11.1.5 风致振动控制

柔性支架在风荷载作用下可能发生以下振动：

| 振动类型 | 机理 | 控制措施 |
|---------|------|---------|
| 涡激振动 | 气流绕过索体产生涡街 | 设置阻尼器、调整索间距 |
| 驰振 | 升力系数随攻角变化 | 增大索张力、设置稳定索 |
| 颤振 | 弯扭耦合振动 | 提高结构刚度、设置交叉支撑 |
| 风雨振 | 雨水形成索表面水线 | 设置阻尼器、索表面处理 |

工程中常采用以下措施抑制风致振动：
1. 在索上安装阻尼器（质量调谐阻尼器）
2. 设置横向稳定索，提高结构阻尼
3. 采用螺旋线缠绕索表面，改变空气动力特性
4. 在相邻索之间设置连接索

#### 11.1.6 预应力张拉施工

柔性支架的预应力张拉施工应遵循以下要求：

1. 张拉设备（千斤顶、油泵）应经过标定。
2. 张拉力控制采用“力值+伸长量”双控法。
3. 张拉顺序应由设计确定，宜对称、分级张拉。
4. 张拉完成后应进行索力检测，偏差不应超过设计值的±10%。

#### 11.1.7 柔性支架的监测

对于大跨度和超跨度柔性支架，建议设置监测系统：
- 索力监测（索力传感器）
- 位移监测（激光位移计）
- 振动监测（加速度计）
- 温度监测（温度传感器）

### 11.2 屋顶光伏支架设计

#### 11.2.1 概述

屋顶光伏支架是安装在建筑物屋面上的光伏支撑系统。与地面支架相比，屋顶支架的设计需特别注意：
1. 既有结构的荷载承载能力
2. 屋面的防水性能
3. 风荷载的上拔效应
4. 施工安全和屋顶作业安全

#### 11.2.2 屋顶类型与支架形式

**（1）混凝土平屋顶**

- 支架形式：配重式支架（无穿孔固定）
- 固定方式：混凝土配重块压重
- 特点：不破坏屋面防水层
- 适用条件：屋面承载力满足配重荷载要求

**（2）彩钢瓦屋顶**

- 支架形式：夹具式支架
- 固定方式：专用夹具夹持在彩钢瓦的波峰或咬合部位
- 特点：不穿孔，不破坏屋面
- 适用条件：彩钢瓦类型适配（直立锁边、角驰型等）

**（3）瓦屋顶（琉璃瓦、陶瓦）**

- 支架形式：挂钩式支架
- 固定方式：使用专用挂钩钩住瓦下挂瓦条或结构檩条
- 特点：需要部分掀瓦作业
- 适用条件：结构檩条承载力满足要求

#### 11.2.3 荷载验算

**（1）既有结构复核**

在既有建筑物上安装光伏支架时，应进行结构安全复核：
- 屋面结构板承载力验算
- 主结构（梁、柱、基础）承载力验算
- 屋面防水层受力分析

**（2）风荷载计算**

屋顶光伏支架的风荷载应按GB 50797-2012和GB 50009的规定计算。屋面边角区域的风荷载集中效应应特别考虑：
- 屋角区域：体型系数可取-1.5
- 屋边区域：体型系数可取-1.3
- 屋面中部：体型系数可取-1.0

#### 11.2.4 防水与构造

1. **配重式支架**：在配重块与屋面之间设置橡胶垫或防水卷材隔离层，防止磨损屋面防水层。
2. **夹具式支架**：夹具与屋面的接触面应设置橡胶垫片，保证密封防水。
3. **挂钩式支架**：挂钩穿过瓦片处应进行防水密封处理，采用耐候密封胶或防水涂料。
4. **电缆穿管**：电缆应穿管敷设，管道穿越屋面处应做防水处理。

#### 11.2.5 施工安全

1. 屋顶作业必须系挂安全带或安全绳。
2. 材料应分散堆放，避免屋面荷载集中。
3. 施工前应编制专项施工方案和安全措施。
4. 恶劣天气（大风、雨雪、高温）应停止作业。

### 11.3 水上光伏支架设计

#### 11.3.1 概述

水上光伏电站是指在湖泊、水库、近海等水体上建设的光伏发电站。根据水深和地质条件，水上光伏支架可分为桩基式和漂浮式两大类。

#### 11.3.2 桩基式水上光伏

桩基式适用于水深较浅（一般<3m）的水域，支架基础采用打入桩或灌注桩。

**设计要点**：
1. **水位变化**：设计应考虑丰水期和枯水期的水位变化，组件最低点距最高洪水位（50年一遇）不应小于0.6m。
2. **防腐**：水位变动区是腐蚀最严重的区域，钢桩在变动区的镀层厚度应达到100μm以上。
3. **防撞**：在通航或可能有漂浮物撞击的区域，应设置防撞设施。
4. **船舶靠泊**：预留运维船只靠泊位置。

#### 11.3.3 漂浮式水上光伏

漂浮式适用于水深较深（>3m）或地质条件复杂的水域，采用浮体支撑光伏组件。

**结构组成**：
1. **浮体**：高密度聚乙烯（HDPE）浮体，提供浮力。
2. **连接件**：不锈钢或镀锌连接件，将浮体连成整体。
3. **支架**：在浮体上安装的金属支架，支撑组件。
4. **锚固系统**：将浮体方阵固定于水体中，防止漂移。

**设计要点**：
1. **浮力计算**：浮体的排水体积应能承载组件、支架和浮体本身的总重量，并考虑风浪、雪荷载等额外荷载。
2. **锚固设计**：锚固系统应能抵抗风、浪、流共同作用下的水平力。
3. **电缆敷设**：采用水密电缆或漂浮电缆，电缆弯曲半径应满足要求。
4. **方阵布置**：漂浮方阵之间应设置通道，便于运维和巡视。

#### 11.3.4 环境适应性设计

1. **水体污染控制**：施工和运维过程中严禁向水体排放污染物。
2. **水生生态保护**：方阵布置时应留出生态通道，减少对水生生物的影响。
3. **抗冰设计**：在寒冷地区，应考虑冰荷载对支架和浮体的作用。

### 11.4 建筑光伏一体化（BIPV）设计

#### 11.4.1 定义与分类

建筑光伏一体化（Building Integrated Photovoltaic，BIPV）是将光伏组件与建筑围护结构（屋面、幕墙、采光顶等）有机结合，使光伏组件同时具备发电功能和建筑构件功能的绿色技术。

BIPV按集成部位可分为：
- **光伏屋面**：光伏组件作为屋面防水层和饰面层
- **光伏幕墙**：光伏组件作为玻璃幕墙的替代材料
- **光伏采光顶**：光伏组件作为透光顶棚
- **光伏遮阳**：光伏组件作为建筑遮阳构件

#### 11.4.2 设计原则

BIPV设计应遵循以下原则：

1. **功能优先**：光伏组件必须首先满足建筑功能（防水、保温、采光、美观），发电功能为附加功能。
2. **协同设计**：光伏系统与建筑围护结构应同步设计、协同施工。
3. **性能匹配**：光伏组件的透光率、颜色、尺寸应与建筑风格协调。
4. **安全可靠**：BIPV系统应满足建筑结构安全、防火安全、用电安全要求。

#### 11.4.3 结构设计要点

1. **荷载传递**：BIPV光伏系统作为建筑围护结构的一部分，其承受的风荷载、雪荷载等应通过支承结构传递至主体结构。
2. **建筑气密性与水密性**：BIPV系统的安装应保证建筑围护结构的气密性和水密性，接缝处应采用密封胶或防水卷材处理。
3. **保温连续性**：BIPV系统不应破坏建筑保温体系的连续性，应避免热桥效应。
4. **温度效应**：BIPV组件在发电时产生热量，应考虑热膨胀对连接节点的影响。

#### 11.4.4 电气与防火安全

1. BIPV系统中的连接线通常敷设于幕墙立柱、横梁等密闭结构中，其温度环境对电线要求较高，应选用耐温等级较高的电缆。
2. BIPV系统的直流侧电压可能高达1500V，应设置快速关断装置。
3. 防火设计应符合建筑防火规范的要求，组件级应设置防火隔离措施。

### 11.5 光伏阳光房设计

#### 11.5.1 定义

光伏阳光房是指采用光伏组件与金属框架搭建的全明非传统建筑，其屋顶或部分墙面由光伏组件构成。阳光房既能提供室内活动空间（保温、隔热、防风雨），又能利用光伏组件发电。

#### 11.5.2 结构特点

光伏阳光房的结构特点：
1. **框架结构**：采用镀锌钢管或不锈钢矩形管焊接或螺栓连接成框架。
2. **透光屋顶**：屋顶采用双玻透光光伏组件，兼顾发电与采光。
3. **围护结构**：四周可采用玻璃、PC板或开放式设计。
4. **防水系统**：组件间采用搭接、导流槽等构造防水。

#### 11.5.3 设计要点

**（1）结构尺寸**
- 前立柱高度：不宜低于2.2m
- 组件安装倾角：不应低于10°（便于排水）
- 跨度：宜控制在3-6m

**（2）荷载考虑**
- 雪荷载：按项目所在地基本雪压取值
- 风荷载：阳光房位于建筑屋顶或地面，风荷载应按实际高度和体型确定
- 活荷载：上人屋面取2.0kN/m²

**（3）防水设计**
- 组件搭接长度：不小于100mm
- 密封胶：采用耐候硅酮密封胶
- 导流槽：沿组件下边缘设置排水导流槽
- 防水节点应进行淋水试验验证

**（4）防腐要求**
- 钢框架：热浸镀锌65μm以上
- 不锈钢框架：可免涂装
- 连接件：采用不锈钢材质

#### 11.5.4 热工设计

1. 阳光房应具有良好的保温性能，双层中空组件或加装保温玻璃。
2. 应设置通风口或天窗，夏季排热。
3. 组件与结构之间应设置隔热垫片，减少热桥。

### 11.6 光伏阳光棚设计

#### 11.6.1 定义

光伏阳光棚是一种更开放、更侧重视野和遮阳功能的轻型光伏建筑结构，在国外常称为光伏凉棚（Solar Pergola）。阳光棚通常具有大跨度、开放式结构，顶部由光伏组件覆盖，起到遮阳、挡雨和发电的作用。

#### 11.6.2 与阳光房的区别

| 项目 | 阳光房 | 阳光棚 |
|------|-------|-------|
| 围护程度 | 四面围护（玻璃或板材） | 四面敞开或半敞开 |
| 功能定位 | 室内活动空间 | 户外休闲遮阳 |
| 保温要求 | 较高 | 较低 |
| 采光要求 | 高（透光组件） | 中等（可选用不透光组件） |
| 防水要求 | 高（室内防雨） | 中等（局部防雨） |

#### 11.6.3 结构形式

光伏阳光棚的常见结构形式：
1. **单坡式**：组件向一侧倾斜，排水顺畅。
2. **双坡式**：中间高、两侧低，造型美观。
3. **平顶式**：倾角较小，适合现代风格。
4. **弧形顶**：造型独特，但施工复杂。

#### 11.6.4 设计要点

1. **悬挑控制**：斜梁和横梁的悬挑不宜超过1500mm。
2. **立柱布置**：立柱间距不宜大于4m，需满足美观和承载力要求。
3. **排水设计**：组件倾角不应小于5°，设置排水沟槽。
4. **防腐**：焊接处应打磨后涂刷红丹底漆和镀锌漆。
5. **紧固件**：优先采用不锈钢材质。

### 11.7 农光互补光伏支架设计

#### 11.7.1 定义与特点

农光互补是指在农业用地上建设光伏电站，同时进行农业种植或养殖，实现土地的综合利用。农光互补支架的设计需兼顾光伏发电和农业生产的双重需求。

**主要特点**：
1. 支架高度较高（≥2.5m），满足农业机械通行和作物生长空间。
2. 支架跨度较大，减少立柱数量，便于农业作业。
3. 组件倾角可能受农业种植需求限制。

#### 11.7.2 设计参数

| 参数 | 典型值 | 说明 |
|------|------|------|
| 离地高度（组件最低点） | ≥2.5m | 满足农业机械通行 |
| 立柱间距 | 4-8m | 减少对种植的干扰 |
| 透光率 | 30%-50% | 满足作物光合需求 |
| 组件排列 | 稀疏排列 | 增加透光 |

#### 11.7.3 结构设计要点

1. **高支架设计**：立柱高度增加，长细比增大，应适当加大截面尺寸或增设支撑。
2. **大跨度设计**：可采用刚性支架（加劲）或柔性支架。
3. **基础要求**：基础埋深应满足抗拔要求，高支架的上拔力更大。
4. **安全净距**：组件与地面之间的净距应满足农用机械高度要求。

### 11.8 山地光伏支架设计

#### 11.8.1 地形特点

山地光伏电站的地形复杂，具有以下特点：
1. **地势起伏**：同一方阵内可能存在较大高差。
2. **坡向变化**：不同坡向的辐照条件不同。
3. **土质不均**：岩土条件变化大。
4. **施工困难**：大型施工设备进场困难。

#### 11.8.2 适应地形措施

1. **顺坡布置**：支架随地形坡度布置，减少土方工程量。
2. **立柱可调节**：采用可调高立柱，适应不同高差。
3. **不等跨布置**：根据地形条件灵活调整立柱间距。
4. **微型基础**：采用微型灌注桩或钢螺旋桩，适应性强。

#### 11.8.3 结构设计要点

1. 山地支架的设计应考虑地形坡度对风荷载的影响。
2. 基础应按实际地质条件进行差异化设计。
3. 应特别注意山体滑坡、泥石流等地质灾害的影响。
4. 施工便道和材料运输方案应在设计阶段统筹考虑。

### 11.9 跟踪支架驱动系统设计

#### 11.9.1 驱动系统组成

跟踪支架的驱动系统是核心部件，其可靠性直接影响支架的运行安全。驱动系统主要由以下部分组成：

1. **电机**：提供动力（直流电机或交流电机）。
2. **减速器**：降低转速、增加扭矩（蜗轮蜗杆减速器或行星减速器）。
3. **驱动轴或推杆**：将动力传递至支架转动部分。
4. **角度传感器**：实时反馈支架角度。
5. **控制器**：根据天文算法和传感器反馈控制电机动作。
6. **限位开关**：防止支架超出设计角度范围。

#### 11.9.2 驱动功率计算

驱动功率应根据以下因素计算：
1. 支架转动惯量
2. 风荷载力矩
3. 摩擦阻力矩
4. 惯性力矩
5. 安全裕量（至少1.5倍）

驱动功率的估算公式：

P = (M_total × ω) / η

其中：M_total为总阻力矩，ω为转动角速度，η为驱动系统效率（0.6-0.8）。

#### 11.9.3 大风保护策略

跟踪支架在大风天气必须采取保护措施：

1. **大风归位**：当风速超过设定值（通常为15-20m/s）时，控制器发出指令使支架归位至水平（0°）或最小迎风角位置。
2. **风速仪布置**：在方阵周边布置风速仪，测风数据实时传输至控制器。
3. **手动紧急归位**：在控制器故障时，应具备手动操作归位功能。
4. **断电保护**：在电源中断时，驱动系统应具备自锁功能，防止支架自由摆动。

#### 11.9.4 可靠性设计

1. 驱动系统关键部件应具有冗余设计（如双电源、备用电机）。
2. 控制器应具有故障自诊断和远程监控功能。
3. 润滑系统应满足长期运行需求，并定期维护。
4. 驱动系统的工作温度范围应适应项目所在地的气候条件。


## 第十二章 工程实践常用术语（江湖黑话）

### 12.1 假梁（Dummy Beam）

#### 12.1.1 定义

假梁是指不主要承担竖向荷载，仅起辅助安装、连接或传递局部荷载作用的梁。在实际工程中，假梁多用于支架系统的辅助连接件。

假梁与主梁的区别在于其“辅助性”——假梁不作为主要承力构件，其失效不会导致结构整体倒塌，但可能影响支架的安装精度或局部功能。

#### 12.1.2 应用场景

光伏工程中假梁的常见应用场景：
1. **横向连接梁**：在相邻支架之间设置假梁，用于敷设电缆或作为检修通道。
2. **临时安装梁**：在组件安装过程中用于临时支撑组件，安装完成后可拆除或保留。
3. **设备支撑梁**：用于支撑汇流箱、逆变器、接线盒等电气设备。
4. **防坠落横梁**：在支架下方设置假梁，用于悬挂防坠落安全网。

#### 12.1.3 设计要点

1. 假梁虽然不承担主要荷载，但仍应按实际可能承受的荷载进行验算。
2. 在强风区或近海环境，假梁的设计荷载应额外考量，防止在极端天气下脱落。
3. 假梁的连接节点应可靠，不得因振动而松动。
4. 假梁的材料和防腐要求应与主结构一致。

### 12.2 外挑（Outward Overhang）

#### 12.2.1 定义

外挑是指结构构件端部超出其支撑点向外延伸的部分，即悬挑结构。外挑部分由悬挑端至支撑点的距离称为悬挑长度。

外挑是光伏支架设计中常见的结构形式，尤其是在屋顶光伏和阳光房中应用广泛。

#### 12.2.2 外挑的设计控制

外挑结构需特别注意以下问题：

1. **倾覆风险**：外挑长度过大可能引起结构整体倾覆，尤其是配重式支架。
2. **挠度控制**：外挑端的挠度通常大于简支梁，需严格控制。
3. **连接节点受力**：外挑根部的连接节点承受最大的弯矩和剪力，是设计的关键。

#### 12.2.3 外挑长度的经验值

| 结构类型 | 外挑长度限制 | 说明 |
|---------|------------|------|
| 固定支架主梁 | 不宜超过跨度的1/4 | 保证结构稳定 |
| 阳光房斜梁 | 不宜超过1500mm | 控制挠度 |
| 屋顶支架 | 不宜超过500mm | 防止倾覆 |
| 组件悬挑 | 不宜超过200mm | 组件自身悬挑限制 |

#### 12.2.4 计算要点

外挑构件的计算应在外挑端和支撑点分别进行：
1. 外挑端：验算最大挠度
2. 支撑点：验算支座反力和连接节点承载力
3. 抗倾覆：验算整体稳定性

### 12.3 内挑（Inward Overhang）

#### 12.3.1 定义

内挑是指构件跨中向内的延伸部分（如加强撑的延伸段、水平支撑在跨内的连接点）。内挑在现行规范中没有对应的官方术语，属于工程口头用语。

内挑不同于外挑，它指向结构内部的延伸，通常用于构件的连接和加强。

#### 12.3.2 典型应用

1. **加强撑内挑段**：斜撑在跨中向内延伸，与主梁连接的节点段。
2. **水平支撑内挑**：水平支撑在跨中向内挑出，连接相邻檩条。
3. **设备支托内挑**：用于支托电气设备的内部悬臂。

#### 12.3.3 设计要点

1. 内挑构件虽然悬挑长度较短，但连接节点的抗扭性能是关键。
2. 内挑段的受力状态复杂，可能在扭矩和剪力共同作用下破坏。
3. 应通过节点加劲或增加壁厚来提升抗扭承载力。

### 12.4 上人/不上人（Accessible/Non-Accessible）

#### 12.4.1 定义

上人/不上人是用于描述屋顶或屋面是否需要承载人的活动重量的术语。该术语直接影响了活荷载的取值。

- **上人屋面**：允许人员在其上行走、活动的屋面。
- **不上人屋面**：不允许人员在其上行走、活动的屋面。

#### 12.4.2 荷载取值

根据GB 50009《建筑结构荷载规范》：

| 类型 | 活荷载标准值 | 备注 |
|------|------------|------|
| 上人屋面 | 2.0 kN/m² | 可用于住宅、办公等 |
| 不上人屋面 | 0.5 kN/m² | 不可作为活动场地 |
| 屋顶花园 | 3.0 kN/m² | 按实际可能的人员密度 |

#### 12.4.3 光伏屋面的判定

在实际工程中，安装了光伏组件的屋顶：
- 如果设置有检修通道且可以安全通行，应按“上人”荷载取2.0kN/m²
- 如果只有专业维护人员借助安全设备才能到达，可按“不上人”取0.5kN/m²，但施工阶段荷载应按实际施工方法确定

建议在光伏屋顶设计中，即使按不上人屋面取值，也应在设计文件中明确“不得擅自上人”的安全警示。

### 12.5 吃风（Wind Load Path）

#### 12.5.1 定义

“吃风”是工程现场的口头用语，指结构受力体系能顺畅地将风荷载传递到基础，而不被“憋住”。这是一个定性的、经验性的评价，而非定量指标。

一个“吃风”好的支架方案，意味着：
1. 传力路径短、直接，没有多余的传递环节。
2. 构件刚度匹配合理，不会在某个环节产生刚度突变。
3. 荷载能快速扩散，不会在局部形成应力集中。
4. 基础能有效承担最终传递来的荷载。

#### 12.5.2 “吃风”的设计原则

1. **传力路径最短化**：从风荷载作用点到基础的路径越短，荷载传递越直接。
2. **刚度渐变**：相邻构件的刚度不宜突变过大。
3. **水平力可靠传递**：风荷载的水平分力应有专门的传递路径（如支撑体系）。
4. **避免“孤岛”** ：不应有孤立受力的构件，每个构件都应有明确的荷载传递路径。

#### 12.5.3 “不吃风”的表现

以下情况被认为是“不吃风”的表现：
1. 风荷载在某个构件上产生很大的局部应力，但该应力无法有效传递到其他构件。
2. 结构在风荷载下产生异常振动，说明刚度不足或阻尼过小。
3. 某些构件的连接节点先于构件本身破坏，说明节点承载力不足。

### 12.6 甩头（Cantilever End）

#### 12.6.1 定义

“甩头”与“外挑”意思相近，专指构件末端自由悬挑的部分，是工程现场对悬挑端的口语称呼。

与“外挑”相比，“甩头”更侧重于口头沟通，常用于现场技术交底和问题讨论。

#### 12.6.2 工程应用

在现场交流中，“甩头”常用于以下场景：
- “这根檩条甩头太长了，得加支撑” —— 表示檩条的悬挑端过长。
- “甩头那里要加一块垫板” —— 表示在悬挑端需要加强处理。
- “甩头弯了，得换一根” —— 表示悬挑端发生了过大变形。

#### 12.6.3 设计与施工注意事项

1. 甩头长度应严格控制，一般不宜超过构件跨度的1/4。
2. 甩头端的荷载可能会产生较大的位移，应进行专项验算。
3. 甩头端的连接节点应特别注意，防止在振动中松动。

### 12.7 打桩/锤桩与引孔

#### 12.7.1 打桩/锤桩

“打桩”或“锤桩”是指用打桩机直接将钢桩或混凝土桩打入地基的施工方法。光伏工程中常用的打桩方式包括：

- **锤击法**：用柴油锤或液压锤将桩打入地层。适用于预制混凝土桩、型钢桩等。
- **振动法**：利用振动锤的振动将桩沉入地层。适用于钢螺旋桩、钢管桩等。

打桩施工速度快，一般不需要额外处理，可以直接使用。但打桩会对周围土体产生挤压和振动，应关注对邻近建筑物和已施工桩的影响。

#### 12.7.2 引孔

“引孔”是指先使用钻机在预定桩位钻孔，然后将桩放入孔中的施工工艺。

引孔的适用条件：
- 坚硬地层（如密实砂层、风化岩层）
- 振动敏感区域（邻近建筑物）
- 大直径桩的施工

引孔后放桩的承载力主要依靠侧摩阻力提供，桩端承载力贡献较小。

#### 12.7.3 两种工艺的比较

| 项目 | 打桩（锤击/振动） | 引孔后放桩 |
|------|-----------------|----------|
| 施工速度 | 快 | 较慢 |
| 设备要求 | 打桩机 | 钻机+打桩机 |
| 承载力来源 | 挤土效应+侧摩阻力 | 侧摩阻力为主 |
| 适用地层 | 软土至中等硬度 | 坚硬地层 |
| 对周围影响 | 振动大、挤土效应 | 振动小 |
| 成本 | 较低 | 较高 |

### 12.8 放倒（Lay Down Installation）

#### 12.8.1 定义

“放倒”是指在地面（或平整场地上）将光伏阵列（含支架和组件）组装成一个整体后，再用吊车整体“放倒”至安装位置的施工方法。

该工法的正式名称为“地面拼装整体吊装法”或“地面预组装法”，但在现场口语中统称为“放倒”。

#### 12.8.2 工艺优势

放倒工法的主要优势：
1. **减少高空作业**：绝大部分组装工作在地面完成，安全风险显著降低。
2. **提高安装精度**：地面组装更容易控制尺寸精度和对齐度。
3. **缩短安装工期**：避免了大量高空逐件安装的时间消耗。
4. **节省人工成本**：高空作业的人工成本通常高于地面作业。

#### 12.8.3 适用条件与注意事项

1. 适用于场地开阔、吊车可以进入的平地项目。
2. 整体重量应考虑吊车的起重能力，分块大小应满足吊装要求。
3. 整体刚度应满足起吊过程中的变形控制要求，必要时设置临时加固。
4. 起吊方案和索具布置应经过计算。
5. 起吊过程中应有专人指挥，确保安全。

### 12.9 其他常见行话

#### 12.9.1 “吃土”

指基础埋深较浅或基础与土体未充分结合，在风荷载作用下基础发生上拔或位移的现象。在强风地区，若基础“吃土”不深，容易造成支架倾覆。

#### 12.9.2 “走线”

指在支架上敷设电缆的过程。“走线”是否合理直接影响电缆的安全和使用寿命，设计中应预留走线通道和固定点。

#### 12.9.3 “偏腿”

指立柱安装时垂直度偏差过大，造成“一条腿长一条腿短”的现象。“偏腿”会导致支架受力不均，产生附加弯矩。

#### 12.9.4 “放样”

指在施工前进行实地测量和标记，确定基础位置和标高的工序。准确的“放样”是保证支架安装精度的前提。


## 第十三章 施工技术指导

### 13.1 施工准备

#### 13.1.1 技术准备

1. **图纸会审**：施工前应组织图纸会审，由设计单位进行技术交底，施工单位应充分理解设计意图和技术要求。
2. **施工方案编制**：编制专项施工方案，包括施工工艺、施工进度计划、安全措施、应急预案等。
3. **材料计划**：根据图纸编制材料采购计划，明确材料规格、数量和技术要求。
4. **测量放线**：确定光伏方阵的定位基准线、基础位置和标高。

#### 13.1.2 现场准备

1. **场地平整**：清除场地内的障碍物，平整施工区域。
2. **施工道路**：修建施工便道，确保材料运输和机械通行。
3. **临时设施**：搭建临时仓库、工具房、生活区等。
4. **水电供应**：确保施工用水和用电。

#### 13.1.3 材料检验

1. 进场材料应具有质量合格证明文件。
2. 钢材：检查规格、材质证明和外观质量。
3. 镀锌件：检查镀锌层厚度和外观。
4. 螺栓：检查规格、性能等级和外观。
5. 混凝土：检查配合比报告和原材料检验报告。

### 13.2 支架加工制作

#### 13.2.1 切割与下料

1. 钢材下料应采用机械切割（如剪板机、锯床）或自动切割（如数控等离子切割）。
2. 下料尺寸偏差：长度偏差±2mm，角度偏差±0.5°。
3. 切割面应平整，无毛刺、裂纹和氧化铁皮。
4. 构件编号应在下料后立即标记，避免混淆。

#### 13.2.2 螺栓孔加工

1. 螺栓孔应采用钻床或数控冲床加工，不得用割炬割孔。
2. 孔位偏差：同组螺栓孔之间偏差±1mm。
3. 孔径偏差：+0.5mm～0mm。
4. 孔边缘应无毛刺，孔壁应光滑。

#### 13.2.3 焊接加工

1. 焊工应持有有效的焊工资格证书。
2. 焊接前应清除焊接区域的氧化皮、油污、水分等杂物。
3. 焊接工艺参数应符合焊接工艺评定报告的要求。
4. 焊接完成后应清除焊渣，检查焊缝外观。
5. 重要焊缝应进行无损检测。

#### 13.2.4 热浸镀锌

1. 镀锌前应彻底清除钢材表面的油污、锈蚀和氧化皮。
2. 镀锌温度应控制在440-460°C之间。
3. 镀锌时间应根据钢材厚度确定，一般2-5分钟。
4. 镀锌后应进行钝化处理。
5. 镀锌层厚度检验：在构件的不同位置测厚，取平均值。

### 13.3 支架安装工艺

#### 13.3.1 安装顺序

光伏支架的安装顺序如下：

```
1. 基础施工 → 2. 基础验收
       ↓
2. 立柱安装 → 4. 立柱垂直度调整
       ↓
3. 主梁安装 → 6. 主梁水平度调整
       ↓
4. 支撑安装 → 8. 整体稳定性检查
       ↓
5. 檩条安装 → 10. 檩条平整度检查
       ↓
6. 组件安装 → 12. 电气连接
       ↓
7. 整体验收
```

#### 13.3.2 立柱安装

1. 立柱安装前，检查基础顶面标高和平整度。
2. 将立柱吊装至基础顶面，对准预埋螺栓。
3. 使用调平螺母调整立柱垂直度。
4. 垂直度允许偏差：不大于柱高的1/1000，且不大于10mm。
5. 调整完成后拧紧螺母，进行二次灌浆（如需要）。

#### 13.3.3 主梁与檩条安装

1. 主梁安装前，检查立柱顶标高，必要时进行调整。
2. 将主梁吊装至立柱顶部，通过连接板或U形螺栓固定。
3. 主梁的水平度偏差：不大于长度的1/1000。
4. 檩条安装应按从一端到另一端的顺序进行。
5. 檩条与主梁的螺栓连接应紧固，不得漏拧。

#### 13.3.4 倾斜角度控制

1. 支架的倾斜角度应符合设计要求，偏差不应超过±2°。
2. 倾角测量可采用角度尺、倾角仪或激光水平仪。
3. 同一排支架的倾角应一致，偏差不超过±1°。
4. 倾角调整应在主梁安装阶段完成，组件安装前复核。

#### 13.3.5 安装质量检查

| 检查项目 | 允许偏差 | 检测方法 |
|---------|---------|---------|
| 立柱垂直度 | ≤H/1000，≤10mm | 线垂+钢尺 |
| 主梁水平度 | ≤L/1000，≤10mm | 水准仪 |
| 支架倾角 | ±2° | 倾角仪 |
| 螺栓扭矩 | ±10% | 扭矩扳手 |
| 相邻立柱间距 | ±5mm | 钢尺 |
| 对角线差 | ≤10mm | 钢尺 |

### 13.4 基础施工

#### 13.4.1 微型灌注桩施工

1. **定位放线**：根据设计图纸确定桩位，用木桩或钢筋标记。
2. **成孔**：采用小型钻机成孔，孔径偏差±20mm，孔深偏差±50mm。
3. **清孔**：清除孔底沉渣，沉渣厚度≤50mm。
4. **钢筋笼制作与安装**：按设计要求制作钢筋笼，放入孔中，保证保护层厚度。
5. **灌注**：灌注混凝土（或水泥砂浆），连续灌注，分层振捣。
6. **预埋螺栓安装**：在混凝土初凝前安装预埋螺栓，精确定位。
7. **养护**：覆盖养护，养护期不少于7天。

#### 13.4.2 钢螺旋桩施工

1. **定位放线**：确定桩位。
2. **旋入**：采用专用旋桩机将螺旋桩旋入土层至设计深度。
3. **深度控制**：以设计桩顶标高为准，偏差±20mm。
4. **扭矩控制**：旋入扭矩应满足设计要求，作为承载力控制的辅助参数。
5. **垂直度控制**：桩身垂直度偏差≤1%。

#### 13.4.3 混凝土扩展基础施工

1. **基坑开挖**：按设计尺寸开挖，坑底应平整，无扰动。
2. **垫层施工**：铺设碎石垫层或素混凝土垫层。
3. **钢筋绑扎**：按图纸绑扎钢筋网，保护层垫块就位。
4. **模板安装**：安装侧模，加固牢靠。
5. **混凝土浇筑**：连续浇筑，振捣密实。
6. **预埋螺栓安装**：在混凝土浇筑前安装预埋螺栓，浇筑过程中不得移位。
7. **拆模养护**：混凝土达到规定强度后拆模，覆盖养护。

#### 13.4.4 基础验收标准

| 检查项目 | 允许偏差 | 检测方法 |
|---------|---------|---------|
| 基础位置 | ±10mm | 全站仪 |
| 基础顶面标高 | ±10mm | 水准仪 |
| 预埋螺栓位置 | ±5mm | 钢尺 |
| 预埋螺栓外露高度 | ±5mm | 钢尺 |
| 基础截面尺寸 | ±10mm | 钢尺 |
| 混凝土强度 | ≥设计强度 | 试块抗压 |

### 13.5 组件安装

#### 13.5.1 安装前检查

1. 检查组件外观，无破损、裂纹、气泡等缺陷。
2. 检查组件背面接线盒，无破损，连接线完好。
3. 检查组件边框，无变形、锈蚀。
4. 核对组件型号和规格，与图纸一致。

#### 13.5.2 组件安装顺序

1. 从方阵的一端开始，逐块安装组件。
2. 将组件放置在檩条上，初步对准安装位置。
3. 安装螺栓或压块，初步固定。
4. 调整组件的间距和对齐度。
5. 紧固螺栓或压块至规定扭矩。
6. 连接组件之间的电缆。

#### 13.5.3 安装质量要求

1. 组件之间的间隙：均匀一致，偏差≤2mm。
2. 组件的平整度：相邻组件之间的高差≤2mm。
3. 组件的直线度：整排组件的边线偏差≤5mm。
4. 螺栓扭矩：按设计要求，偏差±10%。
5. 压块紧固：压块应压紧组件边框，不得松动。

#### 13.5.4 安装注意事项

1. 安装过程中不得踩踏组件表面，防止隐裂。
2. 组件安装后应及时连接接地线。
3. 连接线应固定可靠，不得悬垂摆动。
4. 插拔连接器时应确认极性正确，不得短路。
5. 安装时注意天气情况，大风天气应暂停作业。

### 13.6 预应力张拉施工（柔性支架）

#### 13.6.1 张拉准备

1. 张拉设备（千斤顶、油泵、压力表）应经过标定，且在有效期内。
2. 索的端部锚具应安装到位。
3. 张拉平台应安全可靠。
4. 张拉前应检查索的安装状态，确保无扭曲、打结。

#### 13.6.2 张拉过程

1. **分级张拉**：按设计要求的张拉程序进行，通常分3-5级张拉至设计值。
2. **双控法**：以张拉力为主控参数，以索伸长量为校核参数。
3. **持荷稳定**：张拉至设计值后，持荷1-2分钟，再锁紧锚具。
4. **对称张拉**：应对称进行张拉，避免结构偏心受力。

#### 13.6.3 质量验收

1. 张拉力偏差：不超过设计值的±5%。
2. 伸长量偏差：不超过计算伸长量的±6%。
3. 锚固可靠：锚具无滑移、裂纹。
4. 索的线形：符合设计要求。

### 13.7 冬雨季施工措施

#### 13.7.1 冬季施工

1. 混凝土施工：采用热水搅拌，添加防冻剂，覆盖保温。
2. 焊接施工：采取预热措施，焊接环境温度不低于-5°C。
3. 基础施工：优先采用预制桩或钢螺旋桩，减少湿作业。
4. 安全措施：防滑、防冻、防坠落。

#### 13.7.2 雨季施工

1. 基坑开挖后应及时浇筑，防止雨水浸泡。
2. 混凝土浇筑应避开大雨天气，如遇下雨应及时覆盖。
3. 电气设备和工具应有防雨措施。
4. 雨后施工前应检查地基和基础的稳定性。

### 13.8 施工安全与质量控制

#### 13.8.1 安全管理

1. 项目应设置专职安全员，负责现场安全管理。
2. 所有施工人员应接受安全教育和培训。
3. 高处作业必须系挂安全带，设置安全网。
4. 吊装作业应有专人指挥，吊车应经过检验。
5. 施工现场应设置安全警示标志。

#### 13.8.2 质量控制体系

1. 建立“三检制”：自检、互检、专检。
2. 每道工序完成后应进行检查，验收合格后方可进入下一道工序。
3. 隐蔽工程应在隐蔽前进行验收。
4. 建立质量追溯制度，所有工序应有记录。

#### 13.8.3 关键质量控制点

| 控制点 | 控制内容 | 控制标准 |
|-------|---------|---------|
| 基础施工 | 位置、标高、混凝土强度 | 设计图纸要求 |
| 支架安装 | 垂直度、水平度、倾角 | 设计图纸要求 |
| 螺栓连接 | 扭矩、防松措施 | 设计扭矩值 |
| 焊接施工 | 焊缝外观、无损检测 | GB 50205 |
| 组件安装 | 间距、平整度、电气连接 | 设计图纸要求 |


## 第十四章 质量验收标准

### 14.1 验收依据

光伏支架工程质量验收应以以下标准和文件为依据：
1. **GB 50205-2020《钢结构工程施工质量验收标准》**
2. **GB 50797-2012《光伏发电站设计规范》**
3. **NB/T 10115-2018《光伏支架结构设计规程》**
4. **GB 51101-2016《太阳能发电站支架基础技术规范》**
5. **经审批的设计图纸和技术文件**
6. **工程合同**

### 14.2 原材料验收

#### 14.2.1 钢材验收

1. 钢材应有质量合格证明文件，包括材质证明书。
2. 钢材表面不得有裂纹、结疤、折叠、分层等缺陷。
3. 钢材的化学成分和力学性能应符合设计要求和标准规定。
4. 钢材应按规格、品种分开堆放，做好标识。

#### 14.2.2 焊接材料验收

1. 焊条、焊丝应有质量合格证明文件。
2. 焊条应保存在干燥环境中，使用前按说明书进行烘焙。
3. 焊剂应干燥、无杂质。

#### 14.2.3 紧固件验收

1. 螺栓应有质量合格证明文件。
2. 螺栓的规格、性能和等级应符合设计要求。
3. 螺栓表面不得有裂纹、毛刺、锈蚀等缺陷。

#### 14.2.4 镀锌件验收

1. 镀锌构件应有镀锌质量证明文件。
2. 镀层应均匀、光滑，无漏镀、起泡、剥落。
3. 镀层厚度应满足设计要求。

### 14.3 支架加工验收

#### 14.3.1 几何尺寸偏差

| 检查项目 | 允许偏差 | 检查方法 |
|---------|---------|---------|
| 构件长度 | ±3mm | 钢尺 |
| 构件直线度 | L/1000，≤5mm | 拉线+钢尺 |
| 截面尺寸偏差 | ±2mm | 游标卡尺 |
| 螺栓孔距偏差 | ±1mm | 钢尺 |
| 孔位偏差 | ±0.5mm | 游标卡尺 |

#### 14.3.2 焊接质量验收

1. 焊缝外观检查：焊缝应成型良好，无裂纹、气孔、夹渣、未熔合等缺陷。
2. 焊缝尺寸检查：焊缝的焊脚尺寸和长度应满足设计要求。
3. 无损检测：设计要求进行无损检测的焊缝，应进行UT（超声波）或RT（射线）检测。

### 14.4 支架安装验收

#### 14.4.1 安装偏差

| 检查项目 | 允许偏差 | 检查方法 |
|---------|---------|---------|
| 立柱垂直度 | H/1000，≤10mm | 线垂+钢尺 |
| 主梁水平度 | L/1000，≤10mm | 水准仪 |
| 支架倾角 | ±2° | 倾角仪 |
| 相邻立柱间距 | ±5mm | 钢尺 |
| 对角线差 | ≤10mm | 钢尺 |
| 组件安装平面度 | ±5mm | 拉线+钢尺 |

#### 14.4.2 螺栓连接验收

1. 螺栓应紧固，无松动。
2. 外露丝扣不应少于2扣。
3. 高强度螺栓的预拉力应符合设计要求。
4. 防松措施（弹簧垫圈、防松胶等）应到位。

#### 14.4.3 防腐验收

1. 检查镀层外观和厚度。
2. 焊接节点处的防腐修补应到位。
3. 涂装应均匀，无漏涂、流挂。

### 14.5 基础验收

#### 14.5.1 混凝土基础验收

1. 混凝土强度：以试块抗压强度评定，达到设计强度等级。
2. 基础位置偏差：±10mm。
3. 基础顶面标高偏差：±10mm。
4. 预埋螺栓位置偏差：±5mm。
5. 预埋螺栓外露高度偏差：±5mm。

#### 14.5.2 桩基础验收

1. 桩位偏差：±20mm。
2. 桩顶标高偏差：±20mm。
3. 桩身垂直度偏差：≤1%。
4. 单桩承载力：静载试验或高应变检测合格。

### 14.6 防腐质量验收

#### 14.6.1 镀锌层验收

1. 外观：镀层均匀，无漏镀、起泡、剥落。
2. 厚度：采用磁性测厚仪检测，测点不少于5处/件。
3. 附着力：划格法或弯曲试验合格。

#### 14.6.2 涂装验收

1. 外观：颜色均匀，无流挂、漏涂、皱皮。
2. 厚度：测厚仪检测，干膜厚度符合设计要求。
3. 附着力：划格法试验，附着力等级不低于2级。

### 14.7 跟踪支架调试验收

1. 跟踪支架安装完成后应进行系统调试。
2. 调试内容包括：角度传感器校准、限位开关测试、控制器参数设置。
3. 跟踪精度测试：在设定角度下测量实际角度，偏差不超过±2°。
4. 大风保护测试：模拟大风信号，验证归位功能。
5. 通讯测试：控制器与中央监控系统的通讯正常。
6. 连续运行测试：连续运行24小时，无异常。

### 14.8 柔性支架专项验收

1. 索的安装位置和线形符合设计要求。

5. 风致振动测试：在设计风速范围内，结构振动在可接受范围内。

## 第十六章 运维阶段结构安全评估

### 16.1 运维期结构检测内容

#### 16.1.1 检测频率

| 检测类型 | 检测频率 | 内容 |
|---------|---------|------|
| 日常巡检 | 每月1次 | 外观检查、螺栓松动检查 |
| 年度检测 | 每年1次 | 全面结构检测 |
| 专项检测 | 极端天气后 | 风灾/雪灾后的结构评估 |
| 全面评估 | 每5年1次 | 包括检测、评估和加固建议 |

#### 16.1.2 日常巡检内容

1. 支架外观：检查是否有明显的变形、锈蚀。
2. 螺栓连接：检查是否有松动、脱落。
3. 组件安装：检查是否有松动、异响。
4. 基础：检查是否有沉降、开裂。
5. 接地：检查接地线是否完好。

### 16.2 支架锈蚀检测与评估

#### 16.2.1 锈蚀检测方法

1. **目视检测**：观察锈蚀程度和范围。
2. **膜厚检测**：使用磁性测厚仪检测镀层剩余厚度。
3. **局部打磨检测**：在代表性位置打磨表面，测量剩余钢材厚度。
4. **超声波测厚**：无损检测钢材厚度。

#### 16.2.2 锈蚀等级评定

| 等级 | 锈蚀程度 | 处理措施 |
|------|---------|---------|
| 1级 | 轻微锈蚀，仅表面变色 | 清洁后涂装防护 |
| 2级 | 明显锈蚀，出现锈斑 | 打磨除锈后涂装 |
| 3级 | 严重锈蚀，镀层剥落 | 局部更换构件 |
| 4级 | 截面显著减小 | 立即更换 |

#### 16.2.3 锈蚀后的承载力评估

当构件因锈蚀导致截面损失超过10%时，应进行承载力复核验算。截面损失率按剩余截面与原截面的比值计算。

### 16.3 连接节点松动检测

#### 16.3.1 检测方法

1. **扭矩检测**：使用扭矩扳手检查螺栓扭矩。
2. **敲击检测**：用小锤敲击螺栓头，听声音判断是否松动。
3. **目视检测**：检查防松标记是否错位。

#### 16.3.2 处理措施

1. 发现松动螺栓应立即紧固至设计扭矩。
2. 频繁松动的螺栓应更换为防松型螺栓或涂防松胶。
3. 焊接节点出现裂纹应及时补焊。

### 16.4 基础沉降观测

#### 16.4.1 观测方法

1. 在基础顶面设置沉降观测点。
2. 采用水准仪定期测量观测点标高变化。
3. 计算相邻基础之间的沉降差。

#### 16.4.2 沉降限值

1. 相邻基础沉降差≤0.001L（L为基础间距），固定支架。
2. 相邻基础沉降差≤0.0005L，跟踪支架。
3. 最大沉降量≤50mm。

#### 16.4.3 处理措施

1. 沉降量超限时，应查明原因。
2. 轻微的沉降差可通过调整支架连接件补偿。
3. 严重的沉降问题需进行基础加固。

### 16.5 风灾与雪灾后的结构评估

#### 16.5.1 风灾后评估

强风（台风、飓风）过后应立即进行结构评估：
1. 检查支架是否整体变形或倾斜。
2. 检查组件是否有破裂或脱落。
3. 检查连接节点是否开裂或松动。
4. 检查基础是否有上拔或位移。
5. 对受损结构进行安全性评估，确定修复方案。

#### 16.5.2 雪灾后评估

大雪后应关注：
1. 检查支架是否有异常变形。
2. 检查组件是否有压裂。
3. 及时清除组件上的积雪，防止荷载累积。
4. 检查雪荷载是否已对结构造成不可逆损伤。

### 16.6 结构加固与修复

#### 16.6.1 加固方案

1. **截面增大**：在原有构件上加焊附加钢材。
2. **增设支撑**：增加支撑构件，减小计算长度。
3. **更换构件**：严重锈蚀或损坏的构件直接更换。
4. **基础加固**：增大基础截面或增设锚杆。

#### 16.6.2 修复材料

1. 与原有结构采用相同的材料规格。
2. 修复用钢材应有质量证明文件。
3. 修复用涂料应与原有涂层配套。


## 第十七章 经济分析与方案比选

### 17.1 支架方案技术经济比选

#### 17.1.1 比选指标体系

光伏支架方案的经济比选应从以下方面进行：

| 指标类别 | 具体指标 | 说明 |
|---------|---------|------|
| 经济指标 | 单位投资（元/W） | 初始投资成本 |
| 经济指标 | 度电成本LCOE（元/kWh） | 综合经济性 |
| 经济指标 | 内部收益率IRR（%） | 投资回报率 |
| 技术指标 | 发电量（kWh/kWp） | 发电效率 |
| 技术指标 | 可靠性 | 故障率、可用率 |
| 技术指标 | 适应性 | 对场地条件的适应能力 |
| 环境指标 | 环境影响 | 对生态的影响 |

#### 17.1.2 比选方法

1. **投资回收期法**：比较各方案的静态投资回收期。
2. **净现值法**：比较各方案的全生命周期净现值。
3. **LCOE法**：比较各方案的平准化度电成本，是最常用的核心指标。
4. **综合评分法**：对多指标赋权后综合评价。

### 17.2 固定支架与跟踪支架经济性对比

#### 17.2.1 成本对比

| 支架类型 | 增量投资（相对固定） | 发电增益 | 维护成本 |
|---------|-------------------|---------|---------|
| 固定支架 | 基准 | 基准 | 基准 |
| 可调支架 | +5%~+10% | +3%~+8% | 低 |
| 平单轴跟踪 | +30%~+50% | +10%~+20% | 中 |
| 双轴跟踪 | +80%~+120% | +20%~+35% | 高 |

#### 17.2.2 选型建议

1. 辐照资源好的地区（年辐照>1500kWh/m²），跟踪支架经济性较好。
2. 电价高的地区，跟踪支架的收益增量更显著。
3. 土地成本高的地区，跟踪支架通过提高单位面积发电量可降低土地成本占比。
4. 辐照一般、电价较低的地区，固定支架的经济性更好。

### 17.3 钢支架与铝合金支架经济性对比

| 项目 | 钢支架 | 铝合金支架 |
|------|-------|----------|
| 材料成本 | 低（基准） | 高（2-3倍） |
| 重量 | 重 | 轻（约钢的1/3） |
| 防腐成本 | 镀锌费用 | 阳极氧化费用 |
| 运输成本 | 较高 | 较低 |
| 安装成本 | 较高（重量大） | 较低（重量轻） |
| 维护成本 | 较高（防腐维护） | 较低（耐腐蚀） |

选型建议：一般项目优先选用钢支架；屋顶荷载受限、腐蚀环境、外观要求高的项目可考虑铝合金支架。

### 17.4 全生命周期成本分析（LCC）

全生命周期成本分析应考虑以下阶段：

1. **初始投资**：材料费、加工费、运输费、安装费。
2. **运营成本**：维护费、检修费、故障处理费。
3. **发电收益**：各方案的发电量差异产生的收益差异。
4. **残值**：支架的可回收价值（钢材可回收利用）。

LCC分析有助于选择全生命周期内最经济的方案，而非仅关注初始投资。

### 17.5 优化设计的经济效益

通过优化设计，可在保证安全的前提下降低造价：

1. **截面优化**：根据内力分布优化构件截面，避免“大材小用”。
2. **结构优化**：合理布置支撑，缩短计算长度，减小截面。
3. **材料优化**：合理选择钢材牌号，高强钢在特定条件下更经济。
4. **标准化设计**：减少构件种类，提高加工效率，降低制作成本。

## 附录E 设计计算公式汇总

### E.1 荷载计算

**风荷载标准值**：
w_k = β_z · μ_s · μ_z · w₀

**雪荷载标准值**：
s_k = μ_r · s₀

### E.2 轴心受力构件

**轴心受拉强度**：
N / A_n ≤ f

**轴心受压强度**：
N / A_n ≤ f

**轴心受压稳定**：
N / (φ·A) ≤ f

### E.3 受弯构件

**抗弯强度**：
M_x / (γ_x W_nx) + M_y / (γ_y W_ny) ≤ f

**抗剪强度**：
τ = V·S / (I·t_w) ≤ f_v

**整体稳定**：
M_x / (φ_b W_x) ≤ f

**挠度控制**：
δ_max ≤ L/250（主梁、檩条）

### E.4 压弯构件

**强度**：
N/A_n + M_x/(γ_x W_nx) + M_y/(γ_y W_ny) ≤ f

**平面内稳定**：
N/(φ_x A) + β_mx M_x/(γ_x W_x (1-0.8N/N_EX)) ≤ f

**平面外稳定**：
N/(φ_y A) + η·β_tx M_x/(φ_b W_x) ≤ f

### E.5 连接节点

**螺栓抗剪**：
N_v^b = n_v · (πd²/4) · f_v^b

**螺栓承压**：
N_c^b = d·Σt·f_c^b

**螺栓抗拉**：
N_t^b = (πd_e²/4) · f_t^b

**复合受力**：
√[(N_v/N_v^b)² + (N_t/N_t^b)²] ≤ 1

**角焊缝**：
τ_f = N/(h_e·l_w) ≤ f_f^w

### E.6 地基基础

**地基承载力**：
p_k ≤ f_a
p_k,max ≤ 1.2 f_a

**单桩承载力（估算）** ：
R_a = (1/K)·(Σ q_si·l_i·u + q_p·A_p)

**长细比**：
λ = l₀ / i

### E.7 荷载组合

**可变荷载控制**：
S = γ_G·S_Gk + γ_Q1·S_Q1k + Σ γ_Qi·ψ_ci·S_Qik

**永久荷载控制**：
