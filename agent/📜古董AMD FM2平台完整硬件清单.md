# ⚠️整机重要限制汇总
1. CPU无核显：**显示器必须插独立显卡GT730K接口**
2. 内存上限：主板最大支持16GB DDR3（单条最大8GB）
3. 电源短板：虚标老旧白牌，不建议上功耗更高显卡
4. 性能定位：仅适合网页浏览、办公文档、1080P视频；网游仅最低画质勉强运行
5. 故障提醒：之前带电插拔前置跳线，存在主板IO芯片受损风险
